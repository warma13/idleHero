## 2. 技能效果

召唤一阵冰冷的暴风雪，在目标区域创建一个**持续伤害区域**（脱手技）。

### 2.1 核心机制

- **脱手技**: 释放后自动运行，不需要引导，玩家可继续施放其他技能
- **持续伤害**: 每 1 秒对区域内敌人造成 **[250]% 霜噬伤害**
- **持续冻伤**: 每 tick 对区域内敌人施加 **18% 冻伤**，持续 8 秒
- **智能选点**: 自动选择敌人密度最高的位置作为暴风雪中心（`FindBestAoeCenter`）

### 2.2 伤害数值（按等级）

| 等级 | 每 tick 霜噬伤害 | 公式 |
|------|-----------------|------|
| 1 | 250% | 225 + 1×25 |
| 2 | 275% | 225 + 2×25 |
| 3 | 300% | 225 + 3×25 |
| 4 | 325% | 225 + 4×25 |
| 5 | 350% | 225 + 5×25 |

> 每次升级增加 **25%** 伤害。

### 2.3 同一敌人内置 CD

同一个敌人最快每 **0.5 秒** 被暴风雪命中一次。机制说明：

- 每个暴风雪区域内部维护一个 `enemyHitTimers` 表，以敌人引用为 key
- 命中时记录 0.5 秒倒计时，倒计时未归零的敌人不会被再次命中
- 用于防止多个暴风雪实例叠加时对单个敌人造成过高伤害

### 2.4 冻伤机制

- 每 tick 施加 **18% 冻伤**
- 冻伤累积到 **100%** 时触发 **冻结**（3 秒），冻伤值归零
- 冻伤同时附带 **25% 减速**（2 秒）
- Boss 免疫冻结，但减速仍然生效

## 3. 增强分支（Y 形结构）

```
         [强化暴风雪]        ← 根节点 (line 1)
           /       \
  [法师暴风雪]   [巫师暴风雪]  ← 两个独立分支 (line 2, line 3)
```

- 两个分支**均依赖**根节点"强化暴风雪"
- 两个分支**可以同时学习**，互不排斥

### 3.1 强化暴风雪（根节点）

| 属性 | 值 |
|------|------|
| ID | `blizzard_enhanced` |
| 前置 | 无（增强根节点） |
| 效果 | 暴风雪对**被冻结敌人**造成的伤害提高 **40%[x]** |

**实现细节**：
- 在 fireZone 上设置 `frozenBonus = 0.40`
- 每次 tick 命中敌人时检查 `e.isFrozen`，命中冻结目标伤害乘以 `1.40`
- `[x]` 表示乘算加成（独立乘区）

### 3.2 法师暴风雪（分支 A）

| 属性 | 值 |
|------|------|
| ID | `blizzard_mage` |
| 前置 | `blizzard_enhanced` |
| 效果 | 暴风雪的持续时间延长 **4 秒**（8 → 12 秒） |

**实现细节**：
- 施放时检查 `H.HasEnhance("blizzard_mage")`
- 将 `duration` 从 8 增加到 12
- 同时影响 `blizzardTimer`（巫师暴风雪法力回复持续时间也随之延长）

### 3.3 巫师暴风雪（分支 B）

| 属性 | 值 |
|------|------|
| ID | `blizzard_wizard` |
| 前置 | `blizzard_enhanced` |
| 效果 | 暴风雪激活时，你每有 **20** 点法力上限，法力回复速度就会提高 **1** 点 |

**实现细节**：
- 施放时缓存 `GameState._hasBlizzardWizard = true`
- 在 `StatCalc.GetManaRegen()` 中检查 `blizzardActive` 和 `_hasBlizzardWizard`
- 公式: `bonusRegen = math.floor(GetMaxMana() / 20)`
- 该加成为**加算**（直接加到最终法力回复量上）

## 4. 状态管理

### 4.1 GameState 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `blizzardActive` | boolean | 暴风雪是否处于激活状态 |
| `blizzardTimer` | number | 暴风雪剩余持续时间（秒） |
| `_hasBlizzardWizard` | boolean | 是否拥有巫师暴风雪增强（缓存） |

### 4.2 生命周期

```
施放暴风雪
  ↓
创建 fireZone (持续伤害+冻伤区域)
设置 blizzardActive = true
设置 blizzardTimer = duration (8 或 12)
缓存 _hasBlizzardWizard
  ↓
每帧 BuffRuntime.TickBuffs:
  blizzardTimer -= dt
  ↓
blizzardTimer <= 0:
  blizzardActive = false
  blizzardTimer = 0
```

### 4.3 重置

`BuffRuntime.ResetBuffs()` 中重置:
```lua
GameState.blizzardActive = false
GameState.blizzardTimer = 0
```

## 6. fireZone 数据结构

```lua
{
    x = bestX, y = bestY,       -- 区域中心坐标
    radius = 100,                -- 区域半径
    duration = 8.0,              -- 剩余持续时间
    maxDuration = 8.0,           -- 最大持续时间
    dmgPct = 2.50,               -- 每tick伤害百分比 (250%)
    tickRate = 1.0,              -- tick间隔 (秒)
    tickCD = 0,                  -- 当前tick倒计时
    element = "ice",             -- 元素类型
    source = "blizzard",         -- 来源标识
    frostbitePct = 18,           -- 每tick冻伤量 (18%)
    frozenBonus = 0.40,          -- 冻结敌人额外伤害 (强化暴风雪)
    perEnemyICD = 0.5,           -- 同一敌人内置CD (秒)
    enemyHitTimers = {},         -- 敌人命中时间戳 {[enemy] = remainingCD}
    luckyHitChance = 0.33,       -- 幸运一击几率
}
```

## 7. 与其他技能的联动

| 联动 | 说明 |
|------|------|
| **冰霜弹冻伤** | 冰霜弹施加冻伤 → 累积触发冻结 → 暴风雪(强化)对冻结目标 +40% 伤害 |
| **寒冰甲** | 寒冰甲(神秘)周期施加冻伤 → 加速暴风雪区域内敌人的冻结触发 |
| **冰碎** | 暴风雪持续冻伤可频繁触发冻结 → 冰碎被动可在冻结时获得额外效果 |
| **巫师暴风雪 + 法力池** | 法力上限越高，暴风雪激活时法力回复越多 → 允许更频繁施放其他技能 |
