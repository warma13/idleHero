# 寒冰甲 (Ice Armor)

> 防御技能 · 冰霜元素 · Tier 3

## 基础技能

| 属性 | 数值 |
|------|------|
| 冷却时间 | 20 秒 |
| 持续时间 | 6 秒 |
| 最大等级 | 5 |

在你周围形成一个由寒冰形成的屏障，持续 6 秒，吸收相当于你生命上限一定百分比的伤害。

### 等级数值

| 等级 | 护盾吸收量（生命上限 %） |
|------|--------------------------|
| 1 | 56% |
| 2 | 62% |
| 3 | 67% |
| 4 | 73% |
| 5 | 78% |

---

## 增强节点

### 强化寒冰甲 (ice_armor_enhanced)

> 第 1 层增强

寒冰甲激活时，你的法力回复速度提高 **30%[x]**。

- 乘算系数：× 1.30
- 生效条件：`iceArmorActive == true`
- 作用位置：StatCalc.GetManaRegen，作为独立乘区

---

### 神秘寒冰甲 (ice_armor_mystical)

> 第 2 层增强 · 分支 A

寒冰甲激活时，你会周期性地对近距离敌人造成 **20%** 的冻伤效果，并对被冻结的敌人造成的伤害提高 **15%[x]**。

| 参数 | 数值 |
|------|------|
| 冻伤施加周期 | 1.5 秒 |
| 冻伤强度 | 20% |
| 作用范围 | 100 像素（战斗坐标） |
| 冻结伤害加成 | 15%[x]（乘算，X 伤害桶） |

- 冻伤每 1.5 秒对范围内所有存活敌人施加一次
- 伤害加成为 DamageFormula 第 5 桶（X 伤害）的独立乘区：× 1.15
- 生效条件：目标 `isFrozen == true`

---

### 微光寒冰甲 (ice_armor_shimmering)

> 第 2 层增强 · 分支 B

寒冰甲激活时，你每花费 **50** 点法力，都可使寒冰甲的冷却时间减少 **1 秒**。

| 参数 | 数值 |
|------|------|
| 法力消耗阈值 | 50 点 |
| 每次减少冷却 | 1 秒 |
| 累计方式 | 累加器，跨技能累计 |

- 法力消耗在 Combat.SpendMana 中追踪
- 累计达到阈值后立即扣减 ice_armor 的 skillTimer
- 一次大额消耗可触发多次减 CD（while 循环）
- 仅在寒冰甲激活期间生效
