## 总览: 五区乘算管线

```
最终伤害 = 基础区 × 增伤区 × 双爆区 × 减防区 × 抗性区
         (+ 管线外固定伤害)
         × 技能伤害乘区 (仅技能)
         × 反应区 (仅触发反应时)
```

**设计原则**:
- **区间乘算**: 不同区之间相乘 → 每个区都有独立的提升价值
- **区内加算**: 同一区内的加成相加 → 同区加成边际递减
- **新增加成 → 加入对应区的加算池**
- **新增区 → 往管线追加一个函数**

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  基础区  │ ×  │  增伤区  │ ×  │  双爆区  │ ×  │  减防区  │ ×  │  抗性区  │
│ ATK×倍率 │    │ 1+Σbonus│    │ critDmg  │    │ DEF减免  │    │ 元素减免 │
└─────────┘    └─────────┘    └─────────┘    └─────────┘    └─────────┘
                                                                 │
                                                          ┌──────┴──────┐
                                                          │ +固定额外伤害│
                                                          └──────┬──────┘
                                                                 │
                                                    ┌────────────┴────────────┐
                                                    │ ×技能伤害乘区 (仅技能)   │
                                                    └────────────┬────────────┘
                                                                 │
                                                    ┌────────────┴────────────┐
                                                    │ ×反应区 (仅触发反应时)   │
                                                    └─────────────────────────┘
```

## 区1: 基础区

**输出**: 绝对伤害值 (不是乘数)

```
基础区 = totalAtk × multiplier
```

或当弹体预计算了基础伤害时:

```
基础区 = baseDmg     -- 弹体已包含 ATK×倍率
```

| 参数 | 来源 | 说明 |
|------|------|------|
| totalAtk | `GameState.GetTotalAtk()` | 见属性公式 ATK |
| multiplier | 调用方传入 | 技能倍率, 默认 1.0 |

## 区2: 增伤区

**输出**: 乘数 = `1 + Σbonus`

```
bonus = 0

── 元素增伤 (非物理攻击时) ──
+ elemDmgBonus                          -- 按攻击元素计算 (见下方)

── 药水 ──
+ atkPotionBonus                        -- 攻击药水 (GetAtkPotionMul() - 1)

── 元素印记 (目标被附着时) ──
+ markBonus                             -- elem_mark 技能: 2%/4%/6%

── 套装标记增伤 ──
+ stormMarkBonus                        -- 风暴裁决标记
+ voidMarkBonus                         -- 虚空涌潮标记
+ celestialMarkBonus                    -- 天穹之怒标记
+ celestialHpRatioBonus                 -- 天穹HP比例加成

── 元素收敛 ──
+ convergenceTotal                      -- 每种不同元素 × 5%/级

── 冻结碎裂 ──
+ iceShatterBonus                       -- 冻结碎裂额外伤害

── 套装被动增伤 ──
+ iceBreakerBonus                       -- 碎冰者2件
+ frozenFieldBonus                      -- 永冻之心6件
+ ashFlameBonus                         -- 灰烬之焰6件
+ soulFlameBurnBonus                    -- 净魂烈焰4件 (目标被点燃时 +30%)
+ setPassiveBonus                       -- 套装 stats.atkDmg / normalAtkDmg (普攻)
+ swiftHunterBonus                      -- 迅捷猎手4件 (同目标叠层)
+ dragonFuryAtkBonus                    -- 龙息之怒4件 (普攻后技能增伤)
+ dragonFurySkillBonus                  -- 龙息之怒4件 (技能后普攻增伤)
+ dragonMightBonus                      -- 龙息之怒6件 (龙威全伤害)
+ runeNextSkillBonus                    -- 符文编织4件 (下次技能, 消耗型)
+ runeResonanceSkillDmg                 -- 符文编织6件 (共鸣期间技能)

── 技能/天赋专属加成 ──
+ extraBonuses                          -- 调用方传入, 如技能专属加成

增伤区 = 1 + bonus
```

### elemDmgBonus 元素增伤计算

按**攻击元素** (非武器元素) 计算:

```
elemDmgBonus = equipSum(元素对应词条)    -- 如 fireDmg, iceDmg
             + equipSum("elemDmg")       -- 存量全元素增伤
             + 套装stats.elemDmg         -- 套装被动全元素增伤
```

元素 → 词条映射:
| 元素 | 词条 |
|------|------|
| fire | fireDmg |
| ice | iceDmg |
| poison | poisonDmg |
| arcane | arcaneDmg |
| water | waterDmg |

## 区3: 双爆区

**输出**: 乘数

```
if 暴击:
  双爆区 = CritDmg                      -- 如 1.8 = 180% 暴击伤害
else:
  双爆区 = 1.0
```

暴击判定:
```
critRate = GetCritRate() + extraCritRate + 暗影猎手暴击加成
isCrit = random() < min(1.0, critRate)
```

## 区4: 减防区

**输出**: 乘数 (0~1, 值越小伤害越低)

```
effectiveDef = 敌人DEF
if 目标被减防:
  effectiveDef = effectiveDef × (1 - defReduceRate)

if effectiveDef > 0:
  K = ENEMY_DEF_K × scaleMul             -- K 随关卡缩放
  减防区 = 1 - effectiveDef / (effectiveDef + K)
else:
  减防区 = 1.0
```

| 参数 | 值 | 说明 |
|------|-----|------|
| ENEMY_DEF_K | 100 | 基础减防常数 |
| scaleMul | StageConfig.GetScaleMul(ch, stage) | 关卡缩放值 |

**说明**: K 随关卡缩放确保减防比例在不同章节保持稳定。

## 区5: 抗性区

**输出**: 乘数 (三段曲线)

```
resist = 目标元素抗性[攻击元素]
if 目标被元素削弱:
  resist = resist - elemWeakenRate

── 三段曲线 ──
if resist < 0:
  抗性区 = 1 - resist/2                  -- 负抗有收益但减半
elseif resist < 0.75:
  抗性区 = 1 - resist                    -- 线性 (兼容旧逻辑)
else:
  抗性区 = 1 / (1 + resist/4)            -- 渐近线, 永远打不到0
```

**示例**:
| 抗性值 | 乘数 | 伤害变化 |
|--------|------|---------|
| -0.20 | 1.10 | +10% 伤害 |
| 0.00 | 1.00 | 无变化 |
| 0.30 | 0.70 | -30% 伤害 |
| 0.50 | 0.50 | -50% 伤害 |
| 0.75 | 0.842 | -15.8% (进入渐近) |
| 1.00 | 0.800 | -20% |
| 2.00 | 0.667 | -33.3% |

## 管线外乘区

### 固定额外伤害 (管线外平加)

```
管线伤害 = 管线伤害 + oceanFuryBonusDmg   -- 沧溟之怒4件: 腐蚀目标额外固定伤害
```

### 技能伤害乘区 (仅 damageTag == "skill" 时生效)

```
if 技能伤害:
  skillDmgBonus = GetSkillDmg()           -- 装备+套装+精通 加算
  管线伤害 = 管线伤害 × (1 + skillDmgBonus)
```

**说明**: 技能伤害独立乘区, 与增伤区分开, 避免与普攻增伤相互稀释。

### 反应区 (仅触发元素反应时)

```
reactionMul = GetReactionDmgBonus()       -- 通用反应增伤 (技能+装备+套装+buff)
            + 元素狂热反应加成

── 五元素专属天赋 ──
for elem in {fire, ice, poison, water, arcane}:
  if 反应涉及该元素:
    reactionMul += 专属天赋等级 × 0.15    -- 每级+15%

if reactionMul > 1.0:
  最终伤害 = floor(管线伤害 × reactionMul)
```

## 完整计算流程示例

**场景**: 火元素技能攻击, 200% 倍率, 暴击, 目标有 50 点 DEF, 10% 火抗

```
1. 基础区 = ATK(1000) × 2.0 = 2000

2. 增伤区 = 1 + 火焰增伤(0.25) + 药水(0.05) + 印记(0.04) = 1.34

3. 双爆区 = CritDmg(2.0) = 2.0

4. 减防区:
   K = 100 × 1.0 = 100
   = 1 - 50/(50+100) = 0.667

5. 抗性区 = 1 - 0.10 = 0.90

管线伤害 = 2000 × 1.34 × 2.0 × 0.667 × 0.90
         = floor(3222.5) = 3222

6. 技能乘区 = 1 + SkillDmg(0.15) = 1.15
   = floor(3222 × 1.15) = 3705

7. 如触发反应 (如蒸发 ×1.5):
   reactionMul = 1.0 + 反应精通(0.10) = 1.10
   反应伤害 = 蒸发基础(3705 × 1.5) × 1.10 = floor(6113)
```

## 元素反应系统

### 元素类型

火焰(fire), 冰霜(ice), 毒素(poison), 奥术(arcane), 流水(water), 物理(physical)

### 14 种有序反应

反应由 `底元素 > 触发元素` 决定 (底元素 = 已附着, 触发元素 = 新命中):

| 反应 | 底>触 | 效果 |
|------|-------|------|
| 淬爆 | fire>ice | 1.5x 伤害 |
| 消融 | ice>fire | 1.2x 伤害 + 减防20%(3秒) |
| 燎原 | fire>poison | 1.3x 伤害 + AOE(半径60) |
| 引爆 | poison>fire | 1.6x 伤害 |
| 寒痹 | ice>poison | 1.5秒眩晕 |
| 凝毒 | poison>ice | 1秒眩晕 + AOE减速30%(2秒) |
| 汽爆 | fire>water | 1.3x 伤害 + 减速30%(3秒) |
| 蒸发 | water>fire | 1.5x 伤害 |
| 冰封 | ice>water | 2秒眩晕 (最强控) |
| 霜涌 | water>ice | AOE(半径70) 减速40%(2.5秒) |
| 溶毒 | poison>water | DoT 2.0x 持续4秒 |
| 侵蚀 | water>poison | AOE(半径60) DoT 0.5x 持续3秒 |
| 共鸣 | arcane>any | 1.3x 伤害 |
| 激发 | any>arcane | 1.2x 伤害 + 小AOE(半径50) |

### 元素附着等级

| 来源 | 等级 |
|------|------|
| 普攻 | 弱(1) |
| 技能 | 强(2) |
| 反应产生 | 强(2) |

弱元素参与反应时伤害系数: ×0.7

### 玩家基础元素抗性

| 元素 | 抗性 |
|------|------|
| fire | 10% (术士天生) |
| 其他 | 0% |
