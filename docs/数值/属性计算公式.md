## 通用机制

### 章节数值缩放

装备词条和属性点的数值随章节推进而缩放:

```
装备Tier系数 tierMul(ch):
  ch=1 → 1.0
  ch>1 → 1 + 99 × ln(ch) / ln(100)    -- 对数增长, ch100达100倍

属性点缩放因子 attrScale(ch):
  attrScale = √tierMul(ch)              -- 平方根, 保持属性点贡献 10-17%
```

### 装备属性求和 equipSum(stat)

遍历6个装备槽位，累加所有主词条+副词条中匹配的属性值。

### 称号加成 getTitleBonus(statKey)

来自称号系统的属性加成，按属性类型查询。

### 套装加成

- `GetSetBonusStats()` — 套装固定加值 (stats 字段, 加算)
- `GetSetBonusStatsMul()` — 套装百分比加成 (statsMul 字段, 用于乘算)

## 攻击属性

### ATK (攻击力)

**公式**: 单增减益池

```
base = baseAtk + 加点×POINT_BONUS.atk×attrScale + equipSum("atk")

bonusPool = Σ {
  + getTitleBonus("atk")               -- 称号攻击力加成
  + 力量灌注等级 × 0.08               -- 技能: 每级+8%
  + 怨魂之怒层数加成                   -- 套装BUFF: 每层+3%ATK
  + 战意高扬加成                       -- 主动BUFF
  + 狂暴加成                           -- 主动BUFF
  + 铁壁要塞ATK加成                    -- 套装BUFF: 护盾溢出转ATK
  - 墨汁致盲降低量                     -- debuff: 降低ATK
}

ATK = floor(base × (1 + bonusPool))
```

**参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| baseAtk | 18 | 术士基础攻击力 |
| POINT_BONUS.atk | 6 | 每点攻击力 (×attrScale) |

### AtkSpeed (攻击速度)

**公式**: 三阶段计算 — 渐近线递减 → buff乘法 → debuff乘法

```
── 第1步: 原始攻速 ──
rawSpeed = baseAtkSpeed + 加点×0.05 + equipSum("spd")
rawSpeed = rawSpeed × (1 + 套装atkSpeed加成)   -- 如 swift_hunter 2件 +12%

── 第2步: 渐近线递减 ──
Δ = rawSpeed - 1.0
if Δ > 0:
  cap = baseCap × attrScale(chapter)
  effective = 1.0 + cap × Δ / (Δ + K)
else:
  effective = rawSpeed

── 第3步: buff 乘法 (作用于递减后, 保留爆发感) ──
buffMul = 1.0
  × (1 + 元素狂热攻速加成)             -- frenzyAtkSpeedBonus
  × (1 + 毒牙之焰4件攻速加成)          -- venomFangAtkSpdBonus
  × (1 + 丛林吐息4件攻速加成)          -- groveAtkSpdBonus
  × (1 + 狂暴攻速加成)                 -- berserkAtkSpeedBonus
  × (1 + 速射等级 × 0.08)             -- 技能: 每级+8%
effective = effective × buffMul

── 第4步: debuff 乘法 ──
effective = effective × (1 - 减速率)
effective = effective × (1 - 孢子云攻速降低)

AtkSpeed = max(0.1, effective)          -- 最低 0.1
```

**参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| baseAtkSpeed | 1.0 | 基础攻速 |
| POINT_BONUS.spd | 0.05 | 每点攻速 |
| ATK_SPEED_DR.baseCap | 4.0 | Ch1增量上限 (最大≈5.0) |
| ATK_SPEED_DR.K | 3.0 | 半值常数 |

### CritRate (暴击率)

**公式**:

```
rawCrit = baseCritRate + 加点×0.02 + equipSum("crit")
        + 奥术感知等级 × 0.03           -- 技能: 每级+3%
        + getTitleBonus("crit")         -- 称号暴击率

CritRate = max(0, min(1.0, rawCrit - 沙暴暴击降低))

溢出暴击率 = max(0, rawCrit - 1.0)     -- 超过100%的部分, 用于转换暴伤
```

**参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| POINT_BONUS.crit | 0.02 | 每点暴击率 (+2%) |

### CritDmg (暴击伤害倍率)

**公式**: 全加算

```
CritDmg = baseCritDmg + 加点×0.10 + equipSum("critDmg")
        + 溢出暴击率 × critOverflowRatio   -- 溢出转暴伤
        + 套装mul.critDmg                  -- 套装加成 (加算)
        + getTitleBonus("critDmg")          -- 称号加成 (加算)
```

**参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| baseCritDmg | 1.8 | 术士基础暴击伤害 (180%) |
| POINT_BONUS.critDmg | 0.10 | 每点暴击伤害 (+10%) |
| critOverflowRatio | 2 | 1%溢出暴击率 → 2%暴伤 |

### Range (攻击范围)

**公式**: 渐近线递减 (绝对值)

```
rawBonus = 加点×8 + equipSum("range")

if rawBonus > 0:
  Range = baseRange + maxBonus × rawBonus / (rawBonus + K)
else:
  Range = baseRange
```

**参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| baseRange | 65 | 基础攻击距离 |
| POINT_BONUS.range | 8 | 每点范围增量 |
| RANGE_DR.maxBonus | 150 | 理论最大增量 |
| RANGE_DR.K | 40 | 半值常数 |

**范围倍率因子**: `rangeFactor = currentRange / baseRange` (用于技能/精灵距离缩放)

### Luck (幸运)

**公式**: 纯加算

```
Luck = 加点×0.03 + equipSum("luck") + 药水buff + getTitleBonus("luck")
```

**参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| POINT_BONUS.luck | 0.03 | 每点幸运 (+3%) |

## 生存属性

### MaxHP (最大生命值)

**公式**:

```
── 基础值 ──
base = baseHP + 等级×hpPerLevel
     + HP加点 × HP_BONUS.hp × attrScale      -- 每点+30HP×attrScale
     + floor(equipSum("hp"))
     + VIT加点 × VIT_BONUS.hp × attrScale     -- 兼容旧存档

── 百分比加成池 ──
hpPctBonus = HP加点 × HP_BONUS.hpPct          -- 每点+0.3%
           + equipSum("hpPct")
           + getTitleBonus("hp")
           + VIT加点 × VIT_BONUS.hpPct         -- 兼容旧存档

── 乘算 ──
hpMul = 1.0 + 套装mul.hp + hpPctBonus + 生命药水buff

MaxHP = floor(base × hpMul)
```

**参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| baseHP | 500 | 基础生命值 |
| hpPerLevel | 12 | 每级+12HP |
| HP_BONUS.hp | 30 | HP加点: 每点+30HP (×attrScale) |
| HP_BONUS.hpPct | 0.003 | HP加点: 每点+0.3%最大HP |

### DEF (防御力)

**公式**: 单倍率池

```
base = baseDEF + 等级×defPerLevel
     + VIT加点 × VIT_BONUS.def × attrScale
     + equipSum("def")

mulPool = 套装mul.def + getTitleBonus("def")

DEF = floor(base × (1 + mulPool))
```

**参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| baseDEF | 5 | 基础防御力 |
| defPerLevel | 0.3 | 每级+0.3DEF |
| VIT_BONUS.def | 0.5 | VIT加点: 每点+0.5DEF (×attrScale) |

### DEF 减免率 (玩家受伤)

```
effectiveDef = DEF × (1 - 腐蚀层数 × 腐蚀DEF降低率)
减免率 = effectiveDef / (effectiveDef + defK)
```

| 参数 | 值 |
|------|-----|
| defK | 200 |

### HPRegen (每秒回血)

```
HPRegen = VIT加点 × 1.5 + equipSum("hpRegen")
```

| 参数 | 值 |
|------|-----|
| VIT_BONUS.hpRegen | 1.5 |

### HealMul (治疗倍率)

```
HealMul = 1.0 + VIT加点 × 0.008 + equipSum("healPct")
```

| 参数 | 值 |
|------|-----|
| VIT_BONUS.healPct | 0.008 |

### ShieldMul (护盾倍率)

```
ShieldMul = 1.0 + VIT加点 × 0.005 + equipSum("shldPct")
```

| 参数 | 值 |
|------|-----|
| VIT_BONUS.shldPct | 0.005 |

### LifeSteal (生命偷取)

```
LifeSteal = equipSum("lifeSteal")         -- 纯装备词条
每秒吸血上限 = MaxHP × 0.08              -- 8% MaxHP/秒

吸血效率:
  普攻     = 100%
  技能     = 60%
  召唤物   = 30%
  持续区域 = 20%
```

## 元素属性

### ElemDmg (元素增伤)

```
ElemDmg = equipSum(武器元素对应增伤)      -- 如 fireDmg
        + equipSum("elemDmg")             -- 存量全元素增伤 (不再产出新词条)
        + getTitleBonus("allElemDmg")      -- 称号全元素增伤
```

**说明**: `elemDmg` 词条已从装备生成池移除，但旧装备的数值保留。

### ReactionDmg (反应增伤)

```
ReactionDmgBonus = 1.0 + Σ {
  + 反应精通等级 × 0.05                   -- 技能: 每级+5%
  + 元素过载 {3%, 6%, 10%}               -- 技能: 按等级
  + equipSum("reactionDmg")              -- 装备反应增伤
  + 套装stats.reactionDmg               -- 套装被动
  + 灰烬之焰4件BUFF (+30%)             -- 反应后5秒
  + 元素狂热buff                         -- (在伤害公式中额外加入)
}
```

## 技能属性

### SkillCdMul (技能冷却倍率)

**公式**: 渐近线递减 (无硬顶)

```
totalCDR = 法力亲和等级 × 0.04            -- 技能树
         + equipSum("skillCdReduce")      -- 装备
         + 套装mul.skillCdReduce          -- 套装被动
         + 精通加点 × 0.015              -- 精通属性点

if totalCDR > 0:
  cdMul = 1.0 - maxCDR × totalCDR / (totalCDR + K)
else:
  cdMul = 1.0
```

**参数**:
| 参数 | 值 | 说明 |
|------|-----|------|
| CDR_DR.maxCDR | 0.75 | 渐近线: 最多减75% (永远达不到) |
| CDR_DR.K | 0.50 | 半值常数: 投入0.50时 ≈ 37.5% CDR |

### SkillDmg (技能伤害加成)

**公式**: 加算汇总，在伤害管线外作为独立乘区

```
SkillDmg = equipSum("skillDmg")
         + 套装mul.skillDmg + 套装stats.skillDmg
         + 精通加点 × 0.01               -- 每点精通+1%
```

**使用**: `技能伤害 = 管线伤害 × (1 + SkillDmg)`

## 其他属性

### DebuffResist (减益抗性/韧性)

```
resist = baseResist + TEN加点 × 0.02 + getTitleBonus("debuffResist")
DebuffResist = min(maxResist, resist)

作用:
  效果衰减 = 原始效果 × (1 - resist)
  持续衰减 = 原始持续 × (1 - resist × durFactor)
```

| 参数 | 值 |
|------|-----|
| baseResist | 0 |
| perPointResist | 0.02 |
| maxResist | 0.80 |
| durFactor | 0.5 |

### DPS (每秒伤害)

```
DPS = floor(ATK × AtkSpeed × (1 + CritRate × (CritDmg - 1)))
```

### Power (综合战力)

```
Power = floor(
    ATK×2 + AtkSpeed×50 + CritRate×100 + CritDmg×40
  + Range×1.5 + Luck×80
  + MaxHP×0.1 + DEF×3
  + ElemDmg×60 + ReactionDmg×60
)
```
