# 第四章：基础技能与核心技能详解

## 概述

基础技能（Basic Skills）和核心技能（Core Skills）构成了D4技能树的前两层，是整个Build构建的基础骨架。基础技能负责资源生成和高频触发效果，核心技能则承担绝大多数的伤害输出。本章将对所有职业的基础技能和核心技能进行逐一详解，包括数值、强化效果、机制分析和Build价值评估。

---

## 4.1 野蛐人基础技能详解

### 4.1.1 重击（Bash）

**基础效果（1点）：**
- 对目标造成**30%**武器伤害
- 幸运一击概率：**40%**
- 额外机制：每命中4次敌人，下次重击造成眩晕，持续**1.25秒**（双手武器：2秒）

**技能标签：** Basic, Physical, Melee, Bludgeoning

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Bash（强化重击） | 处于狂暴状态下，每次重击生成额外**5点**怒火 |
| Combat Bash（战斗重击） | 每次击中时提供**2点**怒火；如果命中3个以上敌人，立即激活狂暴状态3秒 |
| Fierce Bash（猛烈重击） | 眩晕效果期间对敌人造成额外**10%**伤害；眩晕敌人受到攻击时有**30%**概率延长眩晕0.5秒 |

**数值分析：**
- 每5级时（5/5投入）：**54%**武器伤害（30% × 1.8）
- 加上崇高词缀加到7级：**66%**武器伤害
- 战斗重击的狂暴激活条件（命中3+）在密集怪物群中极易满足

**Build价值：** ★★★☆（中等价值，主要用于眩晕流和狂暴流Build）

---

### 4.1.2 剥皮（Flay）

**基础效果（1点）：**
- 造成**5%**武器伤害
- 同时施加**40%**流血持续伤害（持续**5秒**）
- 幸运一击概率：**40%**

**技能标签：** Basic, Physical, Melee, Slashing

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Flay（强化剥皮） | 每次剥皮命中时，为你提供**1点**护甲加成（每秒+100护甲，最多可叠加5层） |
| Combat Flay（战斗剥皮） | 对流血状态的敌人攻击时，剥皮额外造成**2%**武器伤害，并额外生成**2点**怒火 |
| Fierce Flay（猛烈剥皮） | 流血效果强化：流血伤害的速度每秒额外+X%（随级别提升），最终总流血量更高 |

**数值分析：**
- 剥皮本身的直接伤害仅为5%，极低
- 但5秒40%的流血伤害 = 每次使用相当于造成45%总伤害
- 5/5满级时：流血伤害 = **约58%武器伤害（5秒）**
- 战斗剥皮 + 流血叠加是野蛐人流血流的核心起手

**Build价值：** ★★★★☆（流血流核心，其他Build较少使用）

---

### 4.1.3 狂乱（Frenzy）

**基础效果（1点）：**
- 造成**22%**武器伤害
- 命中后获得**20%**攻速提升，可叠加，最多3层（最大**60%**攻速）
- 层数持续时间：3秒（每次命中刷新）
- 幸运一击概率：**30%**

**技能标签：** Basic, Physical, Melee, Slashing

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Frenzy（强化狂乱） | 当处于3层满叠加时，获得**5%**狂暴状态使用时间 |
| Combat Frenzy（战斗狂乱） | 3层满叠加时，每次命中生成额外**3点**怒火 |
| Fierce Frenzy（猛烈狂乱） | 3层满叠加时，伤害提升**5%** |

**数值分析：**
- 狂乱是野蛐人攻速最高的基础技能
- 3层满叠（+60%攻速）下，每秒输出约等于单纯数值的1.6倍
- 资源生成：约5.2-5.8次/秒 × 12点怒火 = **约63-70点怒火/秒**（满叠加）
- 这是野蛐人怒火生成最快的基础技能

**Build价值：** ★★★★（高攻速流、快速怒火生成Build的核心）

---

### 4.1.4 突刺（Lunging Strike）

**基础效果（1点）：**
- 向目标突进并造成**33%**武器伤害
- 若目标距离超过**6米**，造成额外**20%**伤害
- 幸运一击概率：**40%**

**技能标签：** Basic, Physical, Melee, Slashing

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Lunging Strike（强化突刺） | 若目标是精英，突刺治愈你**4%**最大生命 |
| Combat Lunging Strike（战斗突刺） | 突刺后获得**3秒**狂暴状态 |
| Fierce Lunging Strike（猛烈突刺） | 突刺距离提升；对精英的额外伤害提升至**35%** |

**数值分析：**
- 突刺是唯一带有位移（冲刺）效果的基础技能
- 远距离加成（>6米）使其在拉开距离的场景下非常实用
- Combat版本的3秒狂暴激活对任何狂暴流Build都是强力起手

**Build价值：** ★★★★（通用价值高，是狂暴流Build的首选基础技能）

## 4.2 野蛐人核心技能详解

### 4.2.1 旋风斩（Whirlwind）

**基础效果（1点）：**
- 每秒对周围**所有敌人**造成**100%**武器伤害
- 持续旋转期间消耗**25点**怒火/秒
- 幸运一击概率：**20%**（每次旋转命中）

**技能标签：** Core, Physical, Melee, Bludgeoning/Slashing（取决于武器绑定）

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Whirlwind（强化旋风斩） | 旋风斩时怒火生成速度+**25%** |
| Violent Whirlwind（猛烈旋风斩） | 对流血状态的敌人额外造成**30%**伤害 |
| Furious Whirlwind（愤怒旋风斩） | 旋风斩时同时使用所有绑定武器，每额外武器造成**16%**伤害 |

**数值分析（5/5满级）：**
- 基础DPS：**180%**武器伤害/秒
- Violent版（对流血目标）：**180% × 1.3 = 234%**/秒
- Furious版（4武器绑定）：**180% + 3×16% = 228%**/秒 + 武器词缀收益
- 怒火持续速率：25点/秒（需要基础技能填充或怒火战吼辅助）

**Build价值：** ★★★★★（旋风斩是野蛐人历史最长的主力技能，几乎所有野蛐人Build的首选核心技能）

---

### 4.2.2 先祖之锤（Hammer of the Ancients）

**基础效果（1点）：**
- 向前方砸下巨锤，造成**50%**武器伤害（直线AOE）
- 额外造成：若处于狂暴状态，伤害提升**35%**
- 幸运一击概率：**50%**

**技能标签：** Core, Physical, Melee, Bludgeoning

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Hammer of the Ancients | 命中处于强固状态的敌人时，有**30%**几率重置怒火消耗 |
| Violent HotA | 激活狂暴状态3秒（每次命中） |
| Furious HotA | 造成压制伤害时，在地面形成余波持续3秒 |

**数值分析（5/5满级）：**
- 基础伤害：**90%**武器伤害
- 处于狂暴状态：**90% × 1.35 = 121.5%**武器伤害
- 幸运一击50%是所有核心技能中最高的

**Build价值：** ★★★★（压制流Build的核心，与狂暴机制高度协同）

---

### 4.2.3 崩山击（Upheaval）

**基础效果（1点）：**
- 造成**77%**武器伤害，影响前方扇形区域
- 引燃地面，对区域内敌人造成持续**1秒**的**灼烧**伤害
- 幸运一击概率：**30%**

**技能标签：** Core, Physical, Fire, Melee

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Upheaval | 怒火满时，崩山击不消耗怒火 |
| Violent Upheaval | 命中3个及以上敌人时，激活狂暴状态4秒 |
| Furious Upheaval | 怒火满时崩山击，伤害提升**80%** |

**数值分析（5/5满级）：**
- 基础伤害：**138.6%**武器伤害（满级）
- Furious满怒火版：**138.6% × 1.8 = 249.5%**武器伤害
- 是野蛐人核心技能中单次伤害最高的

**Build价值：** ★★★☆（满怒火爆发流的选择，相比旋风斩较小众）

## 4.3 法师基础技能详解

### 4.3.1 火弹（Fire Bolt）

**基础效果（1点）：**
- 造成**18%**武器伤害（直接火焰伤害）
- 同时施加灼烧伤害：**43.98%**武器伤害（持续时间：约3秒）
- 每次命中恢复**11点**法力
- 幸运一击概率：**50%**

**技能标签：** Basic, Fire, Pyromancy

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Fire Bolt | 灼烧目标时，你的暴击率+**3%** |
| Glinting Fire Bolt（闪光火弹） | 每次命中在目标位置留下炎爆点，下次技能命中时引爆 |
| Flickering Fire Bolt（摇曳火弹） | 对灼烧目标的幸运一击额外触发：冷却时间降低0.25秒 |

**数值分析：**
- 直接伤害 + DoT = 总计约62%伤害（1点）
- 法力恢复11点是基础技能中较高的
- 与火焰流Build协同最强，是"灼烧→火球爆炸"协同的起点

**Build价值：** ★★★★（火焰Build的核心基础技能）

---

### 4.3.2 冰霜弹（Frost Bolt）

**基础效果（1点）：**
- 造成**38%**武器伤害（冰冷伤害）
- 减速目标**15%**，持续**3秒**
- 第2次命中同一目标时，减速提升至**30%**
- 每次命中恢复**10点**法力
- 幸运一击概率：**50%**

**技能标签：** Basic, Cold, Frost

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Frost Bolt | 冻结的敌人受到冰霜弹的伤害时，有**15%**概率碎裂，造成AOE伤害 |
| Glinting Frost Bolt | 减速量提升至**35%** |
| Flickering Frost Bolt | 连续命中同一敌人4次后，下一次冰霜弹造成**200%**伤害 |

**数值分析：**
- 直接伤害38%是基础技能中较高的（火花有4发但总计低）
- 减速效果使冰法Build可以轻松维持敌人处于减速状态
- 是"冻结→碎裂"机制的起点

**Build价值：** ★★★★★（冰法Build的核心基础技能，配合冰碎片极强）

---

### 4.3.3 电弧鞭笞（Arc Lash）

**基础效果（1点）：**
- 造成**42%**武器伤害（近战范围闪电）
- 击退命中的敌人
- 每次命中恢复**9点**法力
- 幸运一击概率：**30%**

**技能标签：** Basic, Lightning, Shock

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Arc Lash | 持续使用时，每第6次命中触发冲击效果，使敌人眩晕**1.5秒** |
| Glinting Arc Lash | 幸运一击：每次命中有30%概率自动使用已激活的其他技能 |
| Flickering Arc Lash | 每次命中获得**1%**减伤效果，最多叠加**10%**，持续到下次受到伤害 |

**数值分析：**
- 电弧鞭笞是法师唯一的近战基础技能
- 击退效果使其在近战法师Build中有独特价值
- Flickering版的减伤叠加对生存能力有显著提升

**Build价值：** ★★★★（闪电流Build的核心，近战法师流的特殊选择）

---

### 4.3.4 火花（Spark）

**基础效果（1点）：**
- 连续发射**4发**弹射物，每发造成**10%**武器伤害
- 总伤害：**40%**武器伤害
- 每发弹射物可以弹射到相邻敌人
- 每次命中恢复**8点**法力
- 幸运一击概率：**15%**（每发）

**技能标签：** Basic, Lightning, Shock

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Spark | 每第4次幸运一击，触发一次**闪电风暴**效果（小范围AOE） |
| Glinting Spark | 每次命中增加**1%**幸运一击概率，最多叠加**40%**，直至幸运一击触发后重置 |
| Flickering Spark | 每发火花有**10%**概率多弹射一次 |

**数值分析：**
- 4发中每发幸运一击概率15%，单次释放触发至少1次幸运一击的概率：
  1 - (1-0.15)^4 ≈ **47.8%**
- Glinting版的叠加幸运一击使火花成为幸运一击流法师的选择

**Build价值：** ★★★（利基价值，适合特定幸运一击Build）

## 4.4 法师核心技能详解

### 4.4.1 冰碎片（Ice Shards）

**基础效果（1点）：**
- 发射**5片**冰碎片，每片造成**25%**武器伤害
- 对处于冻结状态的目标：每片额外造成**25%**伤害
- 总伤害（vs冻结目标）：**5 × 50% = 250%**武器伤害
- 幸运一击概率：**50%**（每片，但通常不能全部命中同一目标）

**技能标签：** Core, Cold, Frost

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Ice Shards | 冰碎片击中冻结目标时，有**15%**几率立即重置技能 |
| Destructive Ice Shards（破坏冰碎片） | 冰碎片击中一个冻结目标后，碎裂成多个较小碎片，造成额外AOE伤害 |
| Icy Ice Shards（冰封冰碎片） | 冰碎片增加到**7片**（额外2片伤害各为20%），但消耗额外**5点**法力 |

**数值分析（5/5满级）：**
- 基础伤害：**5 × 45% = 225%**武器伤害/次
- vs 冻结目标（5/5）：**5 × 90% = 450%**武器伤害/次
- 法力消耗：**30点**/次
- Enhanced版重置概率：15% × 5（假设全中）= 约53%概率至少重置一次

**Build价值：** ★★★★★（冰法Build的绝对核心，与冻结机制协同极强）

---

### 4.4.2 火球（Fireball）

**基础效果（1点）：**
- 投掷爆炸火球，在目标区域爆炸造成**60%**武器伤害（AOE）
- 爆炸对灼烧状态的敌人：额外造成**10%**伤害
- 幸运一击概率：**30%**（命中范围内的每个目标）

**技能标签：** Core, Fire, Pyromancy

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Fireball | 火球命中精英或Boss时，立即重置一半冷却时间 |
| Destructive Fireball（破坏火球） | 火球命中已处于灼烧状态的敌人，爆炸范围扩大**30%**并造成额外**20%**伤害 |
| Greater Fireball（大火球） | 消耗法力+15，火球伤害提升**50%** |

**数值分析（5/5满级）：**
- 基础伤害：**108%**武器伤害（5/5）
- vs灼烧目标（Destructive，5/5）：**108% × 1.3 = 140.4%** + 更大范围
- Greater版：**108% × 1.5 = 162%** 但消耗更多法力

**Build价值：** ★★★★（通用火球流Build的核心，AOE能力强）

---

### 4.4.3 连锁闪电（Chain Lightning）

**基础效果（1点）：**
- 发射闪电弹射物，造成**36%**武器伤害，弹射最多**6次**
- 每次弹射使伤害提升**10%**
- 最终弹射（第6次）时伤害 = 36% × 1.5 = **54%**武器伤害
- 幸运一击概率：**25%**（每次弹射）

**技能标签：** Core, Lightning, Shock

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Chain Lightning | 命中精英或Boss时，召唤3个球状闪电绕玩家旋转 |
| Destructive Chain Lightning | 每个球状闪电爆炸时，在目标周围造成AOE闪电伤害 |
| Greater Chain Lightning | 弹射次数增加至**8次**，但每次弹射所需的目标切换距离增加 |

**数值分析（5/5满级）：**
- 理论最大弹射伤害（全6次命中同目标）：**约64.8% × 6 = 388.8%**（但不现实）
- 实际AOE场景下：**64.8% × (平均弹射次数)**
- Greater版8次弹射大幅提升AOE效率

**Build价值：** ★★★★（闪电流的核心技能，AOE和单体兼顾）

## 4.5 盗贼基础技能详解

### 4.5.1 穿心箭（Heartseeker）

**基础效果（1点）：**
- 远程攻击造成**24%**武器伤害
- 每次命中：提升暴击率**3%**，最多叠加10层（最大+**30%**暴击率）
- 持续时间：叠加层数在3秒内无命中时衰减
- 每次命中加速精力恢复
- 幸运一击概率：**50%**

**技能标签：** Basic, Physical, Ranged

**强化路径：**

| 版本 | 效果 |
|------|------|
| Enhanced Heartseeker | 暴击率叠加层数上限提升至**15层**（最大+45%暴击率） |
| Fundamental Heartseeker | 命中目标时额外生成**4点**精力 |
| Primary Heartseeker | 每次命中，下次使用的技能穿透+**1次**（可穿透多个目标） |

**数值分析：**
- 10层叠加（Enhanced版为15层）的暴击率加成使穿心箭在持续战斗中极为强大
- 与盗贼暴击流Build协同极强
- 精力生成效率高，是远程Build的首选基础技能

**Build价值：** ★★★★★（盗贼远程暴击流的核心基础技能）

---

### 4.5.2 穿刺（Puncture）

**基础效果（1点）：**
- 近战攻击造成**21%**武器伤害

## 4.8 基础技能与核心技能的协同法则

### 4.8.1 资源循环效率计算

以法师冰法为例：
```
冰霜弹（1次）：
  - 消耗：0（基础技能）
  - 收益：+10法力，减速目标

冰碎片（1次）：
  - 消耗：30法力
  - 收益：5×25%=125%武器伤害（一般）
          5×50%=250%武器伤害（vs冻结目标）

理论平衡点（冰法自然回复场景）：
  法力自然回复：17/秒
  冰碎片消耗：30/次
  不使用基础技能：约1.76秒/次冰碎片

  使用冰霜弹提速：
  每使用1次冰霜弹 → +10法力
  每秒约1次冰霜弹 → +10+17 = 27法力/秒
  冰碎片频率：27/30 ≈ 每1.1秒一次
  效率提升：1.76→1.1秒 ≈ 提升60%DPS
```

### 4.8.2 基础技能选择的原则

1. **与核心技能共享标签**：若核心技能是"冷系"，则选冷系基础技能，共享相关加成词缀
2. **最大化资源效率**：选择每次命中资源收益最高的基础技能
3. **满足控制需求**：某些Build需要特定控制效果来触发加成（如冻结状态下的冰碎片）
4. **强化节点协同**：某些基础技能的强化选项与核心技能高度协同

## 4.9 小结

基础技能和核心技能的选择是Build构建的第一步，也是影响最深远的选择。关键要点：

1. **基础技能不只是"充能器"**：它们的幸运一击、控制效果、被动增益在整体Build中扮演重要角色
2. **核心技能决定Build方向**：5/5满投的核心技能通常提供约180%效果（vs1/5的100%）
3. **强化二选一影响深远**：两个专精方向往往代表完全不同的游玩策略
4. **各职业数值差距悬殊**：同样是"核心技能"，不同职业的百分比从50%到250%不等，需要结合整体伤害公式理解
