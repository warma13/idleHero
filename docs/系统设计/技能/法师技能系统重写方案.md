# 法师技能系统重写方案

> **版本**: v1.1
> **日期**: 2026-03-25
> **状态**: 待审核
> **目标**: 将术士技能系统完全重写为D4式五元素法师技能系统

---

## 1. 设计理念

### 1.1 核心目标

- **五元素全覆盖**: 火焰/冰霜/剧毒/水系/奥术，每个元素拥有独立完整的技能路径
- **纯元素分支**: 不设通用分支，所有节点归属于具体元素，构建方向更清晰
- **D4式集群结构**: 参考暗黑4七层集群架构，适配挂机RPG的8行网格
- **有限取舍**: 总容量 ~179点，可用 ~120点，玩家选择2-3条主路线深入
- **元素协同**: 技能与现有元素反应系统(ElementReactions)深度联动
- **完全重写**: 不做旧ID映射，所有技能ID、名称、效果全部重新设计

### 1.2 D4 → 挂机RPG 适配表

| D4 特性 | 本游戏适配 | 说明 |
|---------|-----------|------|
| 手动施放6技能 | 自动施放所有已学主动技能 | 挂机RPG核心，按CD自动释放 |
| 法力100+17/s回复 | CD冷却管理 | 用CD替代资源消耗 |
| 附魔槽系统 | 暂不引入 | 作为未来扩展预留 |
| 7层×33点门槛 | 8行×行数决定消耗 | baseCost = floor(row/2)+1 |
| 终极/关键被动二选一 | 每元素独立终极 | 挂机RPG允许多终极共存 |
| 强化节点(菱形1点) | 子分支被动(圆形3点) | 保持现有升级深度 |
| 74点总量 | ~120点可用(每2级1点) | 与当前经济一致 |
| 六大伤害桶 | 六区乘算伤害管线 | 详见[第九章](第九章_技能伤害计算公式全解.md) |

### 1.3 与旧系统对比

| 维度 | 旧系统(术士) | 新系统(法师) |
|------|------------|------------|
| 分支数 | 3 (元素反应/攻击强化/群伤) | **5 (火焰/冰霜/剧毒/水系/奥术)** |
| 组织逻辑 | 按功能分 | **按元素分** |
| 技能总数 | ~60节点 | ~46节点 |
| 总容量 | ~200点 | ~179点 |
| 元素覆盖 | 散乱(火在群伤、冰在群伤、毒在群伤) | 每元素独立完整路径 |
| 主动技能 | 14个 | 15个(每元素3个主动+1终极，部分分支含额外主动) |
| 伤害公式 | 自定义 | D4六桶乘算体系（见[第九章](第九章_技能伤害计算公式全解.md)） |

### 2.3 伤害计算体系

技能伤害计算采用 D4 六桶乘算体系，完整公式与理论详见 **[第九章：技能伤害计算公式全解](第九章_技能伤害计算公式全解.md)**。

核心公式（简版）：

```
最终伤害 = 基础伤害 × 主属性乘数 × A伤乘数 × 暴击乘数 × 易伤乘数 × X伤乘数 + 压制
```

**各桶与技能被动的映射**：

| 伤害桶 | 桶内相加，跨桶相乘 | 技能被动示例 |
|--------|------------------|------------|
| 基础伤害 | ATK × 技能系数 | 由 Config.SKILLS.dmgScale 决定 |
| A伤（加法桶） | 元素伤害+X%、状态伤害+X% | 火焰亲和(+6%/lv)、焚烬增伤(+10%/lv) |
| 暴击桶 | 暴击率+X%、暴击伤害+X% | （装备/词缀提供，技能树暂不直接提供暴击） |
| X伤（独立乘） | 每个来源独立×1.X | 奥术穿透(无视抗性)、腐蚀削弱(减防) |
| 特殊机制 | DoT/召唤物/反应 | 灼烧蔓延(DoT扩散)、水精灵(召唤物伤害) |

> **设计约束**: 技能被动主要影响 A伤桶 和特殊机制，避免在技能树中直接提供X伤（X伤来源保留给装备词缀和传奇威能），维持装备与技能各自的价值空间。

## 4. 五元素技能详表

> - `maxLevel`: 被动=3-5, 基础/核心主动=5, 终极=3

> - `isActive=true`: 注册到Config.SKILLS自动施放

> - 伤害公式中的倍率含义详见 [第九章 §9.1.3](第九章_技能伤害计算公式全解.md)
