# 第二章：主属性系统（力量/敏捷/意志/智慧）深度解析

## 2.1 主属性的双重作用机制

暗黑4的四大主属性（力量、敏捷、意志、智慧）每一项都承担**两种功能**：

1. **通用效果**：对所有职业生效的基础属性提升
2. **职业专属效果**：对特定职业的主力输出属性提升（即"主力属性伤害加成"）

这种设计使得不同职业的属性优先级截然不同：法师堆智慧同时提升抗性和输出，而野蛮人堆力量同时提升护甲和输出。

---

## 2.2 力量（Strength）详解

### 2.2.1 通用效果

**护甲计算：**

```
护甲值增量 = 力量点数 × 1
```

即每1点力量 = +1点护甲值。

护甲对减伤的转化公式（见第三章详述）：
```
护甲减伤% = 护甲值 / (护甲值 + 怪物等级参考值)
```

**实际价值估算**（100级T4环境，怪物参考值约11,000）：
- 力量从1,000增加100点 → 护甲1,000→1,100 → 减伤从8.33%→9.09% → 净增0.76%减伤
- 力量从5,000增加100点 → 护甲5,000→5,100 → 减伤从31.25%→31.67% → 净增0.42%减伤

**边际收益递减明显**，高护甲时追加力量的减伤收益下降。

### 2.2.2 职业专属效果（野蛮人、德鲁伊）

**野蛮人**：力量是主力输出属性
```
主属性伤害加成(%) = (力量 × 0.1%)
```
即100点力量 = +10%伤害加成（放入A伤桶）

**计算示例**：
- 野蛮人力量1,000点 → +100%伤害（A伤桶+100%）
- 野蛮人力量2,000点 → +200%伤害（A伤桶+200%）

**德鲁伊**：力量仅提供护甲，无输出加成（德鲁伊主力属性为意志）。

### 2.2.3 力量的获取来源

| 来源 | 典型数值 |
|------|---------|
| 角色基础值（100级） | 约210点（10基础+2/级×100） |
| 声望奖励（全区域） | 最多+15点（可分配） |
| 装备词缀"力量" | +100~300点/件（IP925祖先装备） |
| 装备词缀"全主属性" | +60~150点/件（每个主属性均+） |
| 技能被动 | +5~30点/节点 |
| 魔典板节点 | +5~20点/节点 |

**100级野蛮人满配力量参考**：
- 角色基础：约210点
- 装备（6~8件含力量词缀）：约1,200~2,400点
- 被动+魔典：约100~200点
- **合计：约1,510~2,810点** → 对应主属性伤害+151%~281%

## 2.3 敏捷（Dexterity）详解

### 2.3.1 通用效果

**暴击率加成：**

```
暴击率增量 = 敏捷点数 × 0.02%
```

即每50点敏捷 = +1%暴击率。

**暴击率上限**：100%（无意义超出）。

**敏捷对暴击率的边际价值**（基于期望伤害计算）：

```
期望暴击系数 = 1 + 暴击率 × (暴击伤害% - 1)
```

假设暴击伤害=300%（即×3）：
- 暴击率0%：期望系数 = 1 + 0×2 = 1.0
- 暴击率50%：期望系数 = 1 + 0.5×2 = 2.0
- 暴击率100%：期望系数 = 1 + 1.0×2 = 3.0

**每1%暴击率在此条件下价值**：+2%期望伤害（与暴击伤害正相关）

### 2.3.2 职业专属效果（盗贼、灵魂收割者）

**盗贼**：敏捷是主力输出属性
```
主属性伤害加成(%) = 敏捷 × 0.1%
```
即1,500点敏捷 = +150%伤害（A伤桶）

**灵魂收割者**：
```
主属性伤害加成(%) = 敏捷 × 0.1%
```
与盗贼相同公式。

### 2.3.3 敏捷与暴击率的联合优化

对于盗贼，敏捷同时提供：
1. A伤桶直接加成（每100点=+10%）
2. 暴击率间接加成（每100点=+2%暴击率）

**盗贼1,500敏捷时**：
- A伤直接贡献：+150%（A伤桶）
- 暴击率贡献：+30%（在装备基础暴击率上叠加）

**暴击期望价值**（假设暴击伤害=400%）：
- 暴击率40%（装备）→ 期望系数 = 1+0.4×3 = 2.2
- 新增+30%（敏捷）→ 暴击率70% → 期望系数 = 1+0.7×3 = 3.1
- 暴击期望提升：3.1/2.2 = 约+40.9%

## 2.4 意志（Willpower）详解

### 2.4.1 通用效果

意志提供两项通用效果：

**① 治疗效果加成：**
```
治疗效果加成% = 意志 × 0.1%
```

影响：
- 生命药水回复量
- 技能治疗（如德鲁伊的卵石皮肤治疗、死灵的血涌治疗）
- 再生类效果（如生命再生/秒）

**② 资源生成加成：**
```
资源生成加成% = 意志 × 0.1%
```

影响各职业资源恢复速度。

**重要说明**：意志的资源生成加成对**再生型资源**（魔力、精力）和**生成型资源**（狂怒、精华）的影响不同：
- 再生型：加快每秒自然恢复量
- 生成型：加快技能击中时的资源生成量（每次生成×(1+意志加成%)）

### 2.4.2 职业专属效果（德鲁伊、死灵、灵魂收割者）

**德鲁伊**：意志是主力输出属性
```
主属性伤害加成(%) = 意志 × 0.1%
```

**死灵法师**：意志**不是**主力输出属性（主力为智慧），但意志影响技能治疗量，死灵部分Build依赖血涌消耗HP治疗，因此意志有间接价值。

**灵魂收割者**：意志不是主力输出属性（主力为敏捷），但意志加快灵魂（Vigor）资源生成，对灵魂收割者的爆发窗口有重要意义。

### 2.4.3 意志的特殊价值：堡垒生命互动

```
堡垒生命上限 = 最大生命值 × (1 + 意志治疗加成%)
```

当堡垒生命值高于当前生命时，角色获得：
- 减伤10%（vs所有伤害）
- 免疫"被压制"效果

意志通过提升治疗效果，间接加快堡垒生命的积累速度。

## 2.5 智慧（Intelligence）详解

### 2.5.1 通用效果

**所有元素抗性加成：**
```
元素抗性% = 智慧 × 0.05%（上限70%）
```

即每20点智慧 = +1%所有元素抗性。

**抗性的减伤价值**（以火焰伤害为例）：
- 抗性0% → 承受100%火焰伤害
- 抗性50% → 承受50%火焰伤害（减少50%）
- 抗性70%（上限）→ 承受30%火焰伤害

在折磨T4中，怪物有+10%~+30%穿透抗性的变种（参见第八章），因此满抗性70%实际只有40%~60%效果。

### 2.5.2 职业专属效果（法师、死灵）

**法师**：智慧是主力输出属性
```
主属性伤害加成(%) = 智慧 × 0.1%
```

**死灵法师**：
```
主属性伤害加成(%) = 智慧 × 0.1%（含骨系技能）
```

**死灵双属性权衡**：
- 智慧 → 直接输出+抗性
- 意志 → 治疗效果+资源生成（部分召唤死灵依赖意志）

**纯骨刺Build**：优先智慧（输出 > 生存）
**召唤死灵Build**：意志与智慧各约50点装备配比，兼顾随从生存

### 2.5.3 智慧与抗性上限的关系

100级角色基础智慧约210点 → 抗性+10.5%（各元素）

**装备配合**：
- 一件IP925装备的智慧词缀：约+200~300点智慧 → +10%~15%抗性
- 6~8件装备合计：约+60%~90%智慧带来的抗性
- 加上基础，接近或超过70%上限 → 超出部分无效

**结论**：智慧在接近70%抗性上限后，其抗性价值归零，仅剩输出价值（仅法师/死灵）。

## 2.6 全主属性（All Stats）词缀

部分装备携带"全主属性"词缀，同时提升四种主属性：

| 装备类型 | IP 700普通 | IP 850血腥 | IP 925祖先 |
|---------|-----------|-----------|-----------|
| 全主属性范围 | +25~60 | +50~90 | +80~150 |

**价值计算**（以法师为例，全主属性+100为例）：
- 力量+100：+100护甲（约0.5%~1%减伤，价值低）
- 敏捷+100：+2%暴击率（价值中等）
- 意志+100：+10%治疗/资源生成（价值中等）
- 智慧+100：+5%全抗性（若未达上限，价值高）+A伤+10%（法师）

**全主属性词缀效率**比单一主属性词缀低（因为力量和意志对法师贡献有限），但在属性全面不足时有价值。

## 2.7 主属性与魔典板的协同

魔典板（Paragon Board）中的传奇节点通常需要满足主属性阈值才能激活：

**常见激活条件示例**：

| 传奇节点名称 | 激活条件 | 激活效果 |
|------------|---------|---------|
| 野兽之心（德鲁伊） | 附近有≥40点力量节点 | 变身时+10%技能伤害 |
| 骨髓深处（死灵） | 附近有≥25点智慧节点 | 骨系技能+15%暴击 |
| 闪电反应（法师） | 附近有≥35点智慧节点 | 闪电技能+10%A伤 |
| 战场统治（野蛮人） | 附近有≥50点力量节点 | 命中时+8%压制触发率 |

这一机制使得主属性的价值在魔典板阶段再次拔高——不仅提供直接数值，还激活传奇节点效果。

## 2.8 各职业主属性优先级总结

### 野蛮人

```
优先级：力量 > 意志（堡垒/资源）≥ 敏捷（暴击）> 智慧（仅抗性）
装备配比建议：力量70% / 意志15% / 敏捷10% / 智慧5%
```

### 德鲁伊

```
优先级：意志 > 力量（护甲/变形）≥ 智慧（抗性）> 敏捷
装备配比建议：意志65% / 力量20% / 智慧10% / 敏捷5%
```

### 死灵法师

```
骨刺Build：智慧 > 意志 > 力量 ≥ 敏捷
召唤Build：意志 ≥ 智慧 > 力量 > 敏捷
装备配比建议（骨刺）：智慧60% / 意志25% / 力量10% / 敏捷5%
```

### 盗贼

```
优先级：敏捷 > 意志（资源回复）> 力量（护甲）≥ 智慧
装备配比建议：敏捷70% / 意志15% / 力量10% / 智慧5%
```

### 法师

```
优先级：智慧 > 意志（资源）> 敏捷（暴击）> 力量
装备配比建议：智慧70% / 意志15% / 敏捷10% / 力量5%
```

### 灵魂收割者

```
优先级：敏捷 > 意志（灵魂生成）> 力量（护甲）≥ 智慧
装备配比建议：敏捷65% / 意志20% / 力量10% / 智慧5%
```

## 2.9 主属性的"软天花板"现象

在T4终局装备阶段，由于装备可提供大量主属性词缀，存在"软天花板"现象：

**以法师智慧为例**：
- 70%抗性上限：智慧≥1,400点后，抗性价值归零
- A伤桶边际递减：当A伤总量已超300%时，新增10%的收益远低于换一件X伤词缀装备
- 魔典板激活后：传奇节点已激活，主属性继续堆叠的魔典价值也降低

**结论**：主属性并非越多越好，存在最优配点窗口，超过窗口后应转换到其他属性维度。

## 2.10 本章小结

四大主属性的核心要点：

| 属性 | 通用功能 | 主力输出职业 | 核心公式 |
|------|---------|------------|---------|
| 力量 | +1护甲/点 | 野蛮人、德鲁伊 | 护甲+力量×1，A伤+力量×0.1% |
| 敏捷 | +0.02%暴击率/点 | 盗贼、灵魂收割者 | 暴击率+敏捷×0.02%，A伤+敏捷×0.1% |
| 意志 | +0.1%治疗和资源生成/点 | 德鲁伊 | 治疗/资源效率+意志×0.1%，A伤+意志×0.1% |
| 智慧 | +0.05%全抗性/点 | 法师、死灵 | 全抗性+智慧×0.05%（上限70%），A伤+智慧×0.1% |

**设计精妙之处**：每个职业的主力属性同时提升"攻击"和"防御/功能性"的两项能力，使得属性分配不再是纯输出vs纯生存的非此即彼选择，而是形成"攻防一体"的协同增益。
