# 第四章：攻击属性与伤害计算完整公式

## 4.1 伤害计算的六桶模型综述

暗黑4的伤害系统采用"六桶模型"（Six Damage Buckets），这是理解所有输出计算的核心框架。

```
最终伤害 = 武器伤害
         × 技能系数(%)
         × 主属性修正(A伤桶-主属性)
         × (1 + ΣA伤)           ← 加法桶（A伤桶-装备技能）
         × 暴击期望              ← 暴击桶
         × (1 + 易伤加成)        ← 易伤桶
         × Π(1 + Xi)            ← X伤乘法桶
         + 压制伤害（3%概率触发）  ← 压制桶（独立加算）
```

每个桶的内部是**加法叠加**，不同桶之间是**乘法叠加**。正确理解这点是所有Build优化的数学基础。

## 4.2 武器伤害（Weapon Damage）

### 4.2.1 武器DPS与面板伤害

武器属性面板显示两个关键数字：
1. **武器基础伤害范围**：如"1,200~1,800"
2. **武器DPS**：= 平均伤害 × 攻击速度

```
武器平均伤害 = (最小伤害 + 最大伤害) / 2
武器DPS = 武器平均伤害 × 攻击速度(次/秒)
```

**技能伤害使用"武器平均伤害"而非DPS**：

```
技能基础伤害 = 武器平均伤害 × 技能系数
```

DPS仅用于参考输出强度，实际单次伤害以平均伤害×系数计算。

### 4.2.2 IP（物品强度）与武器伤害的关系

物品强度（Item Power，IP）直接决定武器伤害范围：

| IP区间 | 武器平均伤害（参考一手武器） |
|--------|------------------------|
| IP 1~100 | 5~50 |
| IP 200 | 200~400 |
| IP 400 | 600~900 |
| IP 625（T3解锁） | 1,200~1,600 |
| IP 700（T4基础） | 1,500~2,200 |
| IP 800 | 2,000~3,000 |
| IP 850（血腥） | 2,500~3,500 |
| IP 925（祖先，T4满配） | 3,000~5,000 |

**带崇高词缀（GA）的武器**：
- GA武器伤害词缀约×1.5倍 → IP925 GA武器可达4,500~7,500伤害

**双手武器 vs 单手武器**：
- 双手武器：基础伤害约×1.6~×2.0，但攻速约-30%~-50%
- 最终DPS相近，但单次伤害（爆发）更高

### 4.2.3 武器伤害词缀

除了武器本身携带的固有伤害，武器还可有以下提升词缀：

**① 武器伤害%（仅出现在武器上）**：
```
武器平均伤害 × (1 + 武器伤害%)
```
- 典型范围：IP925祖先武器 +40%~80%
- GA武器伤害词缀：+70%~120%

**② 核心技能伤害%（装备词缀）**：
不直接提升武器伤害，而是进入A伤桶（见4.4节）。

### 4.3.3 技能系数与技能等级的关系

技能每升一级，系数按固定比率增加：

```
技能系数(n级) = 技能系数(1级) × (1 + (n-1) × 单级增幅)
```

大多数技能的单级增幅约为**+5%~+8%**（技能系数数值本身的百分比，非A伤加成）。

例：骨刺1级=120%，5级=153%，单级增幅=约(153-120)/(120×4)=约6.9%/级。

**装备词缀"+X级技能"的价值**：
- 骨刺每+1级 ≈ +6.9%技能系数
- 在基础DPS框架下，+5级骨刺≈+34.5%系数（A伤桶之外的独立乘法）
- 这是为何装备"骨刺+5级"词缀价值极高的原因

## 4.4 A伤桶（加法伤害桶）详解

### 4.4.1 A伤桶的组成

A伤桶内的所有加成**相加合并**后，整体作为一个乘法系数：

```
A伤桶总修正 = 1 + (主属性A伤%) + ΣA伤词缀%
```

以下内容进入A伤桶：

**① 主属性A伤（来自主力属性）**：
```
主属性A伤% = 主力属性点数 × 0.1%
```

**② 技能类型伤害加成**：
- "+X%核心技能伤害"
- "+X%基础技能伤害"
- "+X%终极技能伤害"
- "+X%召唤物伤害"
- "+X%DoT伤害"

**③ 状态效果伤害加成**（同一桶）：
- "+X%对被冰冻敌人的伤害"
- "+X%对燃烧中敌人的伤害"
- "+X%对被击晕敌人的伤害"

**④ 近战/远程伤害加成**：
- "+X%近战伤害"
- "+X%远程伤害"

### 4.4.2 A伤桶的边际收益递减

```
边际增益 = 新增A伤% / (1 + 已有A伤总量%)
```

示例：已有A伤=200%，新增+50%A伤：
```
边际增益 = 50% / (1 + 200%) = 50% / 300% = 16.7%实际提升
```

若A伤=500%时，同样新增+50%：
```
边际增益 = 50% / (1 + 500%) = 50% / 600% = 8.3%实际提升
```

**A伤越多，新增A伤价值越低。** 这是为何高配Build通常优先补X伤（乘法桶）而非继续堆A伤的原因。

---

## 4.5 X伤桶（乘法伤害桶）详解

### 4.5.1 X伤的特点

X伤词缀（传奇词缀和特定稀有词缀）进入独立的乘法桶，各词缀间**相乘**：

```
X伤总修正 = Π(1 + Xi%)
```

每个X伤词缀对现有所有A伤的总基础进行独立乘法，不存在边际递减（除非与其他X伤交互）。

### 4.5.2 常见X伤词缀举例

| 词缀/传奇名称 | X伤数值 | 类型 |
|------------|--------|------|
| 致命毒素（盗贼传奇） | ×1.3（+30%） | 对中毒敌人 |
| 压制震撼（野蛮人） | ×1.4（+40%） | 压制触发时 |
| 永恒冬天（法师） | ×1.2（+20%） | 对冰冻敌人 |
| 激烈火焰（法师） | ×1.25（+25%） | 焚烧状态中 |
| 神圣之刃（盗贼） | ×1.3（+30%） | 对暗影灌注后 |
| 呼朋引伴（德鲁伊） | ×1.35（+35%） | 有随从时 |

### 4.5.3 X伤与A伤的最优分配

**最优均衡原理**：当各桶数值相等时，总DPS最大化。

若：
- 武器伤害 = W
- A伤桶 = (1+A)
- X伤桶1 = (1+X1)
- X伤桶2 = (1+X2)

则总伤害 = W × (1+A) × (1+X1) × (1+X2)

当 (1+A) = (1+X1) = (1+X2) 时，在总"数值预算"固定的前提下，乘积最大。

**实际应用**：A伤总量>X伤总量时，应优先提升X伤；X伤总量>A伤总量时，应优先提升A伤。

## 4.6 暴击系统完整计算

### 4.6.1 暴击率基础

```
暴击率 = 5%（基础）+ 敏捷×0.02%（仅敏捷职业） + 装备暴击率词缀 + 技能被动暴击加成
```

**装备暴击率词缀范围**：
- IP700~850：+3%~8%/件
- IP925祖先：+5%~12%/件
- 首饰（戒指/项链）暴击率词缀价值最高：可达+10%~15%/件

### 4.6.2 暴击伤害基础

```
暴击伤害 = 150%（基础）+ 装备暴击伤害词缀 + 技能被动暴击伤害加成
```

注意：150%暴击伤害意味着暴击 = 基础伤害 × 1.5（即额外+50%，而非×150%）。

**暴击期望伤害系数**：
```
暴击期望系数 = 1 + 暴击率 × (暴击伤害% - 100%)
            = 1 + 暴击率 × 超额暴击伤害%
```

例：暴击率=60%，暴击伤害=400%（即基础的4倍）：
```
暴击期望系数 = 1 + 0.6 × (4.0 - 1.0) = 1 + 0.6 × 3 = 2.8
```
平均伤害 = 基础伤害 × 2.8（约为无暴击的2.8倍）

### 4.6.3 暴击率vs暴击伤害的最优点

固定"暴击类属性总预算"时，何时应堆暴击率、何时应堆暴击伤害？

**最优条件**：
```
dDPS/d(暴击率) = dDPS/d(暴击伤害)

(暴击伤害-100%) = (1/暴击率 - 1) × 100%
```

**换算**：若暴击率=30%，则最优暴击伤害=300%（基础）；若暴击率=60%，最优暴击伤害约=（1/0.6-1)×100% + 100% = 66.7% + 100% = 166.7%——

更准确的结论是：**在暴击率较低时（<50%），应优先提升暴击率；在暴击率较高时（>60%），应优先提升暴击伤害。**

## 4.7 完整DPS计算算例

### 4.7.1 案例：T4野蛮人重击Build

**装备数据**：
- 双手战锤：平均伤害4,200，攻速0.9/秒
- 主属性（力量）：2,200点
- A伤词缀（核心技能+250%，近战+80%，精英+50%，速度+40%）
- X伤词缀（压制+45%，近战X伤+35%）
- 暴击率：65%
- 暴击伤害：520%
- 是否易伤目标：是（+35%）
- 重击技能系数（5级）：155%

**计算过程**：

```
① 武器基础伤害：4,200

② 技能系数：× 155% = 6,510

③ 主属性A伤：2,200 × 0.1% = 220%
   A伤词缀汇总：220% + 250% + 80% + 50% + 40% = 640%
   A伤桶修正：1 + 640% = 7.40

   伤害后：6,510 × 7.40 = 48,174

④ 暴击期望：
   暴击超额 = 520% - 100% = 420%
   期望系数 = 1 + 0.65 × 4.20 = 1 + 2.73 = 3.73
   伤害后：48,174 × 3.73 = 179,729

⑤ 易伤桶（目标易伤）：× (1 + 35%) = × 1.35
   伤害后：179,729 × 1.35 = 242,634

⑥ X伤桶（乘法）：× (1+45%) × (1+35%) = × 1.45 × 1.35 = × 1.958
   伤害后：242,634 × 1.958 = 475,275

最终单次重击期望伤害 ≈ 475,000
```

每秒DPS（仅重击部分，0.9次/秒）：
```
重击DPS = 475,275 × 0.9 = 约427,748/秒
```

加上旋风斩等持续输出，总DPS可达600万~800万/秒（T4精英30%减伤后实际伤害约350万~560万/秒，基本符合T4中可清精英的DPS要求）。

## 4.8 武器专精系统（野蛮人武器绑定）

### 4.8.1 武器绑定规则

野蛮人可装备4件武器（2件单手近战、1件双手劈砍、1件双手挥击）。每个技能必须绑定其中一件武器，使用时以绑定武器的基础伤害计算：

```
技能伤害 = 绑定武器平均伤害 × 技能系数 × ...
```

### 4.8.2 "野蛮的"前缀技能（Furious）

部分强化选项使技能在攻击时**额外触发**其他武器的效果：

```
激怒旋风斩（Furious Whirlwind）：
每次旋风斩命中，对绑定武器以外的所有武器各触发一次武器攻击加成
额外加成 = 其他武器总攻击%加成 × 旋风命中次数
```

**数值示例**：
- 旋风斩绑定单手剑（平均3,000伤害）
- 双手劈砍（平均6,000，附有+15%火焰伤害词缀）
- 双手挥击（平均5,500，附有+18%流血伤害词缀）
- 另一单手武器（平均2,800，+12%精英伤害）

激怒旋风斩触发额外效果：
```
火焰加成激活：+15%（进入A伤桶）
流血加成激活：+18%（进入A伤桶）
精英加成激活：+12%（进入A伤桶）
```
相当于合计多了+45%A伤，同时只需绑定单手武器（攻速更快）。

## 4.9 DoT（持续伤害）计算特殊规则

### 4.9.1 快照机制（Snapshot）

部分DoT在触发时"快照"当前状态：

```
DoT每跳伤害 = (快照时刻的伤害值) × DoT系数 / 跳数
```

**快照要素**：
- 触发时的A伤总量（固定）
- 触发时的易伤状态（目标是否易伤）
- 触发时的BUFF状态（狂暴/战吼等）
- 触发时的主属性值

快照后，DoT期间上述数值变化**不影响**已触发的DoT。

### 4.9.2 DoT刷新与叠加规则

| DoT类型 | 叠加规则 |
|--------|---------|
| 流血（Bleeding） | 最多可叠加5层，各层独立计时 |
| 焚烧（Burning） | 新触发刷新持续时间，不叠层（同来源）；不同来源可叠加 |
| 中毒（Poisoning） | 与焚烧类似，可叠加但有上限 |
| 诅咒DoT（死灵） | 独立层，每次施放叠层（上限约10层） |

## 4.10 本章小结

核心伤害计算公式全览：

```
期望DPS = (武器DPS × 技能系数)
        × (1 + 主属性A伤 + ΣA伤词缀)   ← A伤桶（加法）
        × (1 + CR × (CD% - 100%))       ← 暴击期望
        × (1 + 易伤加成%)               ← 易伤桶
        × Π(1 + Xi%)                   ← X伤乘法
        + 3% × 压制伤害期望值           ← 压制（每次攻击期望）
```

关键设计原则：
1. A伤桶内加法叠加，边际收益递减
2. X伤桶间乘法叠加，边际收益不递减（直至与A伤失衡）
3. 暴击率×暴击伤害是独立的高价值桶
4. 易伤目标时，易伤加成是独立乘法
5. 武器系数决定上限，技能系数决定基础，装备修正放大规模
