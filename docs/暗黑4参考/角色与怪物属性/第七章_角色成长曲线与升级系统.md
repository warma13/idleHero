# 第七章：角色成长曲线与升级系统

## 7.1 经验值与等级系统

### 7.1.1 经验值总量与分布

暗黑4从1级升至100级，总经验需求约**2.7亿经验**（标准难度WT3/WT4环境）。

各等级段所需经验（近似值）：

| 等级区间 | 单级经验需求 | 区间总经验 | 累计经验 |
|---------|------------|---------|---------|
| 1~20 | 500~5,000 | 约5万 | 约5万 |
| 21~40 | 5,000~50,000 | 约55万 | 约60万 |
| 41~60 | 50,000~500,000 | 约550万 | 约610万 |
| 61~80 | 500,000~5,000,000 | 约5,500万 | 约6,100万 |
| 81~100 | 5,000,000~50,000,000 | 约2.1亿 | 约2.7亿 |

**关键结论**：后20级（81~100）占总经验量的近80%，是"最后一公里"最漫长的部分。

### 7.1.2 经验倍率

不同难度和条件下的经验倍率：

| 条件 | 经验倍率 |
|------|---------|
| 世界层级1（WT1，正常） | ×1.0 |
| 世界层级2（WT2，噩梦） | ×1.5 |
| 世界层级3（WT3，折磨一） | ×2.5 |
| 世界层级4（WT4，折磨二） | ×4.0（折磨T1-T4详细见下） |
| 折磨T1 | ×4.0 |
| 折磨T2 | ×6.0 |
| 折磨T3 | ×8.0 |
| 折磨T4 | ×12.0 |
| 四人组队 | 追加×1.0~×2.0（每人经验各自计算，但怪物HP提高） |
| 精英怪 | ×3.0~×5.0（相较同级普通怪） |
| 噩梦地下城签文加成 | +5%~+30%额外经验（签文词缀影响） |
| 经验祝福（战斗通行证） | +10%~+20% |

**最优升级效率（95~100级）**：
- 折磨T4 + 噩梦地下城T30 + 四人组 → 约31.2亿/小时
- 所需时间：2.1亿总经验（81-100段）/ 2.7亿/小时 ≈ 约45~50分钟（速推情况下）

### 7.1.3 升级奖励

每次升级获得：

| 内容 | 数量 |
|------|------|
| 技能点 | +1（每级一点，1~49级为技能树技能点，50级后为魔典点） |
| 属性点（含基础属性成长） | 自动应用，无手动分配（除声望点） |
| 解锁内容 | 特定等级解锁新玩法（见下） |

**关键解锁等级**：

| 等级 | 解锁内容 |
|------|---------|
| 1 | 正常游戏开始 |
| 25 | 声望系统全面开放 |
| 50 | 进入WT3可能性（需完成剧情） |
| 70 | 传奇和独特装备开始高频率掉落 |
| 100 | 进入魔典板系统（Paragon Board） |

## 7.2 主属性成长曲线

### 7.2.1 等级自动成长

角色无需手动分配属性，每升一级自动获得属性成长。以下是近似成长数据（各职业略有差异）：

**主力属性成长**（以野蛮人力量为例）：
```
每级成长量 = 约2.0点
100级力量（仅基础成长）= 10 + 2.0×99 = 约208点
```

**所有主属性成长比率**（主力属性略高）：

| 职业 | 主力属性/级 | 次要属性/级 |
|------|-----------|-----------|
| 野蛮人 | 力量 +2.5/级 | 其余 +1.5/级 |
| 德鲁伊 | 意志 +2.5/级 | 其余 +1.5/级 |
| 死灵 | 智慧 +2.5/级 | 其余 +1.5/级 |
| 盗贼 | 敏捷 +2.5/级 | 其余 +1.5/级 |
| 法师 | 智慧 +2.5/级 | 其余 +1.5/级 |
| 灵魂收割者 | 敏捷 +2.5/级 | 其余 +1.5/级 |

**100级主力属性基础值**（仅等级成长，不含装备）：
```
主力属性 = 基础值(1级) + 2.5×99 ≈ 255点
次要属性 ≈ 基础值(1级) + 1.5×99 ≈ 155点
```

对输出的影响（以野蛮人为例，255点力量）：
- 护甲：+255
- A伤：+255×0.1% = +25.5%（仅靠基础成长的主属性A伤）

## 7.3 技能点系统（1~49级）

### 7.3.1 技能点分配

1~49级，每级获得1个技能点，合计49个技能点。

**技能树解锁门控**（各级技能行解锁条件）：

| 技能行 | 解锁所需前行点数 | 对应技能等级 |
|--------|--------------|------------|
| 第一行（基础技能） | 0 | 1~49任意分配 |
| 第二行（核心技能） | 需第一行≥2点 | — |
| 第三行（防御/召唤） | 需前两行≥6点 | — |
| 第四行（辅助） | 需≥11点 | — |
| 第五行（Key/专精） | 需≥16点 | — |
| 第六行（终极技能） | 需≥23点 | — |
| 第七行（关键被动） | 需≥33点 | — |

**关键被动**要求至少33点已花费，在49点中除去路上的"垫点"（连接不同行所必须花费的点），实际有效技能点约为35~40点。

### 7.3.2 声望技能点

完成各区域的声望任务（共5个区域）后，可额外获得：
- 每区域：+1个技能点（在1级进入技能点池）
- 共5个区域 → **+5个额外技能点**

因此技能总点数 = 49（升级）+ 5（声望）= **54点技能点**。

但关键被动的33点门槛不计声望点，仅计系统技能点。

## 7.4 魔典板系统（100级后）

### 7.4.1 魔典系统概述

角色达到100级后，开始获得魔典经验（Paragon Experience），每累积一定魔典经验获得1个魔典点。

**魔典点总获取数量**：
- 标准：每4点普通魔典经验获得1点（近似）
- 随着游戏版本演进，魔典点上限约为**200~220个**（具体随赛季调整）

### 7.4.2 魔典板布局

**起始板（每职业专属）**：
- 4×4网格 = 16个节点
- 中心节点为职业专属效果

**扩展板**：
- 从起始板边缘连接新板块
- 每个扩展板约25~30节点
- 通过"门户节点"（Legendary Node）激活特定扩展板
- 最多可激活约6~8个扩展板

**总节点数**：约170~220个节点（不同职业/赛季有差异）

### 7.4.3 魔典节点类型与数值

**① 普通节点（白色/灰色）**：
- +5~10点主属性 / +2~5生命 / +1~2%小幅属性
- 数量最多，是"通路"节点

**② 魔术节点（蓝色）**：
- +25~50点主属性 / +5~10%某属性加成
- 需要相邻才能激活

**③ 稀有节点（黄色）**：
- +专精属性（如+10%易伤伤害 / +暴击伤害15%）
- 通常需要满足"附近区域点数"条件才能激活

**④ 传奇节点（橙色）**：
- 特殊机制加成（如：命中后触发特效、状态复制等）
- 每块板最多1个
- 激活条件：该板已放置足够点数（通常10~15点）

**⑤ 插槽节点（Socket）**：
- 可插入雕文（Glyph）
- 激活雕文效果，并激活插槽周围区域的雕文奖励

### 7.4.4 雕文（Glyph）系统

雕文是可升级的宝石，插入魔典板的插槽节点后激活：

```
雕文效果 = 基础效果（1级）× (1 + 升级系数 × 雕文等级)
雕文区域加成 = 以插槽为中心的2格范围内，满足类型要求的节点数 × 区域加成系数
```

**雕文等级**：通过在Pit裂缝中完成关卡获得经验升级
- 最高等级：21级
- 升级路径：每次Pit通关完成后，雕文获得固定经验

**雕文等级对效果的影响**：

| 雕文等级 | 基础效果加成 | 区域触发条件 |
|---------|------------|-----------|
| 1 | 100% | 需区域内≥5点（满足类型） |
| 10 | 约200% | 需区域内≥8点 |
| 21 | 约350% | 需区域内≥12点 |

**典型雕文示例（野蛮人"毁灭者"雕文）**：
- 1级：每点力量主属性节点+0.02%近战伤害（区域内力量节点总加成）
- 21级：每点力量主属性节点+0.07%近战伤害
- 若区域内有30点力量节点：21级雕文提供+30×0.07% = **+2.1%近战伤害**

看似小，但整块魔典板上多个雕文叠加，总计可贡献+20%~+50%的X伤（乘法）或A伤。

## 7.5 声望系统（Renown）

### 7.5.1 声望系统结构

游戏世界分为5个大区域：
1. 断骨荒野（Fractured Peaks）
2. 旱谷（Dry Steppes）
3. 哈维扎尔（Hawezar）
4. 迷宫之地（Kehjistan）
5. 塑魂之地（Scosglen）

每个区域的声望通过以下方式积累：
- 完成支线任务：+20声望
- 发现地区（探图）：+5声望
- 解锁地下城：+30声望
- 完成地区事件：+5~+20声望
- 发现藏室（Cellar）：+5声望

### 7.5.2 声望奖励

每个区域达到特定声望时解锁奖励：

| 声望里程碑 | 奖励 |
|----------|------|
| 第1阶段 | +1技能点（加入技能点池） |
| 第2阶段 | +1魔典点（加入魔典点池） |
| 第3阶段 | +约80骑士团徽章（赛季货币） |
| 第4阶段 | +1技能点（加入技能点池） |
| 第5阶段 | +1魔典点（加入魔典点池） |

5个区域全部满级（每区域5阶段）：
- 总额外技能点：5区域 × 2 = **+10技能点**
- 总额外魔典点：5区域 × 2 = **+10魔典点**

**注意**：声望奖励是**账户级别共享**，换角色后在同赛季内无需重新收集（但新赛季会重置）。

## 7.6 角色成长的完整数值模型

### 7.6.1 从1级到100级的属性积累总览

以野蛮人为例，纯靠等级成长（不含装备）的属性积累：

| 属性 | 1级 | 50级 | 100级（等级成长） |
|------|-----|------|----------------|
| 力量 | 10 | 10+2.5×49=132 | 10+2.5×99=258 |
| 护甲（力量部分） | 10 | 132 | 258 |
| 暴击率 | 5% | 5% | 5%（力量不加暴击） |
| A伤（力量部分） | 1% | 13.2% | 25.8% |
| 最大生命（基础） | 85 | 85+5×49=330 | 85+5×99=580 |

**关键观察**：纯靠等级成长的属性极为有限，装备才是数值的主要来源（约占总属性的80%~90%）。

### 7.6.2 装备对总属性的贡献比例

以100级满配T4野蛮人为例：

| 属性 | 等级成长贡献 | 装备/技能贡献 | 比例（装备:等级） |
|------|-----------|------------|----------------|
| 力量 | 258点 | ~2,400点 | 约90%:10% |
| 最大生命 | 580基础 | ~15,000以上 | 约96%:4% |
| 护甲 | 258 | ~12,000 | 约98%:2% |
| A伤（主属性贡献） | 25.8% | 240%（装备力量） | 约90%:10% |

**结论**：角色数值约90%来自装备，等级本身的贡献主要体现在技能解锁而非属性数值。

### 7.6.3 DPS随等级的增长曲线

以法师骨刺为参考，不同阶段DPS数量级：

| 阶段 | 装备IP | 单次技能期望伤害 | 备注 |
|------|--------|--------------|------|
| 1~20级（新手） | <100 | 约100~500 | 学习阶段 |
| 21~50级（成长） | 100~400 | 约1,000~10,000 | 主线推进 |
| 51~70级（T3入门） | 400~700 | 约5万~30万 | T3清图 |
| 71~90级（T4初期） | 700~850 | 约50万~300万 | T4初进 |
| 91~100级（T4成型） | 850~925 | 约300万~2,000万 | T4清图 |
| 100级魔典满（终局） | 925+GA | 约1,000万~1亿+ | Pit高层 |

## 7.7 本章小结

角色成长系统核心数据：

| 系统 | 主要收益 | 关键门槛 |
|------|---------|---------|
| 等级1~99 | 技能点×49，属性成长×约160 | 50级→WT3，100级→魔典 |
| 声望系统 | +10技能点，+10魔典点 | 每区域5阶段全满 |
| 魔典系统 | 约200个魔典点，雕文加成 | 雕文升级需Pit经验 |
| 装备强化 | 占总属性~90% | IP925+GA是最终目标 |

**成长阶段口诀**：
- 1~50级：推主线解内容
- 50~70级：推声望+WT3装备
- 70~100级：T4装备+精炼
- 100级后：魔典板+Pit雕文升级
