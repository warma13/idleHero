# 第五章：暴击、易伤与特殊伤害机制

## 5.1 暴击系统深度解析

### 5.1.1 暴击的基本触发逻辑

暴击（Critical Strike）是每次攻击独立判断的随机事件：

```
每次攻击:
  roll = random(0, 1)
  if roll < 暴击率:
      本次命中 = 基础伤害 × 暴击伤害%
  else:
      本次命中 = 基础伤害
```

**AoE技能的暴击**：对AoE每个命中目标**独立判断**暴击，不是一次命中全部目标均暴击或均不暴击。

**DoT技能的暴击**：不同设计，有些DoT在触发时判断一次暴击（如法师炽炎的燃烧每跳），有些DoT不能暴击（如流血）。

### 5.1.2 暴击率的全部来源

| 来源类别 | 具体词缀/技能 | 数值范围 |
|---------|-------------|---------|
| 基础暴击率 | 所有职业固有 | 5% |
| 敏捷属性（仅盗贼/灵魂收割者） | 每50点敏捷=+1% | 约+20%~+40%（满配） |
| 装备暴击率词缀 | "+X%暴击率" | +3%~12%/件 |
| 首饰暴击率词缀 | 戒指/项链效率更高 | +8%~15%/件 |
| 技能被动 | 职业专属被动 | +5%~15%（条件触发） |
| 传奇词缀 | 如"冰封诅咒"：对冰冻目标必定暴击 | 100%（条件） |

**条件暴击**：对特定状态目标触发的必定暴击，是Build设计的核心机制之一：

- **必定暴击（对冰冻目标）**：法师"深度冻结"传奇词缀
- **必定暴击（首次攻击）**：盗贼"暗影步"技能后首击
- **暴击率+100%（堆叠效果）**：野蛮人"战争狂热"被动（狂暴时+15%暴击率）

### 5.1.3 暴击伤害的全部来源

| 来源类别 | 具体词缀/技能 | 数值范围 |
|---------|-------------|---------|
| 基础暴击伤害 | 所有职业固有 | 150% |
| 装备暴击伤害词缀 | "+X%暴击伤害" | +20%~50%/件 |
| 首饰暴击伤害词缀 | 戒指/项链效率更高 | +30%~70%/件 |
| 武器暴击伤害词缀 | 匕首/一手武器偏多 | +25%~55%/件 |
| 技能被动 | 如"死亡一击"（死灵） | +20%对骨系技能暴击伤害 |
| 传奇词缀 | 部分传奇提升特定技能暴击伤害 | +30%~70% |
| 魔典稀有节点 | 高价值稀有节点 | +15%~40% |

**实际T4满配暴击伤害**（以法师为例）：
- 基础：150%
- 装备4件暴击伤害词缀（均值+40%×4=+160%）：→310%
- 首饰2件（均值+55%×2=+110%）：→420%
- 武器词缀+50%：→470%
- 技能被动+30%：→500%
- 传奇词缀+60%：→560%
- **合计：暴击伤害约560%（即暴击=基础伤害×5.6）**

### 5.1.4 暴击系统的极限测算

T4终局满暴击Build（以死灵骨刺为例）：

```
暴击率：80%（5%基础 + 智慧贡献0% + 装备60% + 被动15%）
暴击伤害：550%（见上文推算）

期望系数 = 1 + 0.80 × (5.5 - 1.0) = 1 + 0.80 × 4.5 = 4.6

即：平均每次攻击期望伤害 = 基础伤害 × 4.6（若无暴击为1.0的话）
```

## 5.2 易伤（Vulnerable）系统深度解析

### 5.2.1 易伤的定义与触发

易伤（Vulnerable）是施加在**目标**身上的DEBUFF状态。处于易伤状态的目标受到的伤害乘以易伤加成：

```
伤害含易伤 = 基础伤害 × (1 + 易伤伤害加成%)
```

**基础易伤加成：+20%**（即×1.20）

**易伤持续时间**：约2~4秒（根据触发来源不同）

**易伤来源**：

| 职业/来源 | 触发方式 | 冷却/频率 |
|---------|---------|---------|
| 盗贼"穿透箭（强化）" | 命中时概率触发 | 随机 |
| 法师"冰冻目标自动易伤" | 冰冻状态自动附加 | 随冰冻 |
| 死灵"断骨术"传奇词缀 | 骨碎触发 | 随施法 |
| 野蛮人"震骇吼"（支援） | AOE使怪物易伤 | 15秒CD |
| 德鲁伊"毒刺（强化）" | 击中易伤3秒 | 随施法 |
| 装备词缀"幸运一击：使目标易伤" | 幸运一击触发 | 概率 |
| 赛季机制（部分赛季） | 特定条件 | 赛季专属 |

### 5.2.2 易伤伤害加成的叠加

基础易伤加成=20%，但可通过以下途径提升：

**① 装备词缀"对易伤目标的伤害+X%"**：
```
总易伤加成 = 20%（基础）+ Σ"对易伤目标"词缀%
```
- 每件装备：+5%~15%
- 满配可达：20% + 40%~60% = **60%~80%易伤总加成**

**② 传奇词缀（部分）**：
- "盗贼：死亡之标"：对易伤目标额外+30%伤害

**③ 职业专属**：
- 死灵"腐烂"被动：易伤加成×1.1（即20%基础变为22%）

**最终易伤桶系数示例**：
```
易伤总加成 = 60%
易伤桶修正 = 1 + 60% = 1.60（×1.6倍伤害）
```

### 5.2.3 易伤与冰冻的联动

法师冰冻Build中，冰冻→自动附加易伤，是最稳定的易伤触发来源：

**"冰封必然"（Shatter）连锁**：
1. 法师用减速技能使目标Chilled（减速30%）
2. Chilled→冰冻（Freeze）条件满足时自动冻结
3. 冰冻目标自动获得易伤状态（+易伤加成）
4. 法师用"冰霜碎片"（对冰冻目标伤害+50%系数）攻击

此连锁使法师冰冻Build在T4中具有极高爆发上限。

## 5.3 压制伤害（Overpower）深度解析

### 5.3.1 压制的触发机制

压制（Overpower）是暗黑4独有的特殊攻击类型：

```
每次攻击：
  roll = random(0, 1)
  if roll < 3%:
      触发压制
```

**固定触发率：3%**，不受任何暴击率词缀影响，也不与暴击共存（独立判断）。

**注意**：部分装备词缀和技能可**提升压制触发率**（野蛮人专属词缀），但基础压制率固定为3%。

### 5.3.2 压制伤害的计算公式

```
压制额外伤害 = (当前生命值 + 堡垒生命值) × 压制系数
```

**基础压制系数**：~0.5（约×0.5倍生命总量转化为额外伤害）

实际上，压制是**在正常伤害基础上额外追加**的独立伤害：

```
压制时总伤害 = 正常攻击伤害 + 压制额外伤害
压制额外伤害 = (当前HP + 堡垒值) × 0.5（近似系数）
```

**高配野蛮人压制示例**：
- 当前HP：15,000
- 堡垒值：12,000（战吼维持高堡垒）
- 压制额外伤害 = (15,000 + 12,000) × 0.5 = 13,500

若正常重击伤害为475,000（参见第四章算例），触发压制时：
```
压制总伤害 = 475,000 + 13,500 = 488,500
```

乍看压制额外量不大，但高配野蛮人有专属压制强化词缀：

### 5.3.3 野蛮人压制强化机制

**传奇词缀"震撼大地"**：
```
压制时，额外追加 = (当前HP + 堡垒) × 压制系数 × 震撼加成%
```

**被动"战火燃烧"**：压制触发时，额外造成2秒焚烧效果（按压制伤害量的一定比例）

**重击（HotA）的特殊压制交互**：
- 重击5级基础系数155%
- 若同时触发压制：压制系数在HotA基础上额外×2.0~×3.0（传奇加成）
- 极端情况：HotA压制伤害可达数十亿

**重击压制极限计算**（全套压制强化Build）：

```
基础重击伤害：475,000（见第四章）

压制额外伤害（基础）：13,500
压制传奇加成（×50%伤害桶外额外乘法，见具体词缀）：
假设词缀"压制额外+200%伤害"：
压制追加 = 13,500 × (1 + 200%) = 13,500 × 3 = 40,500

HotA与压制协同传奇词缀（如"震骇核心"）：
所有压制伤害额外×3（针对HotA）

实际压制额外 = 40,500 × 3 = 121,500
压制总伤害 = 475,000 + 121,500 = 596,500（单次期望，3%概率）
```

而极端高配堡垒Build（HP+堡垒共3万以上，多重压制词缀叠加）：
- 单次HotA压制触发可达**数亿~数十亿伤害**（这是"HotA压制Build"的核心理论）

### 5.3.4 压制是否能暴击？

**可以**。压制本身触发后，若同时判定暴击（独立roll）：

```
压制暴击：同时触发压制判断(3%)和暴击判断(暴击率%)
```

这是两个独立的随机判断，可同时触发，概率为：
```
压制暴击率 = 3% × 暴击率%
例（暴击率80%）：3% × 80% = 2.4%概率同时触发
```

压制暴击时：
```
压制暴击伤害 = 正常暴击伤害 + 压制额外伤害 × 暴击伤害%
```

## 5.4 幸运一击（Lucky Hit）系统

### 5.4.1 幸运一击的触发逻辑

幸运一击（Lucky Hit）是触发各类二级效果的概率系统：

```
幸运一击概率 = 技能固有幸运率 × (1 + 幸运率词缀加成%)
```

**每个技能都有固定的"幸运一击率"**：

| 技能 | 幸运一击率 | 备注 |
|------|-----------|------|
| 法师火球 | 33% | 中等 |
| 法师冰霜碎片（每枚） | 5% | 低单次，5枚合计25% |
| 法师闪电箭 | 15% | 中等 |
| 野蛮人旋风斩/次 | 6% | 低单次，高频率 |
| 野蛮人重击 | 25% | 高单次 |
| 盗贼扭曲刀刃（命中） | 30% | 高 |
| 盗贼扭曲刀刃（回旋） | 30% | 高 |
| 死灵骨刺 | 20% | 中等 |
| 德鲁伊龙卷风/次 | 6% | 低单次，多次命中 |

### 5.4.2 幸运率词缀的影响

```
有效幸运率 = 技能固有率 × (1 + 幸运率加成%)
```

**幸运一击率词缀**：
- 装备：+5%~15%幸运一击率
- 满配约+30%~50%幸运率加成

以旋风斩（基础幸运率6%）为例：
- 无词缀：6%/次，约5次/秒 → 0.3次幸运/秒
- +50%幸运率：6%×1.5=9%/次，约5次/秒 → 0.45次幸运/秒

对于高频技能（旋风斩），幸运一击的触发频率较可观。

### 5.4.3 幸运一击可触发的效果

| 效果类型 | 触发内容 | 来源 |
|---------|---------|------|
| 资源恢复 | 幸运一击：恢复X点资源 | 装备词缀 |
| 状态施加 | 幸运一击：使目标易伤 | 装备词缀 |
| 生命恢复 | 幸运一击：恢复X生命 | 装备词缀 |
| 冷却重置 | 幸运一击：重置冷却(概率) | 传奇词缀 |
| 召唤 | 幸运一击：召唤X（部分死灵技能） | 技能专属 |
| 状态效果 | 幸运一击：使目标冻结/麻痹 | 技能被动 |
| 爆炸触发 | 幸运一击：触发法师附魔效果 | 法师专属 |

## 5.5 流血（Bleeding）系统详解

### 5.5.1 流血的触发与叠加

流血是**物理DoT**，与暗影/元素DoT独立：

```
流血每秒伤害 = 触发伤害 × 流血系数 / 持续时间（秒）
```

**流血叠加规则**：
- 每层流血独立计时
- 最多可叠5层
- 每层来自不同的触发事件（可同一来源多次触发）

**流血来源**：
- 野蛮人技能（血腥洗礼、快斩连击强化版）
- 盗贼技能（扭曲刀刃某强化版）
- 部分装备词缀"击中时流血"
- 传奇词缀（如"渗血者"）

### 5.5.2 流血伤害公式

```
单层流血每秒 = 触发攻击伤害 × 流血DoT系数
5层叠满：总流血/秒 = Σ(各层流血每秒伤害)
```

流血是**物理伤害**，受护甲影响（怪物有护甲减免流血），但**不受元素抗性影响**。

## 5.6 灼烧（Burning）系统详解

### 5.6.1 灼烧基础机制

灼烧是**火焰DoT**，由多种技能和词缀触发：

```
灼烧每秒伤害 = 触发时基础伤害 × 灼烧系数（通常0.25~0.5）× (1 + 火焰伤害A伤桶)
```

**关键特性**：
1. 灼烧伤害受目标**火焰抗性**影响
2. 不同来源的灼烧可**叠加**（但同来源刷新不叠）
3. 灼烧可以**暴击**（若触发为暴击，灼烧全程以暴击值计算）

### 5.6.2 法师炽炎Build的灼烧数学

法师"炽炎"技能（引导型）：
- 每秒造成约175%武器伤害的火焰直伤
- 同时每秒追加约30%武器伤害的持续灼烧
- 对燃烧中的目标有额外加成词缀（+X%对燃烧目标伤害）

**炽炎灼烧叠加**：
```
炽炎持续引导：触发约5次/秒
每次触发：附加1层灼烧（175%×30%≈52.5%武器伤害/秒的灼烧）
5层叠满时：灼烧总/秒 = 约5×52.5%=262.5%武器伤害/秒
```

加上炽炎直伤（175%/秒）：
```
炽炎总伤 = (175% + 262.5%)/秒 = 437.5%武器伤害/秒（基础，不含A伤等修正）
```

## 5.7 中毒（Poisoning）系统详解

### 5.7.1 中毒基础机制

中毒是**毒素DoT**，受毒素抗性影响，主要由盗贼技能和部分德鲁伊技能触发：

```
中毒每秒伤害 = 触发伤害 × 中毒系数 × (1 + 毒素A伤桶)
```

**中毒的特殊性**：
- 中毒**不能暴击**（与流血相同）
- 中毒伤害**可以触发幸运一击**（盗贼毒素灌注专属）
- 盗贼"毒素灌注"：使所有技能命中附加中毒

### 5.7.2 盗贼毒素Build的DPS贡献

盗贼扭曲刀刃+毒素灌注：
```
每次扭曲刀刃命中：
  直伤：武器×85%（命中）+ 武器×168%（回旋）
  毒素：附加中毒（武器×约60%/秒，持续5秒 → 总毒素=武器×300%）

毒素是比直伤更高的伤害来源！
```

在扭曲刀刃Build中，直伤约占25%，毒素约占75%。

## 5.8 特殊伤害状态互动汇总

### 5.8.1 "冲击强化"系列

部分效果在目标处于特定状态下额外爆发：

| 效果名称 | 触发状态 | 效果 |
|---------|---------|------|
| 粉碎（Shatter）：法师 | 目标冻结时 | 造成×100%额外物理伤害 |
| 骨碎（Bone Splinters）：死灵 | 目标标记时 | 额外爆裂 |
| 电荷（Charge）：闪电系 | 目标被麻痹时 | +50%闪电技能伤害 |
| 野性（Feral）：德鲁伊 | 目标被减速时 | +20%野性技能伤害 |

### 5.8.2 状态复合效果

多重状态叠加时，各修正**独立计算后乘法叠加**：

例：目标同时处于"冰冻+易伤+麻痹"状态（多属性法师技能）：
```
伤害修正 = 基础
         × 1.2（易伤基础）
         × 1.5（对冰冻目标+50%，技能加成）
         × 1.3（对麻痹目标+30%，传奇词缀）
总修正 = 1.2 × 1.5 × 1.3 = 2.34
即造成基础伤害的2.34倍
```

## 5.10 本章小结

特殊伤害机制核心要点：

**暴击系统**：
- 基础5%率，150%伤害
- 期望系数 = 1 + CR×(CD%-100%)
- 对冰冻/标记目标可实现"必定暴击"条件

**易伤系统**：
- 基础+20%伤害（独立乘法桶）
