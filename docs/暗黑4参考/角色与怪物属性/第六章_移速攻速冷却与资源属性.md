# 第六章：移速、攻速、冷却与资源属性

## 6.1 移动速度（Movement Speed）系统

### 6.1.1 基础移动速度与计算公式

```
有效移动速度 = 基础移动速度 × (1 + Σ移速加成%) × 减速系数
```

**基础移动速度**：所有职业相同，基准=100%（约定为"正常步行速度"）

**减速系数**：若受到减速效果时：
```
减速后移速 = 当前移速 × Π(1 - 减速幅度%)
```
多个减速效果**乘法叠加**（而非加法）：
- 两个减速各30%：`(1-30%)×(1-30%) = 0.7×0.7 = 0.49`，即减速到49%
- 不是`100%-30%-30%=40%`

### 6.1.2 移速的来源

| 来源类型 | 数值范围 | 备注 |
|---------|---------|------|
| 装备词缀"+X%移动速度" | +5%~15%/件 | 常见在靴子 |
| 靴子专用移速词缀 | +10%~20%（靴子比其他部位高） | 靴子是主要移速来源 |
| 技能BUFF（如盗贼"暗影步"后移速+） | +15%~30%（3~5秒） | 技能触发型 |
| 击杀后移速BUFF | +20%（5秒） | 传奇词缀常见 |
| 狂暴（Berserking）移速加成 | +25% | 野蛮人专属BUFF |
| 德鲁伊狼形态移速 | +20%（无技能/空手时） | 形态专属 |

**移速上限（软上限）**：
- 约+165%（即265%绝对移速）
- 超过此值后系统开始施加软性限制
- 在不同版本中有所调整（2.0版本后约为+155%）

### 6.1.3 移速对效率的影响

移速不参与伤害计算，但影响**农怪效率**（单位时间内处理的怪物数量）：

```
清图效率增益 = 移速加成% × 移动时间占总时间比例
```

在噩梦地下城（ND）中，移动时间约占总时间30%~50%：
- 移速+50% → 移动时间缩短33%
- 清图效率提升：50% × 40% = +20%效率

在Pit裂缝中，移动时间更短（密集怪），移速价值相对降低。

## 6.2 攻击速度（Attack Speed）系统

### 6.2.1 攻击速度的构成

```
有效攻击速度 = 武器基础攻速 × (1 + Σ攻速加成%)
```

**武器基础攻速**（次/秒）：

| 武器类型 | 基础攻速 |
|---------|---------|
| 匕首（盗贼） | 1.7/秒 |
| 爪（盗贼/灵魂收割者） | 1.6/秒 |
| 一手剑 | 1.4/秒 |
| 一手斧 | 1.3/秒 |
| 一手锤 | 1.1/秒 |
| 权杖（法师/死灵） | 1.3/秒 |
| 双手剑 | 1.0/秒 |
| 双手斧 | 0.9/秒 |
| 双手锤 | 0.8/秒 |
| 法杖（两手） | 0.95/秒 |
| 弓（盗贼） | 1.0/秒 |

### 6.2.2 攻速词缀的来源

| 来源 | 数值范围 |
|------|---------|
| 装备词缀"+X%攻击速度" | +5%~15%/件 |
| 戒指/手套攻速词缀 | +7%~17%（首饰较高） |
| 狂暴状态（野蛮人） | +30%攻速（BUFF持续时） |
| 技能被动（如盗贼"运动员"） | +15%攻速（条件触发） |
| 传奇词缀（部分） | +20%~40%（条件性） |

### 6.2.3 攻速的实际影响

攻速影响：
1. **基础技能（Basic Skills）的DPS**：直接正比
2. **资源生成速度**：高频技能→更快资源恢复
3. **DoT触发频率**：旋风斩/闪电等持续技能的每秒命中次数

对于施放型技能（Core Skills，按键触发），攻速影响的是"动画帧率"而非触发频率——技能出手更快，但每次伤害量相同。

**攻速对旋风斩的影响**：
```
旋风斩每秒命中次数 ≈ 攻速 × 1.5（基于动画旋转速度）
旋风斩DPS = 每次命中伤害 × 每秒命中次数
```
例：攻速1.5/秒时，旋风斩约9次/秒命中；攻速1.0/秒时约6次/秒。攻速+50%→旋风斩DPS+50%。

### 6.2.4 攻速的收益曲线

**攻速的价值因技能类型而异**：

| 技能类型 | 攻速价值 | 原因 |
|---------|---------|------|
| 持续类（旋风斩、炽炎） | 极高 | 命中次数正比于攻速 |
| 基础技能（快斩、刺击） | 高 | 触发频率正比 |
| 施放型大招（HotA、陨石） | 中 | 主要影响技能前摇 |
| DoT类（流血、毒素） | 中（初始触发） | 触发频率影响DoT叠层 |
| 召唤型 | 低 | 不影响随从攻击 |

## 6.3 冷却缩减（Cooldown Reduction，CDR）系统

### 6.3.1 CDR基础公式

```
实际冷却时间 = 基础冷却时间 × (1 - ΣCDR%)
```

**注意**：CDR是**加法叠加**后再从基础CD中扣除：
- CDR词缀1：+10%
- CDR词缀2：+15%
- CDR词缀3：+20%
- 总CDR：10%+15%+20% = 45%
- 实际CD = 基础CD × (1-45%) = 基础CD × 55%

### 6.3.2 CDR的来源

| 来源 | 数值范围 |
|------|---------|
| 装备词缀"冷却缩减+X%" | +5%~12%/件 |
| 头盔CDR词缀 | +7%~15%（头盔偏高） |
| 技能被动 | +5%~15%（特定技能类型） |
| 魔典板节点 | +5%~12%/节点 |
| 传奇词缀（部分） | +10%~25%（特定技能类型） |

### 6.3.3 CDR的实际价值

CDR的价值取决于技能的CD时长和对Build的重要性：

**高价值CDR场景**：
- 野蛮人"战吼"（CD 25秒）：+30%CDR后变17.5秒，覆盖率从48%→69%
- 法师"冰甲"（CD 14秒，持续10秒）：+30%CDR后CD=9.8秒，接近100%覆盖
- 盗贼"影遁"（CD 20秒）：高CDR后可高频触发暗影状态加成

**边际收益分析**：

```
CDR已有0%时，新增10%CDR：CD×90%（减少10%）
CDR已有50%时，新增10%CDR：CD×(1-60%)=CD×40% vs 原CD×50%（减少20%）
```

**CDR的边际收益递增**（与A伤相反）！高CDR时，继续堆CDR的边际收益**越来越高**。

### 6.3.4 技能最低冷却时间

部分技能设有最低CD下限，防止CDR将CD削减至0：

| 技能 | 基础CD | 最低CD |
|------|--------|--------|
| 野蛮人战吼 | 25秒 | 约5秒 |
| 法师冰甲 | 14秒 | 约2秒 |
| 死灵骨盾 | 7秒 | 约1秒 |
| 终极技能（各职业） | 45~90秒 | 约15~25秒 |

### 6.3.5 条件CD重置

独立于CDR词缀的CD重置机制：

- **幸运一击：重置X技能冷却（X%概率）**：部分传奇词缀
- **击杀精英：重置X技能**：部分被动
- **德鲁伊"自然平衡"关键被动**：每次形态切换有概率重置技能CD
- **法师"能量涌动"传奇词缀**：使用基础技能后X%概率重置核心技能CD

## 6.4 资源系统深度解析

### 6.4.1 资源类型与基础参数

| 职业 | 资源名 | 初始量 | 最大量 | 再生方式 |
|------|--------|--------|--------|---------|
| 野蛮人 | 狂怒（Fury） | 0 | 100 | 主动生成（击中/被击） |
| 德鲁伊 | 精魂（Spirit） | 0 | 100 | 主动生成（基础技能） |
| 死灵 | 精华（Essence） | 100 | 100 | 被动再生+主动生成 |
| 法师 | 魔力（Mana） | 0 | 100 | 被动再生 |
| 盗贼 | 精力（Energy） | 100 | 100 | 被动再生 |
| 灵魂收割者 | 灵魂（Vigor） | 0 | 100 | 主动生成（基础技能） |

### 6.4.2 资源生成公式

**主动生成型（狂怒/精魂/灵魂）**：
```
每次技能命中资源生成 = 技能固有生成量 × (1 + 资源生成加成%)
```

资源生成加成来源：
- 意志属性：每点意志+0.1%资源生成
- 装备词缀"命中时恢复X资源"：固定值，不受百分比影响
- 技能强化：如"强化刺击"（盗贼）+5精力/次

**被动再生型（魔力/精力）**：
```
每秒自然恢复 = 基础恢复/秒 × (1 + 意志×0.1%) + 固定恢复词缀
```

**各职业基础自然恢复**：

| 职业 | 资源 | 基础自然恢复/秒 |
|------|------|--------------|
| 法师 | 魔力 | 约15/秒 |
| 盗贼 | 精力 | 约10/秒 |
| 死灵 | 精华 | 约3/秒 |

### 6.4.3 资源上限扩展

部分装备和被动可扩展资源上限：

| 职业 | 扩展来源 | 效果 |
|------|---------|------|
| 死灵 | 被动"精华积累" | 精华上限+25 |
| 盗贼 | 被动"精力储备" | 精力上限+25 |
| 法师 | 魔典节点 | 魔力上限+20 |

**资源上限的价值**：
- 更大资源池→更少断档时间
- 对于爆发型技能（依赖一次性消耗大量资源）价值更高

### 6.4.4 野蛮人狂怒系统深度解析

野蛮人的狂怒完全依赖主动生成，无自然恢复：

```
狂怒净生成率 = 基础技能生成量×攻速 - 核心技能消耗量×使用频率
```

**关键平衡点**：
- 旋风斩：每秒消耗约25点狂怒（15/秒×约1.7消耗/触发）
- 快斩：每次命中生成约13狂怒，每秒约5次（约65狂怒/秒）
- 净平衡：65-25=40/秒（有盈余，战场上可维持旋风斩）

**狂怒超限机制**（传奇词缀"不屈"）：
```
若狂怒已满且继续生成，超限部分转化为X%伤害加成（5秒内）
```

### 6.4.5 法师魔力系统深度解析

法师魔力：初始0，被动再生，技能消耗：

**魔力消耗**：

| 技能 | 魔力消耗 |
|------|---------|
| 冰霜碎片（每次） | 20 |
| 火球 | 30 |
| 闪电箭（每次） | 15 |
| 炽炎（每秒引导） | 10/秒 |
| 冬霜新星 | 40 |

**魔力恢复**：
- 基础：15/秒
- 意志1,000点：+100%恢复 = 30/秒
- 基础技能"火花"：命中时回复魔力3~5点
- 传奇"无尽之源"：消耗资源时有概率完全不消耗

**魔力建设方程**（以冰霜碎片循环为例）：
```
冰霜碎片消耗 = 20魔力/次
火花回魔 = 约4/次，约3次/秒 = 12/秒
基础再生 = 15/秒
总恢复 = 27/秒
冰霜碎片需求 = 20/次 × 最大施放频率
若每秒能施放1.35次（27/20），则持续循环无断档
```

### 6.4.6 盗贼精力系统深度解析

盗贼精力：初始满100，被动再生约10/秒：

**精力消耗**：

| 技能 | 精力消耗 |
|------|---------|
| 刺击（基础技能） | 生成+5精力 |
| 扭曲刀刃 | 消耗30精力 |
| 穿透箭 | 消耗25精力 |
| 扇射 | 消耗30精力 |
| 移形换影 | 消耗25精力 |

**精力流转循环**（扭曲刀刃Build）：
```
每次循环：
  刺击×2：+10精力
  扭曲刀刃×1：-30精力
  净变化：-20精力
  自然再生：每次循环约1~2秒 = +10~20精力
  循环维持：基本平衡，大约每3~4次循环需要额外刺击1次
```

## 6.5 各属性协同增益分析

### 6.5.1 攻速×意志的协同

```
资源生成效率 = 技能生成量 × 攻击速度 × (1 + 意志效果%)
```

高攻速使基础技能触发频率更高，意志提升每次生成量，两者相乘产生倍增效果：
- 攻速+50%，意志效果+50%：资源生成总提升 = 1.5 × 1.5 = +125%（相对基础）

### 6.5.2 CDR×技能覆盖率

```
防御技能覆盖率 = 技能持续时间 / 实际冷却时间
```

当CDR使覆盖率达到100%时，后续CDR价值归零（技能已满覆盖）。应将多余CDR转换为其他属性。

### 6.5.3 移速×清图路线优化

移速对效率的贡献主要体现在：
- **地图切图速度**（关键：噩梦地下城/开放世界活动）
- **技能追击范围**（移速快则不会因躲避而丢失目标）
- **受伤躲避**（移速快→更易避开BOSS攻击）

## 6.6 本章小结

速度类与资源属性核心公式：

| 属性 | 公式 | 上限/关键点 |
|------|------|-----------|
| 移动速度 | 基础×(1+Σ移速%) | 软上限+165% |
| 攻击速度 | 武器基础×(1+Σ攻速%) | 动画帧率限制 |
| 冷却缩减 | 基础CD×(1-ΣCDR%) | 技能最低CD限制 |
| 资源生成 | 固有量×(1+资源加成%) | 资源上限 |
| 资源再生 | 基础再生×(1+意志%) + 固定 | — |

**设计逻辑**：
- 攻速 = 输出效率的频率乘数（影响持续型技能最大）
- CDR = 防御/关键技能的覆盖率（边际收益递增，但有饱和点）
- 资源 = 技能循环的稳定性（资源断档=空白输出时间）
