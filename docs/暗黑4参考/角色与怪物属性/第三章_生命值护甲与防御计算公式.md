# 第三章：生命值、护甲与防御计算公式

## 3.1 防御体系总架构

暗黑4的防御体系采用**多层叠加**的设计，任何一次伤害命中角色时，会依次经过以下减免层：

```
原始伤害
    ↓ 第一层：元素抗性减伤（火/冰/闪/毒/物理/暗影各自计算）
    ↓ 第二层：护甲减伤（仅物理和部分技能物理伤害）
    ↓ 第三层：直接减伤%词缀（各类"受到伤害降低X%"词缀）
    ↓ 第四层：堡垒减伤（如有堡垒生命且>当前生命）
    ↓ 第五层：屏障（Barrier，先扣屏障，不算减伤）
    → 实际生命值损耗
```

注意：护甲**仅作用于物理类伤害**（包含大多数近战攻击），对元素类伤害无效。

## 3.2 生命值系统完整解析

### 3.2.1 最大生命值计算公式

```
最大生命值 = 基础生命值 × (1 + 生命%词缀总和) + 固定生命词缀总和
```

**基础生命值组成**：
```
基础生命值 = 职业基础值 + 等级成长值 + 意志贡献值
```

**各职业基础生命（1级）**：

| 职业 | 1级基础生命 | 每级增加 |
|------|------------|---------|
| 野蛮人 | 85 | +5 |
| 德鲁伊 | 75 | +5 |
| 死灵法师 | 70 | +4 |
| 盗贼 | 70 | +4 |
| 法师 | 65 | +4 |
| 灵魂收割者 | 75 | +5 |

**等级成长后（100级）各职业基础生命**：

| 职业 | 100级基础生命 | 说明 |
|------|-------------|------|
| 野蛮人 | 85 + 5×99 = 580 | 最耐打近战职业 |
| 德鲁伊 | 75 + 5×99 = 570 | |
| 灵魂收割者 | 75 + 5×99 = 570 | |
| 死灵法师 | 70 + 4×99 = 466 | |
| 盗贼 | 70 + 4×99 = 466 | |
| 法师 | 65 + 4×99 = 461 | 最脆 |

**意志对生命值的间接贡献**：
```
生命值来自意志 = 意志点数 × 5（近似公式，实际随版本更新有波动）
```
注：意志主要通过"治疗效果"间接影响有效生命，直接生命影响相对较小。部分被动技能会直接提供"生命+意志×X"的加成。

### 3.2.2 装备生命值词缀

两类装备生命词缀叠加方式不同：

**① 固定值生命（+X生命上限）**：
```
在基础生命计算后，直接加法叠加
```
- IP700装备：+200~500生命
- IP850装备：+400~800生命
- IP925祖先装备：+600~1,200生命
- GA（崇高词缀）：×约1.5，上限约+1,800生命

**② 百分比生命（+X%生命上限）**：
```
在基础生命计算时，进入(1 + 生命%)的乘法项
```
- 典型范围：+5%~20%/件

**最大生命值实际计算示例**（100级野蛮人，全套T4祖先装备）：

| 来源 | 贡献生命 |
|------|---------|
| 职业基础（100级） | 580 |
| 装备固定生命（7件×800平均） | +5,600 |
| 装备百分比生命（7件×12%=+84%） | × (1 + 0.84) = ×1.84 |
| 技能被动（"铁骨"/"+X生命"） | +约500 |
| 魔典板节点 | +约300~500 |

计算过程：
```
基础生命 = 580
含被动 = 580 + 500 = 1,080
百分比生命后 = 1,080 × 1.84 = 1,987
加上装备固定生命 = 1,987 + 5,600 + 400(魔典) = 7,987
```

注：实际高配野蛮人（含全套崇高词缀）生命约12,000~18,000，因崇高词缀可以大幅放大百分比词缀的效果。

### 3.2.3 生命再生机制

```
每秒生命再生量 = 基础再生量 + 生命再生词缀 + 意志×治疗效果加成
```

**基础再生**：暗黑4中，角色在战斗外有少量生命再生，战斗中**无基础再生**（设计上强调药水依赖）。

**生命再生词缀**（装备）：
- 典型值：每秒+30~80生命再生
- 这个数字在高配下相对于万级HP极低，主要用于小伤补偿

**生命药水**（唯一一瓶，D4改为单瓶大剂量）：
- 1~20级：恢复约40%最大生命
- 21~40级：恢复约35%最大生命
- 41~70级：恢复约30%最大生命
- 71~100级：恢复约25%最大生命（高级药水升级后约40%）

药水冷却：5秒（不受CDR影响）

## 3.3 护甲系统完整解析

### 3.3.1 护甲减伤公式

暗黑4使用以下护甲减伤公式：

```
护甲减伤率 = 护甲值 / (护甲值 + K)
```

其中 **K** 是"怪物参考等级系数"：

| 角色等级/敌对层级 | K值（近似） |
|----------------|-----------|
| 1级 | 75 |
| 50级 | 1,200 |
| 100级（WT1/2） | 5,500 |
| 100级（WT3/折磨T1） | 8,000 |
| 100级（折磨T2） | 9,500 |
| 100级（折磨T3） | 10,500 |
| 100级（折磨T4） | 11,000~12,000 |

**重要**：这意味着同样的护甲值，在更高折磨难度下减伤效率降低，需要更高护甲才能维持同等减伤率。

### 3.3.2 护甲减伤的上限行为

公式 `护甲/(护甲+K)` 是一个趋向但永远不会到达100%的曲线。暗黑4并没有设置"护甲减伤硬上限"（与抗性不同），但实际上：
- 护甲 = K时：减伤50%（称为"等K点"）
- 护甲 = 3K时：减伤75%
- 护甲 = 9K时：减伤90%

对于T4（K≈11,500）：
- 等K点：需要11,500护甲，减伤50%
- 75%减伤：需要34,500护甲（极难实现）
- **实际高配野蛮人护甲约9,000~15,000** → 对应减伤约43%~56%

### 3.3.3 护甲的来源

**① 力量转化（第二章已述）**：1点力量=1点护甲

**② 装备护甲词缀**：
- 护甲类型（胸甲、腿甲等防具类装备）带有大量护甲值
- IP700：+500~1,200护甲/件
- IP850：+1,000~2,000护甲/件
- IP925祖先：+1,500~3,000护甲/件
- GA护甲词缀：约2,000~4,500护甲/件

**③ 技能被动**：
- 野蛮人"铁皮"：+护甲10%~25%（百分比加成）
- 德鲁伊"岩石皮肤"：+护甲20%（施放石肤时）
- 盗贼"闪避滚"：激活时护甲+0（盗贼以回避代替护甲）

**④ 主动技能护甲BUFF**：
- 野蛮人"战吼"（嘲讽）：+100%护甲持续12秒
- 死灵"骨盾"：护甲+30%（持续类BUFF）

### 3.3.4 护甲百分比加成的叠加规则

```
有效护甲 = (基础护甲值 + 力量 + 固定护甲词缀) × (1 + Σ护甲%加成)
```

**叠加示例**：
- 基础护甲（含力量）：5,000
- 固定护甲词缀：+3,000 → 合计8,000
- 野蛮人战吼：+100%护甲
- 被动技能：+15%护甲
- 有效护甲：8,000 × (1 + 1.0 + 0.15) = 8,000 × 2.15 = **17,200护甲**

此时（T4，K=11,500）：
```
减伤率 = 17,200 / (17,200 + 11,500) = 17,200 / 28,700 = 59.9%
```

战吼激活时，单物理伤害减少约60%，效果显著。

## 3.4 元素抗性系统完整解析

### 3.4.1 元素类型与对应抗性

| 元素类型 | 对应抗性 | 主要来源技能/环境 |
|---------|---------|----------------|
| 火焰（Fire） | 火焰抗性 | 焚烧DEBUFF、法师火球 |
| 冰霜（Cold） | 冰霜抗性 | 冰冻/减速效果 |
| 闪电（Lightning） | 闪电抗性 | 麻痹效果 |
| 毒素（Poison） | 毒素抗性 | 中毒DoT |
| 物理（Physical） | 物理抗性 | 多数近战/弓箭攻击 |
| 暗影（Shadow） | 暗影抗性 | 部分精英词缀/法术 |

### 3.4.2 抗性减伤公式

```
受到伤害 = 原始伤害 × (1 - 元素抗性%) × (其他减伤层)
```

注意：不同元素的抗性**各自独立计算**，提升火焰抗性不影响冰霜抗性。

### 3.4.3 "全抗性"词缀

许多装备提供"全元素抗性"（+X%所有元素抗性），同时提升所有6种抗性。

- 智慧→抗性计算公式：`全抗性% += 智慧 × 0.05%`（上限70%）
- 单元素抗性词缀：+15%~25%/件（IP925）
- 全元素抗性词缀：+8%~15%/件（IP925，但同时影响所有元素）

### 3.4.4 抗性穿透（Resistance Penetration）

部分怪物精英词缀和BOSS具有**抗性穿透**属性：

```
有效抗性 = 角色抗性% - 怪物穿透%（最低0%）
```

- 折磨T1精英：偶有+5%穿透
- 折磨T4精英/BOSS：约+10%~+25%穿透
- Pit裂缝守护者：约+20%~+30%穿透

这意味着在T4 Pit中，即使满70%抗性，有效抗性可能只有40%~50%，承受约50%~60%元素伤害。

### 3.4.5 抗性减少效果（Curse/DEBUFF对角色）

死灵的诅咒技能、部分BOSS自带降抗效果：
```
受到的元素伤害倍率 = 1 / (1 - 抗性%) × 降抗效果
```

例：角色冰霜抗性70%，遭受"冰霜诅咒"降低30%冰霜抗性：
```
有效抗性 = 70% - 30% = 40%
```
原本承受30%冰霜伤害 → 现在承受60%冰霜伤害，实际伤害×2。

## 3.5 堡垒生命（Fortify）系统

### 3.5.1 堡垒生命的产生与消耗

堡垒生命（Fortify Value）是暗黑4独有的"超额生命"机制：

```
堡垒值 = 所有"产生X堡垒"效果的叠加值
堡垒值上限 = 最大生命值（不能超出）
```

**产生堡垒的方式**：
- 野蛮人技能"战吼"：每次战吼产生X%最大生命的堡垒值（约15%~25%）
- 德鲁伊被动"天然治愈"：每次变形恢复2%最大生命堡垒
- 死灵"骨盾"技能：消耗精华生成护盾（注意：护盾≠堡垒）
- 装备词缀"击中后产生X堡垒"：约每次击中生成0.5%~2%最大生命堡垒

**堡垒值随时间消耗**：堡垒值每秒以一定比率衰减（约2%~5%/秒），不会永久保持。

### 3.5.2 堡垒激活效果

当**堡垒值 > 当前生命值**时，角色进入"堡垒状态"（Fortified State）：

```
堡垒减伤 = 10%（对所有来源的伤害）
```

附加效果：在堡垒状态下，角色**免疫被"压制"效果**击倒。

**技能加强**：野蛮人"铁骨"被动：
```
堡垒状态下，最大生命额外+X%
```

### 3.5.3 堡垒与护甲的叠算示例

假设野蛮人T4战斗中：
- 护甲减伤：55%
- 堡垒减伤（已激活）：10%
- 战吼护甲BUFF已结束

面对一次原始1,000点物理攻击：
```
步骤1：护甲减伤
伤害后 = 1,000 × (1 - 55%) = 450

步骤2：堡垒减伤
伤害后 = 450 × (1 - 10%) = 405

最终扣血 = 405
```

两层叠算：实际承受40.5%而非(1-55%-10%=35%)加法减伤。

## 3.6 屏障（Barrier）系统

### 3.6.1 屏障的定义与机制

屏障是独立于生命值的临时护盾，**优先于生命值承受伤害**：

```
受到攻击时：
    if 屏障值 > 0:
        实际扣除屏障 = min(屏障值, 伤害值)
        溢出伤害 = max(0, 伤害值 - 屏障值)
    扣除生命 = 溢出伤害 × (其他减伤层)
```

**重要**：屏障承受的伤害**不经过护甲和堡垒减伤**（因为已经先于其他计算层消耗）。

换言之：**屏障≠减伤**，屏障是额外HP池。

### 3.6.2 屏障来源

| 来源 | 屏障量 |
|------|--------|
| 法师"冰甲"技能 | +最大生命30%（持续10秒） |
| 死灵"骨盾" | +最大生命约30%~50%（消耗精华） |
| 装备词缀"击杀后护盾" | +最大生命5%~15%（5秒） |
| 符文"保护符文" | +最大生命X%（按符文等级） |
| 传奇词缀（部分） | 触发型，数量不等 |

### 3.6.3 屏障的性价比分析

法师最依赖屏障生存（基础生命最低）：

```
有效生命（含屏障）= 实际最大生命 + 屏障值 / (1 - 实际减伤率)
```

近似：
```
有效生命 = 8,000（生命） + 2,400（30%屏障）= 10,400
（护甲减伤40%时）有效生命等效 = 8,000/0.6 + 2,400/1.0 = 13,333 + 2,400 = 15,733
```

即屏障在有护甲减伤情况下的价值 = 屏障值本身（因为屏障先被消耗，护甲层无法作用于屏障范围内的伤害）。

## 3.7 直接减伤词缀体系

### 3.7.1 减伤词缀类型

暗黑4的减伤词缀非常多样，大多数是**加法叠加**（先汇总后乘以基础值）：

```
实际减伤率 = 1 - (1 - 护甲减伤%) × (1 - 抗性减伤%) × (1 - Σ直接减伤%)
```

但直接减伤词缀互相之间是**加法叠加**后计算：

```
Σ直接减伤 = 减伤词缀1 + 减伤词缀2 + ... （相加，不相乘）
```

### 3.7.2 各类直接减伤词缀来源

| 词缀类型 | 典型数值范围 | 触发条件 |
|---------|------------|---------|
| 受到近战伤害降低X% | 5%~15%/件 | 无（恒久效果） |
| 受到远程伤害降低X% | 5%~15%/件 | 无 |
| 受到精英伤害降低X% | 4%~12%/件 | 命中来源为精英怪 |
| 对折磨怪受到伤害降低X% | 3%~10%/件 | 折磨难度专属 |
| 受到伤害降低X%（无条件） | 3%~8%/件 | 无 |
| 受到元素X伤害降低X% | 5%~20%/件 | 特定元素 |

**注意**：上述词缀均在装备词缀池内，须占用词缀槽位。优先在头盔、护甲这类"支援型部位"配置减伤词缀。

### 3.7.3 全减伤叠算示例

一个T4盗贼的完整受伤计算（一次精英近战攻击，原始1,500物理伤害）：

```
角色数据：
- 物理抗性：30%（盗贼智慧低）
- 护甲：4,500（盗贼专注敏捷，护甲较低）
- 护甲K值（T4）：11,500 → 护甲减伤 = 4500/(4500+11500) = 28.1%
- 直接减伤词缀：近战-12%，精英-8%（合计-20%）
- 无堡垒状态（盗贼较少堆堡垒）
- 屏障：0

计算流程：
步骤1：抗性减伤
  1500 × (1-30%) = 1050

步骤2：护甲减伤
  1050 × (1-28.1%) = 1050 × 0.719 = 755

步骤3：直接减伤
  755 × (1-20%) = 604

实际扣血 = 604（原始1500的40.3%）
```

## 3.8 回避（Dodge）系统（盗贼专属）

### 3.8.1 回避机制

盗贼具有**回避率**（Dodge Chance）属性，其他职业无此属性：

```
回避率 = 基础回避率 + 技能加成 + 装备加成
```

- 基础回避率：4%（盗贼固有）
- "影步"被动：+5%回避
- "灵活"被动：+4%回避
- 装备词缀"回避率+X%"：+3%~8%/件

**最大堆叠**：约20%~35%（需专门Build）

### 3.8.2 回避的计算时机

回避**在所有减伤计算之前**判定：

```
if random() < 回避率:
    完全避开攻击（0伤害）
else:
    正常计算减伤
```

即回避是一次"概率性无敌"，不是减少伤害，而是完全避免伤害。

### 3.8.3 回避的期望有效生命贡献

```
有效生命提升倍率 = 1 / (1 - 回避率)
```

- 回避率20% → 有效生命 = 实际生命 × 1/(1-0.2) = ×1.25
- 回避率35% → 有效生命 = ×1.538

对于盗贼（约6,000~8,000生命），35%回避率等效额外+约2,300~3,000有效生命。

## 3.9 防御属性综合优先级建议

### 3.9.1 T4生存阈值参考

在折磨T4中，精英怪物的单次攻击约造成：
- 普通精英：800~2,000伤害
- 精英BOSS变体：2,000~5,000伤害
- 世界BOSS：单次技能5,000~15,000伤害

**建议各职业T4最低防御阈值**：

| 职业 | 最低生命 | 推荐护甲（减伤%） | 最低全抗性 |
|------|---------|----------------|-----------|
| 野蛮人 | 8,000 | 9,000（45%+） | 50%（堡垒补） |
| 德鲁伊 | 8,000 | 8,000（40%+） | 55% |
| 死灵 | 6,000 | 6,000（34%+） | 60%（依赖骨盾） |
| 盗贼 | 6,000 | 4,500（28%+） | 50%（回避补） |
| 法师 | 5,500 | 4,000（25%+） | 60%（屏障补） |
| 灵魂收割者 | 7,000 | 7,000（38%+） | 55% |

### 3.9.2 防御属性边际收益排序

针对不同防御阶段，各属性边际收益排序：

**生存较低（刚进T4）**：生命值 > 抗性 > 护甲 > 减伤词缀

**生存达到基础阈值后**：护甲% > 减伤词缀 > 生命% > 抗性（若已达70%）

**满配终局**：堡垒机制 > 屏障维持 > 回避（盗贼）> 细化各类减伤词缀

## 3.10 本章小结

防御体系核心公式汇总：

| 防御机制 | 公式/关键数值 | 上限 |
|---------|-------------|------|
| 护甲减伤 | 护甲/(护甲+K) | 无硬上限，K≈11,500(T4) |
| 元素抗性减伤 | 原伤×(1-抗性%) | 抗性上限70% |
| 直接减伤叠加 | Σ减伤词缀后×(1-总%) | 无硬上限（但边际收益递减） |
| 堡垒减伤 | -10%（满足条件时） | 固定10% |
| 屏障 | 额外HP池，先扣 | 最大生命×N% |
| 回避 | 概率完全避免 | 约40% |

防御的设计哲学是**层层折扣**（而非加法）：
```
最终承受伤害率 = (1-护甲减伤) × (1-抗性减伤) × (1-直接减伤) × (1-堡垒减伤)
```

每一层都是乘法关系，叠加多层减伤的价值超过单一加法堆叠。
