### 10.4.2 边际效用递减模型

```
装备养成的边际价值递减曲线：

第N件传奇的价值（相对第1件）：
  第1件（新Build所需）：价值 = 100%
  第10件（已有完整Build）：价值 = 约30%（新装备大概率不如现有）
  第50件（精选过后）：价值 = 约8%（极低概率有提升）
  第100件（满配状态）：价值 = 约2%（几乎都是垃圾）
  
边际值公式（近似）：
  V(n) = 100% × e^(-0.04n)（指数衰减）
  
  其中n=已有符合条件的装备件数
  
实际含义：
  当你已经有完整Build（约15~20件核心装备到位）：
  继续刷传奇的价值 ≈ 每件只有开始时的约2~8%
  
  此时的刷取应转向：
  1. 换GA词缀（质量维度的提升，不受边际衰减影响）
  2. 精工材料（完全独立于装备RNG）
  3. 神话暗金（每件都是阶梯性提升，边际衰减从~第5件才明显）
```

## 10.6 玩法设计哲学总结

### 10.6.1 暗黑4玩法体系的五个核心设计原则

**原则一：无单一最优解**
不存在一种玩法能在所有维度上最优，每种玩法都有其独特价值：
- ND最强→经验
- 洪潮最强→装备总量和神话暗金
- Pit最强→精工材料
- 低语最强→独特装备CP值
- Boss最强→定向材料

**原则二：时间价值梯度**
游戏内容的时间价值随玩家进度动态变化：
- 同样的内容，满配玩家的收益是新玩家的2~5倍（因为能进更高难度）
- 这创造了持续进步的动力，避免了"到顶就无聊"的问题

**原则三：随机性与确定性的平衡**
- 装备掉落：高度随机（抵消单调感）
- 精工系统：确定性积累（给予持续努力的回报保证）
