# 第二章：主线任务、支线任务与地下城系统

> 暗黑4的主线与支线任务构成了玩家从等级1到进入终局的核心通道，而约170个各具特色的地下城则是贯穿整个游戏的主要战斗场景。本章将系统分析主线六幕的结构与产出，支线任务的奖励价值，以及地下城的怪物分类、地下城词缀机制与掉落产出公式。

---

## 2.1 主线任务系统

### 2.1.1 主线六幕概览

暗黑4的主线故事分为六幕（Act），每幕都集中在特定地区，包含15~25个主线任务：

| 幕次 | 地区 | 关键进度 | 完成后解锁 | 建议等级 |
|------|------|---------|---------|---------|
| 第一幕 | 分裂之丘 | 抵达光明城库拉斯特 | 地下城探索 | 1~15 |
| 第二幕 | 干旱荒地/腐化之地 | 寻找魔法碎片 | 更多地区 | 15~30 |
| 第三幕 | 凋零之地 | 追踪莉莉斯 | 据点系统开放 | 25~40 |
| 第四幕 | 全地区 | 关键剧情转折 | 世界事件触发 | 35~50 |
| 第五幕 | 光辉平原 | 圣战准备 | 地狱洪潮 | 45~55 |
| 第六幕 | 腐化之地/分裂之丘 | 最终决战 | 解锁WT3/WT4挑战 | 50~60 |

### 2.1.2 主线任务奖励机制

每个主线任务完成后提供以下奖励（以WT2基准为例）：

```
主线任务奖励 = 经验值 + 金币 + 可能的特殊装备奖励

经验值公式：
  基础经验 = 任务等级 × 系数（约500~2,000/等级）
  实际经验 = 基础经验 × 世界层级倍率 × 等级差修正

金币产出：
  约：任务等级 × 80~150金币/任务（WT2）

特殊奖励（约30%主线任务）：
  - 指定传奇威能箱子（包含特定Aspect）
  - 专属外观解锁（坐骑装饰/载具皮肤）
  - 声望奖励（所有主线任务均给声望+20~+50）
```

### 2.1.3 主线完成总奖励估算

以WT2模式完整推进主线（约100个主线任务）为例：

| 奖励类型 | 总计（估算） |
|---------|-----------|
| 经验值 | 约1,500万~2,000万（占60级所需的约30%） |
| 金币 | 约800,000~1,200,000 |
| 传奇装备（任务奖励） | 约8~12件定向传奇 |
| 声望点数 | 约2,000~3,000（每区域均匀分布） |
| 外观解锁 | 约15~20个独特外观 |

## 2.2 支线任务系统深度分析

### 2.2.1 支线任务总量与分布

暗黑4共有约**300+支线任务**，分布在全5个地区，分为以下几类：

| 支线类型 | 数量（约） | 特点 | 重复性 |
|---------|---------|------|-------|
| 普通支线（NPC委托） | ~150 | 单次，完成即消失 | 无（仅新赛季重置） |
| 地区事件支线 | ~50 | 限时触发，冷却后可重复 | 有（约30分钟CD） |
| 地下城支线（首通奖励） | ~170 | 完成地下城首次通关奖励 | 无（但地下城可重刷） |
| 据点支线 | ~15 | 据点攻取附带的支线任务 | 无 |

### 2.2.2 支线任务价值评估

支线任务的核心价值体现在**声望贡献**上：

```
声望贡献模型：
  每个支线任务：+20声望
  完整刷完一个地区的支线（约60任务）：+1,200声望
  5个地区合计：+6,000声望
  
  5个地区声望共分为25个等级（每区5级），总声望需求约：
  (200+400+800+1,200+1,600) × 5区域 = 22,500声望总需求

  支线任务+据点+地下城首通的声望贡献：
  支线：300任务×20 = 6,000
  地下城首通：170个×30 = 5,100
  据点：15个×100 = 1,500
  其他探索：约8,000
  合计约：20,600（接近满声望所需的22,500）
```

**声望的最终奖励价值（全区域满声望）：**
- 永久技能点加成：+10点（全5区）
- 装备词缀点加成：+5点（全5区）
- 金币奖励：约5,000,000
- 赤红碎片（PvP货币）：约600

### 2.2.3 效率最优支线刷取策略

对于希望快速满声望的玩家，按声望/时间效率排序：

1. **据点攻取（+100声望/据点）**：约15~25分钟/据点，含大量精英和精华战斗
2. **地下城首通（+30声望）**：与刷经验同步完成，零边际时间成本
3. **支线任务（+20声望）**：路途中顺手完成，避免专程跑路

## 2.3 地下城系统全面分析

### 2.3.1 地下城数量与分类

暗黑4约有**170个地下城**（含DLC新增约20个），按怪物类型分类：

| 怪物阵营 | 代表地下城 | 主要怪物类型 | 精英怪特色 |
|---------|---------|-----------|---------|
| 不死生物（Undead） | 圣光修道院、幽灵墓地 | 骷髅/僵尸/幽灵 | 高生命值，毒素/减速 |
| 恶魔（Demons） | 腐败大厅、地狱祭坛 | 恶魔/地狱生物 | 高爆发伤害，充能冲刺 |
| 野兽（Beasts） | 狂野森林、猎人洞穴 | 狼/熊/食人魔 | 快速移动，打断施法 |
| 邪教徒（Cultists） | 暗血礼堂、骸骨祭坛 | 人类施法者/近战 | 远程骚扰，召唤增援 |
| 奇特生物（Anomalies） | 以太裂缝、魔法遗迹 | 元素生物 | 元素免疫，伤害反弹 |

### 2.3.2 地下城结构与完成条件

每个地下城的完成流程遵循以下标准结构：

```
地下城完成流程：
阶段1：入口探索
  - 进入地下城入口（地图上蓝色标识）
  - 接受当前地下城的"首通任务"（如有）

阶段2：完成地下城目标
  所有地下城均有明确的完成目标，分为以下几种：
  
  类型A：击杀目标（约40%地下城）
    "击杀 X 个[特定怪物]" → 通常需要击杀50~200个特定类型
    时间：约3~6分钟
    
  类型B：机关激活（约25%地下城）
    "激活3个能量核心/祭坛" → 需要找到并激活特定物件
    时间：约5~10分钟（含找物件时间）
    
  类型C：清空房间（约25%地下城）
    "清空X个密封房间" → 逐个清空特定区域
    时间：约4~8分钟
    
  类型D：复合目标（约10%地下城）
    结合以上多种目标
    时间：约8~15分钟

阶段3：击败BOSS
  完成目标后，地下城BOSS解锁
  每个地下城有1~2个BOSS（包含小BOSS和最终BOSS）

阶段4：宝箱奖励
  击败最终BOSS后，出现"英雄宝箱"（保证1件传奇/独特装备）
```

### 2.3.3 地下城产出详细数据

**正常地下城（非噩梦版）产出（WT4 T4基准）：**

| 产出类型 | 每次产出量 | 备注 |
|---------|---------|------|
| 经验值（全清） | 约800,000~1,500,000 | 取决于地下城大小和密度 |
| 金币 | 约15,000~30,000 | 含掉落金币+宝箱金币 |
| 传奇装备 | 约3~7件 | 含最终BOSS宝箱必掉传奇 |
| 通用材料（皮革/铁矿等） | 约50~120件 | |
| 传奇材料（腐化精华等） | 0~2件 | 精英/BOSS击杀掉落 |
| 宝石残片 | 约5~15件 | |
| 金色宝箱（英雄宝箱） | 1个 | 保证1件传奇，有几率独特 |

## 2.4 地下城怪物详细数据

### 2.4.1 普通怪物（Common Enemies）

普通怪物是地下城中数量最多的敌人，其属性遵循严格的公式：

```
普通怪物基础属性公式（WT4 T4，怪物等级100）：

生命值 = 基础生命 × 类型系数 × 难度系数
  基础生命（等级100近战怪）= 约120,000
  类型系数：
    远程施法者：× 0.75（脆弱）
    普通近战：× 1.0（基准）
    盾牌怪：× 1.5（坚韧）
    特殊精英前置怪：× 1.2
  难度系数（T4）：× 20.0（对应×20的生命值倍率）

实际HP（T4普通近战怪）= 120,000 × 1.0 × 20 = 2,400,000（约240万HP）

攻击力 = 基础攻击 × 难度系数
  基础攻击（等级100近战）= 约1,500伤害/次
  实际（T4）= 1,500 × 10 = 15,000伤害/次（玩家护甲减免后约6,000~9,000）
```

### 2.4.2 精英怪物（Elite Enemies）

精英怪是带有3种特殊词缀的强化敌人，在所有玩法中都是最重要的产出来源：

**精英生成规则：**
```
每个地下城区域中，精英怪出现频率约：
  普通区域：每约15~25个普通怪 → 1个精英包（2~5个精英）
  Boss房前区域：密度通常更高（约1个精英包/8~12普通怪）

精英基础属性（相对同等级普通怪）：
  生命值：× 5~8（不同精英词缀影响）
  攻击力：× 2~3
  经验值：普通怪的 × 5~10
  掉落率：装备掉落率 × 3~5（是普通怪的3~5倍）
```

**精英词缀体系（Elite Affixes）：**

| 词缀名称 | 效果描述 | 威胁等级 | 应对策略 |
|---------|---------|---------|---------|
| 灼烧（Burning）| 周期性地面燃烧 | ★★☆ | 远离燃烧地面 |
| 冰封（Frozen）| 触地冰冻玩家 | ★★★ | 保持位移技能 |
| 闪电链（Chaining）| 闪电跳弹减速 | ★★☆ | 分散站位 |
| 毒素云（Toxic）| 持续AOE毒素 | ★★★ | 保持移动 |
| 死亡脉冲（Death Pulse）| 死亡时爆炸 | ★★☆ | 避开尸体 |
| 加速（Vampiric）| 命中后回血 | ★★★ | 快速击杀 |
| 岩石盔甲（Jailer）| 限制玩家移动 | ★★★★ | Unstoppable反制 |
| 黑暗传送（Shadow Clone）| 生成分身 | ★★☆ | 优先击杀分身 |
| 瘟疫（Plagued）| 范围感染DoT | ★★★ | 清除感染 |
| 嗜血（Bloodlust）| 受伤后加速 | ★★★ | 稳定持续伤害 |

### 2.4.3 冠军怪物（Champion Enemies）

冠军怪是介于精英和BOSS之间的特殊敌人，通常是区域守护者或特殊怪物：

```
冠军怪属性：
  生命值：× 15~20（相对同等级普通怪）
  攻击力：× 3~5
  词缀数量：通常1~2个（比精英少）
  
产出特点：
  经验值：精英的×2~3
  掉落：保证1件蓝/黄装备，约20%几率传奇
  材料掉落：约3~5倍普通怪
```

### 2.4.4 地下城BOSS详细数据

每个地下城的最终BOSS都有固定的生命值、攻击模式和掉落表：

```
地下城BOSS基础规格（WT4 T4，等级100）：

生命值 = 普通怪生命值 × 25~40（取决于BOSS设计）
  约2,400,000 × 30 = 72,000,000（约7200万HP）

攻击阶段：
  阶段1（100%~60%HP）：普通攻击模式
  阶段2（60%~30%HP）：激活特殊技能，伤害+20%，新增机制
  阶段3（30%~0%HP）：狂暴（Enrage），所有伤害+40%，攻速+25%

掉落保证：
  必掉：1件传奇或独特装备（英雄宝箱）
  高概率（约45%）：2件传奇
  特殊BOSS（约15%几率）：1件独特装备
  材料（普通）：约8~15件各类工艺材料
  传奇心脏/材料（特定BOSS）：约25%几率
```

## 2.5 地下城噩梦版（Nightmare Dungeon）签文系统

（噩梦地下城的详细玩法在第五章展开，此处仅介绍签文获取机制）

### 2.5.1 签文来源

噩梦签文（Sigil）是进入噩梦地下城的"钥匙"，获取途径：

| 来源 | 获取方式 | 掉落量（约） |
|------|---------|-----------|
| 噩梦地下城通关奖励 | 每次通关获得2~4张签文 | 2~4张/次 |
| 地狱洪潮神秘箱 | 开箱有几率获得 | 约1~3张/次 |
| 精英怪掉落 | 偶发掉落 | 约0.5%/精英 |
| 猎魔人赏金（特定任务） | 完成特定委托 | 约1~2张/次 |
| 制作系统（低级升高级） | 3张低级→1张高级 | 消耗型 |

### 2.5.2 签文等级体系

```
签文等级 = 噩梦地下城的难度等级（T1~T100）

签文等级与掉落的关系：
  T1~T20：掉落WT4低级装备（IP 700~800）
  T21~T40：掉落WT4中级装备（IP 750~850）
  T41~T60：掉落WT4高级装备（IP 800~900）
  T61~T100：掉落WT4顶级装备（IP 860~925），包含更高崇高词缀几率

签文等级对怪物的影响：
  怪物生命值 = 基础 × (1 + 签文等级 × 0.08)
  示例：T30签文→生命值 = 基础 × (1 + 30×0.08) = 基础×3.4
  即T30噩梦地下城的怪物比同等级普通地下城血量多约240%
  
  奖励加成 = 1 + 签文等级 × 0.015（经验加成）
  示例：T30→额外经验+45%
```

|---------|---------|---------|---------|

| 格里盖尔（Garan Hold） | 凋零之地 | 德鲁伊堕落者 | 堕落德鲁伊领袖 |
| 坠日岩（Sunken Ruins） | 干旱荒地 | 不死军团 | 女巫法师 |
| 暗影伏击所（Shadowed Plunge）| 光辉平原 | 蛇毒邪教 | 蛇神使者 |
| ... | ... | ... | ... |

### 2.6.2 据点攻取完成条件

```
据点攻取标准流程：
1. 进入据点区域（地图上红色头骨标识）
2. 完成据点内的特殊目标（各据点不同）：
   - 击败特定数量的精英（多数据点）
   - 拯救被俘的NPC
   - 击毁特殊建筑/机关
3. 击败据点最终BOSS（约1~3个Boss）
4. 据点转变为"已占领"状态：
   - 出现新的NPC和商人
   - 解锁附近的地下城入口
   - 解锁传送点（Waypoint）

攻取时间：约15~30分钟（首次，含探索）
重复清场（已占领后可再次触发）：约8~15分钟
```

## 2.7 本章小结

主线任务提供的核心价值在于：推进世界层级解锁（WT3/WT4）、积累声望奖励（+10技能点）以及构建对游戏世界的基础认知。支线任务则是声望系统的主要贡献来源，其奖励的间接价值（永久技能点）远超任务本身的直接奖励。地下城是暗黑4中使用频率最高的战斗场景，精英怪（高经验×5~10、高掉落×3~5）和BOSS（保证传奇产出）是产出密度的核心来源。在进入终局前，建议将全5地区的声望刷满，以获得最大化的永久属性加成。
