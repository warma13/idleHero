# 第三章：折磨难度系统与深渊裂缝（Pit of the Artificer）

> 深渊裂缝（Pit of the Artificer，简称Pit）是暗黑4中最核心的终局挑战内容，也是获取精工打造材料（Masterworking Materials）的唯一稳定来源。本章将从折磨难度的内部数值结构出发，全面解析Pit的关卡机制、怪物规格、积分系统与奖励产出，并给出精工材料的效率最优刷取公式。

## 3.1 折磨难度（Torment）内部机制

### 3.1.1 折磨四档数值差异精确对比

折磨难度于1.2.0版本（赛季3）取代原有的"世界层级4内部分级"，形成T1~T4四档：

**怪物属性倍率完整表：**

| 折磨等级 | 怪物HP倍率 | 怪物伤害倍率 | 暴击率加成 | 精英词缀数量 | 推荐最低角色装备IP |
|---------|----------|-----------|---------|-----------|---------------|
| T1 | ×2.5（基于WT4基准） | ×2.0 | +15% | 2~3 | 600+ |
| T2 | ×5.0 | ×3.5 | +20% | 3~4 | 700+ |
| T3 | ×10.0 | ×6.0 | +25% | 3~4 | 800+ |
| T4 | ×20.0 | ×10.0 | +30% | 4（满词缀） | 850+ |

**折磨等级对经验产出的加成（叠加在WT4的×2.0基础上）：**

```
总经验倍率 = WT4基础倍率(2.0) × 折磨加成倍率

折磨加成倍率：
  T1：× 1.25 → 总经验倍率 = 2.0 × 1.25 = 2.50
  T2：× 1.50 → 总经验倍率 = 2.0 × 1.50 = 3.00
  T3：× 2.00 → 总经验倍率 = 2.0 × 2.00 = 4.00
  T4：× 3.00 → 总经验倍率 = 2.0 × 3.00 = 6.00

对比WT1基准（1.0）：T4折磨总经验倍率为WT1的6倍
即同样击杀1只等级相近的怪物：
  WT1：1,000经验（基准）
  T4：6,000经验
```

### 3.1.2 折磨解锁条件详解

```
T2解锁：完成"深渊裂缝第20层"（Pit Tier 20）限时通关
  时限：15分钟内击败最终守护者（Boss）
  推荐装备：IP 680+，完整传奇威能套装
  主要挑战：保证单体DPS足够击败T20 Pit Boss

T3解锁：完成"深渊裂缝第40层"限时通关
  时限：15分钟
  推荐装备：IP 780+，开始精工强化（至少8级）
  主要挑战：Boss生命值是T20的约5倍，DPS要求大幅提升

T4解锁：完成"深渊裂缝第60层"限时通关
  时限：15分钟
  推荐装备：IP 860+，精工12级+崇高词缀
  主要挑战：Boss生命值是T20的约15倍
```

```
装备掉落品质公式：

## 3.2 深渊裂缝（Pit of the Artificer）完整解析

### 3.2.1 Pit 系统概述

深渊裂缝是暗黑4中由 **人工者（Artificer）** 主导的专项挑战地下城，于1.4版本（赛季5）引入。其核心特点：

1. **无需签文**：与噩梦地下城不同，Pit不消耗签文，可以无限次重复进入
2. **限时机制**：每次Pit有**15分钟时限**，超时则传送失败
3. **积分系统**：消灭怪物获得积分，积分决定最终守护者的解锁
4. **精工材料产出**：通关后保证产出精工材料，等级越高产出越多
5. **分级上限**：目前Pit共有**200层**（T1~T200），每层难度递增约3~5%

### 3.2.2 Pit 关卡结构

```
深渊裂缝标准流程：

第一阶段：怪物狩猎（约8~12分钟）
  - 进入随机生成的裂缝空间（地图约为普通地下城的60%大小）
  - 地图中散布大量怪物：约200~300只普通怪 + 约20~30个精英包
  - 消灭怪物积累"裂缝能量（Rift Energy）"
  - 目标：在时限内积累足够能量（≥100%）
  
  积分机制：
    普通怪：+0.3%能量/只
    精英怪：+2%能量/只（全部成员）
    冠军怪：+3%能量/只
    特殊精英包（闪光精英）：+5%能量/包
  
  积累至100%能量 → 裂缝守护者（Rift Guardian）生成

第二阶段：守护者击杀（约3~5分钟）
  - 守护者（Rift Guardian）是Pit的最终Boss
  - 守护者等级 = 当前Pit层级 × 调整系数
  - 时限：守护者生成后剩余时间（通常约3~7分钟）
  - 击败守护者 → 领取奖励
  
超时处理：
  若时限结束时守护者未被击败：
    "传送失败"——传送回城，但仍可领取部分奖励（约50%）
    不视为通关，不触发"升层"解锁
```

| 快速刺客 | 10% | 约300,000 | 约15,000/偷袭 | 0.3% |

**精英怪物（Pit专属加强版）：**

Pit中的精英怪具有以下特殊修正：
```
Pit精英生命值加成：
  普通精英：基础HP × Pit层级系数
  层级系数 = 1.0 + (Pit层级 - 1) × 0.05
  示例：T60 → 系数 = 1.0 + 59×0.05 = 3.95
  
  T60精英实际HP ≈ 普通WT4精英HP × 3.95
  ≈ 12,000,000（约1200万HP）

Pit精英词缀特点：
  T1~T20：随机2~3个词缀（同普通地下城）
  T21~T50：随机3~4个词缀，含"Pit专属词缀"
  T51~T100：固定含1个超强词缀+随机3个词缀
  T100+：固定2个超强词缀+随机2个词缀
```

**Pit专属精英词缀（T21+）：**

### 3.2.4 Pit 守护者（Rift Guardian）详细数据

守护者是Pit的最终Boss，具有独特的机制：

```
守护者属性计算：

生命值 = 基础HP(T1) × Pit层级倍率
  基础HP(T1) ≈ 50,000,000（5000万）
  层级倍率 = (Pit层级)^1.4（指数增长）
  
  T20：50,000,000 × 20^1.4 ≈ 50,000,000 × 56.2 ≈ 28.1亿
  T40：50,000,000 × 40^1.4 ≈ 50,000,000 × 150.6 ≈ 75.3亿
  T60：50,000,000 × 60^1.4 ≈ 50,000,000 × 278.5 ≈ 139.3亿
  T100：50,000,000 × 100^1.4 ≈ 50,000,000 × 631.0 ≈ 315.5亿

攻击力 = 基础攻击 × Pit层级倍率（线性）
  基础攻击(T1) ≈ 5,000伤害/次
  T60：5,000 × 60 = 300,000伤害/次（玩家最大防御减免后约60,000~120,000）
```

### 3.2.5 Pit 能量积分效率优化

在限时15分钟内积累100%能量的最优策略：

```
时间需求估算：
  假设Build每秒可以清除：
  - 5只普通怪/秒（AOE清场Build）
  - 或1只精英/秒（单体Build）

  能量积累速度：
    AOE Build：5×0.3% = 1.5%/秒（仅普通怪）
    单体Build：1×2% = 2%/秒（仅精英）
    
    混合场景（70%普通+30%精英）：
    AOE：5×70%×0.3% + 精英清场1×30%×2% = 1.05% + 0.6% = 1.65%/秒
    
  达到100%所需时间：
    AOE：100% / 1.65% = 约60.6秒（仅积分阶段）
    
  实际移动+战斗时间：
    地图探索：约40%额外时间
    总积分阶段：约85~100秒 = 约1.5分钟
    
  加上守护者击杀（约2~4分钟），总Pit时间：约3.5~5.5分钟（顶配Build）
  普通玩家完成T60：约8~12分钟
```

**高效积分策略：**
1. **优先精英**：精英每只+2%，是普通怪效率的6.7倍，应优先清精英
2. **聚怪技术**：将大量普通怪聚集后AOE秒杀，避免一只一只打
3. **不绕路**：专注高密度区域，不跑冷清的死角
4. **守护者生成后立即切单体**：守护者生成后应立刻集中输出，不要继续清小怪

## 3.3 深渊裂缝奖励产出系统

### 3.3.1 精工材料产出规则

精工打造材料是Pit的核心奖励，分为三级：

| 材料名称 | 来源 | 用途 |
|---------|------|------|
| 奥术尘埃（Arcane Dust） | 精炼时分解传奇装备 | 精工强化1~4级 |
| 忘却之石（Forgotten Stone） | Pit T1~T60通关 | 精工强化5~8级 |
| 混沌晶体（Veiled Crystal） | Pit T61+通关 | 精工强化9~12级 |

**精确产出公式：**
```
忘却之石产出（Pit T1~T60）：
  基础产出 = 2 + floor(Pit层级 / 10)
  示例：
    T10：2 + 1 = 3块
    T30：2 + 3 = 5块
    T60：2 + 6 = 8块
  
  通关奖励（守护者击败）：×1.5倍（向下取整）
  限时清楚（5分钟内）：额外+2块
  
  超时失败但仍杀掉守护者：×1.0倍（无加成）

混沌晶体产出（Pit T61+）：
  基础产出 = floor((Pit层级 - 60) / 15) + 1
  示例：
    T61~T75：1块/次
    T76~T90：2块/次
    T91~T105：3块/次
    T120：5块/次
    T150：7块/次

奥术尘埃（所有Pit层级）：
  固定产出：5块（无论层级）
```

### 3.3.2 装备掉落产出

```
Pit通关装备掉落：

守护者击败奖励（保证掉落）：
  - 1~2件传奇装备（100%）
  - 1件独特装备（约15%几率，T40+）
  - 1件独特/神话暗金（约3%几率，T80+）

额外掉落（过程中从精英/守护者身上掉落）：
  精英掉落率：约45%（每只精英有45%概率掉落1件传奇）
  守护者额外：额外掉落约3~5件装备（含传奇）
  
Pit T80+专属：
  神话暗金掉落几率 = 0.1% × (Pit层级 - 79)
  示例：T100 = 0.1% × 21 = 2.1%几率掉落神话暗金
```

### 3.3.3 Pit 等级选择策略

**最优精工材料效率（忘却之石/次）：**

| Pit层级 | 完成时间（顶配Build）| 忘却之石产出 | 效率（块/分钟） |
|---------|-------------------|-----------|--------------|
| T20 | 约3分钟 | 4块 | 1.33/分 |
| T30 | 约4分钟 | 5块 | 1.25/分 |

**结论：T20是忘却之石的最优效率层级（1.33块/分钟）**

```
最优层级选择原则：
  当前精工强化阶段 → 推荐Pit层级：
    精工1~4级（消耗奥术尘埃）：任意Pit层级（刷经验顺带）

## 3.4 Pit 多人组队机制

### 3.4.1 组队对Pit的影响

```
2人组队：
  怪物HP × 1.7（非2倍，减少难度增幅）
  每人获得奖励 × 0.9（略低于单人）
  能量积累速率：双倍（两人同时清怪）
  实际完成时间：比单人约快40%（综合效率提升约35%）

3人组队：
  怪物HP × 2.2
  每人获得奖励 × 0.85
  能量积累速率：约×2.5（三人协同）
  实际完成时间：比单人约快55%

4人组队（满队）：
  怪物HP × 2.8
  每人获得奖励 × 0.80
  能量积累速率：约×3.0
  实际完成时间：比单人约快65%
  但每人奖励降低20%

综合每人材料效率（以单人Pit T20为基准）：
  单人T20：1.33块/分钟 × 1.00 = 1.33块/分钟
  双人T20：单人时间×0.6 = ~1.8分钟 × 0.9奖励×4块 = 约2块/人/分钟
  四人T20：约2.4块/人/分钟（最高效率）

→ 四人组队Pit T20是最高效的精工材料获取方式
```

## 3.5 本章小结

折磨难度系统通过T1~T4的线性伤害/HP倍率为玩家提供了清晰的进阶目标，而深渊裂缝（Pit）则是暗黑4终局最具深度的内容：其积分机制要求在清怪效率和探索覆盖之间取得平衡，而守护者的指数级HP增长（T60约139亿HP）则严格检验Build的单体DPS上限。精工材料的获取效率在Pit T20处达到峰值（1.33块/分钟），而混沌晶体（9~12级精工所需）则只能从T61+获取，形成了清晰的阶梯式进阶压力。
