# 第一章：玩法总览与世界层级系统

> 暗黑4是一款以"刷子游戏（ARPG Loot）"为核心的动作角色扮演游戏。其整个游戏架构分为"前期故事推进"与"后期终局循环"两大阶段，二者通过世界层级（World Tier）系统有机衔接。本章将系统梳理暗黑4的全部玩法类型、世界层级解锁逻辑与各难度下的基准产出体系，为后续各章的详细分析建立完整的框架基础。

## 1.2 世界层级（World Tier）系统详解

世界层级是暗黑4最核心的难度与产出调节机制，共分为**4个层级**，每个层级的敌人强度、经验加成与掉落质量都有明确的数值定义。

### 1.2.1 四大世界层级概览

| 层级 | 名称 | 解锁条件 | 敌人等级 | 经验加成 | 金币加成 | 最高装备IP |
|------|------|---------|---------|---------|---------|-----------|
| WT1 | 冒险者（Adventurer） | 默认解锁 | 跟随玩家（上限35） | 基准×1.0 | 基准×1.0 | ~460 |
| WT2 | 坚忍者（Veteran） | 默认解锁 | 跟随玩家（上限50） | ×1.20 | ×1.15 | ~625 |
| WT3 | 噩梦（Nightmare） | 完成"坚忍者大教堂"挑战 | 跟随玩家（上限73） | ×1.50 | ×1.30 | ~780 |
| WT4 | 折磨（Torment） | 完成"噩梦大教堂"挑战 | 跟随玩家（上限100） | ×2.00 | ×1.50 | 925（祖先级） |

### 1.2.2 世界层级解锁条件详解

**WT3 解锁：坚忍者大教堂（Cathedral of Light，Veteran）**
- 前置：完成游戏主线第三幕
- 挑战内容：独立地下城副本，敌人等级固定为50级（不随玩家等级变动）
- 通关要求：击败最终BOSS"Andariel Clone"（仿安达利亚）
- 平均通关时间：装备齐全的玩家约15~25分钟

**WT4 解锁：噩梦大教堂（Cathedral of Light，Nightmare）**
- 前置：完成WT3状态下的角色升至60级以上（建议）
- 挑战内容：同一地下城，敌人等级固定为70级
- 通关要求：击败强化版最终BOSS
- 平均通关时间：未满装备的玩家约25~40分钟（依Build强度差异）

### 1.2.3 世界层级对产出的精确影响

**经验值计算公式：**

```
实际获得经验 = 怪物基础经验 × 世界层级倍率 × 等级差修正 × 赏金/事件加成

其中：
  世界层级倍率：WT1=1.0，WT2=1.2，WT3=1.5，WT4=2.0
  等级差修正：
    怪物等级 = 玩家等级：× 1.0
    怪物高于玩家1~2级：× 1.05
    怪物高于玩家3~5级：× 1.10（灰色区域，接近最优）
    怪物高于玩家6+级：× 1.15（危险区域，推荐精英玩家）
    玩家高于怪物1~4级：× 0.95
    玩家高于怪物5+级：× 0.85（怪物太弱，明显减少经验）
```

**掉落品质影响：**

| 层级 | 普通（白/灰）掉落率 | 魔法（蓝）掉落率 | 稀有（黄）掉落率 | 传奇（橙）掉落率 | 神话暗金（红）掉落率 |
|------|-----------------|--------------|--------------|--------------|-----------------|
| WT1 | ~65% | ~25% | ~9% | ~1% | 0% |
| WT2 | ~55% | ~28% | ~14% | ~3% | 0% |
| WT3 | ~40% | ~25% | ~25% | ~9.5% | ~0.5% |
| WT4 | ~20% | ~20% | ~35% | ~23% | ~2% |

**注**：以上为参考概率，实际受角色MF（Magic Find）属性、精英/BOSS类型等调整。WT4的传奇+神话暗金合计约25%。

### 1.2.4 折磨难度（Torment Tiers）细分

WT4内部进一步细分为**折磨1~4（T1~T4）**四个子难度，于1.2版本（赛季3）引入：

| 折磨等级 | 敌人生命值倍率 | 敌人伤害倍率 | 额外经验加成 | 解锁祖先装备IP范围 |
|---------|-------------|-----------|-----------|-----------------|
| T1（折磨一） | ×2.5 | ×2.0 | +25% | IP 700~800 |
| T2（折磨二） | ×5.0 | ×3.5 | +50% | IP 750~850 |
| T3（折磨三） | ×10.0 | ×6.0 | +100% | IP 800~900 |
| T4（折磨四） | ×20.0 | ×10.0 | +200% | IP 850~925 |

**折磨等级解锁条件：**

```
T1：进入WT4即自动可用
T2：完成T1的"裂缝第20层"挑战（Pit T20）
T3：完成T2的"裂缝第40层"挑战（Pit T40）
T4：完成T3的"裂缝第60层"挑战（Pit T60）
```

## 1.5 敌人等级与区域等级机制

### 1.5.1 动态怪物等级规则

暗黑4采用动态怪物等级（Enemy Level Scaling）机制，确保游戏全程存在挑战：

```
怪物等级计算：
  基础等级 = 玩家角色等级
  区域等级下限 = 该区域的最低设计等级（例如圣战区域下限为35）
  区域等级上限 = 世界层级对应的上限

  最终怪物等级 = max(玩家等级, 区域下限)，最高不超过世界层级上限
```

**示例：**
一名50级玩家在WT3的干旱荒地（Dry Steppes，区域设计等级30~73）：
- 怪物等级 = max(50, 30) = 50，且不超过WT3上限73
- 结果：怪物等级 = 50（随玩家等级）

同一玩家进入WT4（上限100）：
- 怪物等级 = max(50, 30) = 50（玩家等级制约）
- 随玩家升级至100级，怪物也会升至100级

### 1.5.2 区域难度与最低等级表

| 区域（Region） | 最低等级 | 主题 |
|-------------|---------|------|
| 分裂之丘（Fractured Peaks） | 1 | 雪山、修道院 |
| 干旱荒地（Dry Steppes） | 25 | 荒漠、游牧部落 |
| 凋零之地（Scosglen） | 35 | 森林、德鲁伊领土 |
| 光辉平原（Hawezar） | 45 | 沼泽、蛇神崇拜 |
| 腐化之地（Kehjistan） | 55 | 东方帝国、炼金术 |

## 1.6 角色成长里程碑总览

从新号到T4满配的完整旅程：

```
阶段1：故事推进（约15~25小时）
  目标：完成6幕主线，到达WT3
  建议：一路拾取传奇装备，不要花时间完美Build
  关键节点：
    - 15级：所有技能树层开放
    - 35级：推荐切换WT3（如已解锁）
    - 50级：推荐开始噩梦地下城

阶段2：初级终局（约20~40小时）
  目标：升至70级，切换WT4，进入T1折磨
  主要活动：
    - 噩梦地下城（T1~T20）刷经验
    - 地狱洪潮刷传奇材料
    - 组合完整Build（技能+传奇威能）
  装备目标：IP 700+祖先装备，核心传奇威能到位

阶段3：中级终局（约50~100小时）
  目标：升至90级，进入T2~T3折磨
  主要活动：
    - 深渊裂缝（Pit）刷精工材料
    - 地狱洪潮刷崇高词缀装备
    - 终局BOSS专项刷取（针对性神话暗金）
  装备目标：IP 850+，部分崇高词缀，精工10~12级

阶段4：高级终局（100+小时）
  目标：100级，T4折磨稳定，极限优化装备
  主要活动：
    - Pit T60+（精工材料、测试Build极限）
    - 终局BOSS（烛台幕后Boss等神话暗金刷取）
    - PvP血色之地（Field of Hatred）
  装备目标：IP 925全祖先，崇高词缀最优，精工12级
```

## 1.7 本章小结

世界层级系统是暗黑4所有产出调节的根基：WT1→WT2是新手进阶，WT3→WT4是终局入门，T1→T4是深度刷取的四个阶梯。经验倍率从WT1基准到T4顶端有接近4倍的差异（2.0×基础倍率 × 1+200%额外），装备质量的提升则更为显著——WT4 T4是唯一能掉落IP 925祖先装备和神话暗金的难度。理解了世界层级的产出逻辑，才能在后续章节中正确评估每种玩法的实际价值。
