# 魔力之森 - 实现文档

> 限时挑战副本，玩家在 90 秒内击杀持续涌入的怪物，收集魔力精华获取增益和奖励。

---

## 1. 文件结构

| 文件 | 职责 |
|------|------|
| `scripts/ManaForest.lua` | 核心逻辑：状态管理、战斗入口/退出、怪物生成、精华/增益/涌潮系统、HUD 绘制、GameMode 适配器、存档域注册 |
| `scripts/ui/ManaForestPanel.lua` | 入口面板：独立弹窗，难度选择、次数显示、进入/广告按钮 |
| `scripts/ui/ManaForestResult.lua` | 结算界面：全屏覆盖，展示精华/增益/奖励，再次挑战/返回 |
| `scripts/Config.lua` → `Config.MANA_FOREST` | 所有数值配置（时长、怪物、精华、增益、奖励等） |
| `scripts/GameState.lua` | 存档字段初始化 (`GameState.manaForest`) |
| `scripts/main.lua` | UI 集成：顶栏按钮、面板/结算挂载、结算检测 |

---

## 2. 核心玩法流程

```
玩家点击"魔力之森"按钮
  → ManaForestPanel.Toggle() 打开入口弹窗
  → 选择难度 (普通/困难)
  → 点击"进入森林"
      → ManaForest.EnterFight(difficulty)  消耗次数、初始化战斗状态
      → GameMode.SwitchTo("manaForest")     切换到副本 GameMode
      → BattleSystem.Init()                 开始战斗

战斗中 (90 秒限时):
  → 怪物持续涌入，场上保持上限 (普通 8 / 困难 10)
  → 击杀掉精华 → 积累精华 → 触发 4 阶增益
  → 每 30 秒触发魔力涌潮 → 场上出现 3 个水晶 (存在 8 秒)
  → 拾取水晶 → 额外精华

战斗结束 (超时/全灭/死亡):
  → ManaForest.EndFight(completed)  计算奖励
  → bs.manaForestEnded = true       标记结束
  → main.lua 检测到 → ManaForestResult.Show(result)  弹出结算

结算界面:
  → 显示精华/增益/奖励
  → "再次挑战" → 重新 EnterFight + BattleSystem.Init
  → "返回" → ManaForest.ExitToMain() 恢复关卡 + GameMode.SwitchTo(nil)
```

---

## 3. GameMode 适配器

魔力之森通过 `GameMode.Register("manaForest", adapter)` 注册适配器，位于 `ManaForest.lua` 底部的 `do...end` 块中。

### 适配器接口实现

| 方法 | 作用 |
|------|------|
| `OnEnter()` | 返回 `ManaForest.active`，确认副本已激活 |
| `OnExit()` | 调用 `ManaForest.ExitToMain()` 恢复关卡 |
| `BuildSpawnQueue()` | 返回 `ManaForest.BuildSpawnQueue()` 的怪物队列 |
| `GetBattleConfig()` | 配置限时模式 (`bossTimerMax=90`)、立即开始计时、场上怪物上限 |
| `OnEnemyKilled(bs, enemy)` | 记录击杀、精华掉落、粒子特效；全灭时结束战斗；返回 `true` 跳过正常掉落 |
| `OnDeath(bs)` | 玩家死亡 → `EndFight(false)` 带死亡惩罚 |
| `OnTimeout(bs)` | 超时 → `EndFight(true)` 正常结算 |
| `CheckWaveComplete(bs)` | 所有怪物生成+击杀完毕时结束 |
| `SkipNormalExpDrop()` | 返回 `true`，精华替代经验掉落 |
| `IsTimerMode()` | 返回 `true`，启用倒计时 HUD |
| `DrawWaveInfo(nvg, l, bs, alpha)` | 绘制魔力之森专属 HUD（精华计数、增益进度、涌潮预警） |
| `OnUpdate(dt, bs)` | 每帧更新涌潮计时器、水晶生命周期、精华增益 buff |
| `background` | 专属背景图 `image/battle_bg_forest_*.png` |

### GetBattleConfig 返回值

```lua
{
    isBossWave            = false,
    bossTimerMax          = 90,           -- 限时 90 秒
    startTimerImmediately = true,         -- 开局就开始倒计时
    maxAliveOverride      = 8 或 10,     -- 场上怪物上限
}
```

## 4. 怪物生成

`ManaForest.BuildSpawnQueue()` 构建一个固定长度的怪物队列，BattleSystem 的 Spawner 按 `maxAliveOverride` 控制场上同时存在数量。

### 普通怪

- 行为模板: `swarm` (快速小怪)
- 使用 `MonsterTemplates.Assemble()` 组装，继承章节主题的元素抗性
- 精华掉落: 1 点

### 精英怪

- 行为模板: `bruiser` / `tank` 交替
- HP 倍率: 普通 2.5x / 困难 3.5x
- 按 `calcElitePositions()` 均匀分布在生成序列中
- 精华掉落: 5 点

### 缩放

```lua
scaleMul = StageConfig.CalcScaleMul(maxChapter, 10) * monsterScale
-- monsterScale: 普通 0.80 / 困难 1.30
```

## 5. 精华与增益系统

### 精华来源

| 来源 | 精华值 |
|------|--------|
| 普通怪击杀 | 1 |
| 精英怪击杀 | 5 |
| 涌潮水晶拾取 | 3 |

### 4 阶增益

| 阶级 | 精华门槛 | 效果 |
|------|---------|------|
| Tier 1 | 5 | 攻速 +15% |
| Tier 2 | 15 | 攻速 +15%, 伤害 +10% |
| Tier 3 | 25 | 攻速 +30%, 伤害 +20% |
| Tier 4 | 35 | 攻速 +30%, 伤害 +20%, 暴击 +10% |

增益通过 `bs._manaForestBuff` 传递给战斗系统。

### 涌潮事件

- 每 30 秒触发一次
- 在场地中心区域生成 3 个水晶
- 水晶存在 8 秒后过期
- 临近 3 秒时 HUD 显示 "魔力涌潮即将到来!" 闪烁提示

## 6. 奖励结算

### 结算公式

```
精华→奖励:
  potions   = floor(essence / POTION_RATIO)    -- 普通/8, 困难/6
  forestDew = floor(essence / DEW_RATIO)       -- 普通/7, 困难/5
  gold      = floor(goldBase * goldScale)      -- goldScale 基于 Config.GetGoldScale
  exp       = floor(expBase * scaleMul)

死亡惩罚:
  effectiveEssence = floor(essence * 0.60)     -- 死亡时精华效率 60%

0 击杀:
  → 不消耗次数，不发放奖励

首通奖励 (每日一次):
  → 额外 5 瓶药水 + 3 个森之露
```

### 奖励发放

```lua
GameState.manaPotion.count += potions      -- 魔力药水
GameState.materials.forestDew += forestDew  -- 森之露 (魔力之源升级独占)
GameState.AddGold(gold)
GameState.AddExp(exp)
```

### fightResult 数据结构

```lua
{
    essence           -- 原始精华数
    effectiveEssence  -- 有效精华 (死亡后打折)
    killCount         -- 击杀数
    buffTier          -- 达到的最高增益阶级 (0-4)
    difficulty        -- "normal" | "hard"
    potions           -- 获得药水数
    forestDew         -- 获得森之露数
    gold              -- 获得金币
    exp               -- 获得经验
    firstClear        -- 是否首通
    newRecord         -- 是否刷新精华纪录
    completed         -- true=超时/全灭正常结束, false=死亡
    noKill            -- true=0击杀 (退还次数)
}
```

## 7. 次数系统

| 配置项 | 值 | 说明 |
|--------|-----|------|
| `MAX_DAILY_ATTEMPTS` | 1 | 每日免费次数 |
| `MAX_BONUS_ATTEMPTS` | 5 | 广告可加次数上限 |

- 每日 0 点自动重置 (基于 `os.date` 日期比较)
- 看广告 +1 次，最多 5 次额外
- 0 击杀自动退还次数

### 入口按钮逻辑

```
有剩余次数 → "进入森林" (主按钮)
无剩余 + 可看广告 → "▶ 看广告 +1次" (主按钮替换)
无剩余 + 广告用完 → "今日次数已用完" (禁用)
未通关第4章 → "通关第4章解锁" (禁用)
```

## 8. 存档

### GameState 初始字段

```lua
GameState.manaForest = {
    attemptsToday   = 0,       -- 今日已用次数
    bonusAttempts   = 0,       -- 广告额外次数

    bestEssence     = 0,       -- 单次最高精华
    totalEssence    = 0,       -- 历史总精华

```

### SlotSaveSystem 域注册

在 `ManaForest.lua` 尾部通过 `SlotSaveSystem.RegisterDomain()` 注册：
- `name = "manaForest"`
- `keys = { "manaForest" }`
- `group = "misc"`
- 提供 `serialize` / `deserialize` 函数

## 9. UI 集成 (main.lua)

### 顶栏按钮

在 `quickBuffsBar` 中的"挑战"按钮后面，添加"魔力之森"按钮：

```lua
UI.Panel {
    onClick = Utils.Debounce(function()
        if not GameMode.IsAnyActive() or GameMode.Is("abyss") then
            ManaForestPanel.Toggle()
        end
    end, 0.3),
    children = {
        UI.Label { text = "魔力之森", ... },
    },
}
```

### 面板初始化

```lua
ManaForestPanel.SetOverlayRoot(uiRoot_)
ManaForestPanel.SetStartCallback(function()
    GameMode.SwitchTo("manaForest")
end)
ManaForestResult.SetOverlayRoot(uiRoot_)
ManaForestResult.SetCloseCallback(function()
    GameMode.SwitchTo(nil)
end)
```

### 结算检测 (HandleUpdate 中)

```lua
if BattleSystem.manaForestEnded and not ManaForestResult.IsOpen() then
    ManaForestResult.Show(ManaForest.fightResult)
    BattleSystem.manaForestEnded = false
end
```

### 清理 (TeardownGameUI)

```lua
ManaForestPanel.Close()
ManaForestResult.Close()
```

## 10. HUD 绘制

`ManaForest.DrawHUD()` 在战斗中通过 adapter 的 `DrawWaveInfo` 调用，使用 NanoVG 绘制：

1. **精华计数** — 屏幕顶部居中，显示 "🔮 X 精华 Lv.N"，颜色随阶级变化（灰→蓝→紫）
2. **进度条** — 精华到下一阶级的进度，带圆角矩形样式
3. **涌潮预警** — 涌潮倒计时 ≤3 秒时显示闪烁文字 "⚡ 魔力涌潮即将到来！"

## 11. 关联系统

| 系统 | 关联点 |
|------|--------|
| **魔力之源** (GameState.manaPotion) | 奖励产出魔力药水，自动喝药消耗 |
| **森之露** (GameState.materials.forestDew) | 奖励产出，用于魔力之源升级 (`Config.MANA_POTION_UPGRADE_COSTS`) |
| **日常任务** (DailyRewards) | 通关时 `DailyRewards.Track("manaForestRuns", 1)` |
| **MonsterTemplates** | 怪物组装使用章节主题元素抗性 |
| **StageConfig** | 缩放倍率计算 `CalcScaleMul(chapter, 10)` |
| **关卡保存/恢复** | 进入前保存 `_savedStage`，退出时恢复 `chapter/stage/waveIdx` |
