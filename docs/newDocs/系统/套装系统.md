# 套装系统

## 1. 系统概览

套装系统为角色提供多件联动的强力加成效果。装备同一套装的多件装备后，达到阈值（2件/4件/6件）激活对应档位的套装加成。

**v4.2 变更**：套装装备不再从普通战斗掉落和锻造中产出，改为通过**套装秘境**副本定向刷取。已有的套装装备保留不受影响。

**代码结构**：

| 模块 | 文件 | 职责 |
|------|------|------|
| 套装定义 | `Config.lua` → `EQUIP_SETS` | 8 套套装的效果、颜色、章节归属 |
| 套装计算 | `SetBonus.lua` | 穿戴件数统计、加成激活判定 |
| 套装秘境 | `SetDungeon.lua` | 副本核心逻辑（入口/战斗/奖励） |
| 秘境配置 | `SetDungeonConfig.lua` | 怪物定义、出怪序列 |
| 秘境面板 | `ui/SetDungeonPanel.lua` | 入口 UI（套装选择、难度） |
| 秘境结算 | `ui/SetDungeonResult.lua` | 结算 UI（奖励展示） |

**关联模块**：

| 模块 | 关系 |
|------|------|
| `equip/ItemFactory.lua` | `CreateEquip(quality, chapter, slotId, forceSetId)` 支持强制指定套装 |
| `state/StatCalc.lua` | `SetBonus` 计算的加成参与角色属性汇总 |
| `ui/ChallengePanel.lua` | "套装秘境" 作为挑战面板的第四个标签页 |

## 2. 套装产出方式

### 2.1 套装秘境（唯一产出途径）

| 配置项 | 值 | 说明 |
|--------|-----|------|
| 解锁条件 | 通关第 5 章 | `Config.SET_DUNGEON.UNLOCK_CHAPTER` |
| 每日次数 | 3 次 | 失败不扣次数，可重试 |
| 战斗时长 | 120 秒 | 限时清怪 |
| 出怪数量 | 25 普通 + 3 精英 | 精英 HP ×3 |
| 奖励 | 通关掉落目标套装随机部位 1 件 | 定向选择 |

### 2.2 难度选择

| 难度 | 解锁条件 | 怪物缩放 | 品质权重 [蓝/紫/橙] |
|------|---------|----------|---------------------|
| 普通 | 第 5 章 | ×0.85 | 60% / 35% / 5% |
| 困难 | 第 9 章 | ×1.20 | 30% / 50% / 20% |

### 2.3 套装解锁批次

套装按章节进度分批解锁，玩家只能选择已解锁的套装作为目标：

| 解锁章节 | 可选套装 | 累计可选 |
|---------|---------|---------|
| 第 5 章 | 迅捷猎手、裂变之力 | 2 套 |
| 第 9 章 | 暗影猎手、铁壁要塞 | 4 套 |
| 第 13 章 | 龙息之怒、符文编织、熔岩征服者、极寒之心 | 8 套 |

## 4. 套装加成机制

### 4.1 加成类型

套装效果由 `SetBonus.lua` 计算，提供两种加成方式：

| 类型 | 字段 | 计算方式 | 示例 |
|------|------|---------|------|
| 绝对值加算 | `stats` | 直接加到属性 | `skillDmg +0.15` |
| 百分比乘算 | `statsMul` | 乘算到最终属性 | `hp +20%`, `atkSpeed +12%` |

### 4.2 激活阈值

穿戴同一套装的装备件数达到 2/4/6 件时，分别激活对应档位的加成。高档位包含低档位效果。

### 4.3 buff 机制

部分套装加成包含 `buff` 字段，定义触发条件和特殊效果：

| 触发类型 | 说明 | 使用套装 |
|---------|------|---------|
| `onHit` | 普攻命中时 | 迅捷猎手、裂变之力、熔岩征服者 |
| `onCrit` | 暴击时 | 暗影猎手 |
| `onTakeDmg` | 受击时 | 铁壁要塞、极寒之心 |
| `onSkillCast` | 释放技能时 | 符文编织 |
| `alternating` | 普攻/技能交替 | 龙息之怒 |
| `passive` | 被动持续 | 铁壁要塞(4件) |
| `postBurst` | 爆发后增益 | 暗影猎手(6件)、熔岩征服者(6件) |

仅蓝色（品质 3）及以上的装备可携带套装标记：

| 品质 | 可携带套装 |
|------|----------|

## 6. 套装秘境详细设计

### 6.1 入口

通过主界面 → 挑战 → 套装秘境 标签页进入。

### 6.2 流程

```
选择目标套装 → 选择难度 → 进入秘境 → 限时120秒清怪
→ 全灭通关 → 获得目标套装随机部位1件
→ 超时/死亡 → 失败（不扣次数）
```

### 6.3 怪物配置

| 类型 | 数量 | 特点 |
|------|------|------|
| 蜂群 (swarm) | ~10 | 弱、密集 |
| 强壮 (bruiser) | ~5 | 中等血量 |
| 法师 (caster) | ~5 | 远程 |
| 坦克 (tank) | ~3 | 高血量 |
| 玻璃 (glass) | ~2 | 低血量高攻 |
| 精英 (elite) | 3 | HP ×3，均匀分布 |

怪物强度按玩家最高通关章节缩放，普通难度 ×0.85，困难难度 ×1.20。

### 6.4 存档

| 字段 | 类型 | 说明 |
|------|------|------|
| `attemptsToday` | number | 今日已用次数 |
| `lastDate` | string | 上次重置日期 |
| `totalRuns` | number | 累计挑战次数 |

存档版本 v12（从 v11 迁移时自动初始化）。
