# 核心技能重设计方案（T2 层）

> 版本: v1.0  
> 日期: 2026-04-10  
> 对应代码: `scripts/SkillTreeConfig.lua` T2 SKILLS  
> 设计原则: 每个技能有**独特身份**、**元素联动**、**有意义的增强抉择**

---

## 设计总则

### 核心技能定位
- **主力输出技能**，伤害远高于基础技能
- 消耗法力，有冷却时间（或以法力为代价无CD）
- 门槛: 投入 ≥2 点解锁本层
- 每个技能最高5级，3条增强线（Y形: 强化 → 闪烁/闪耀分支）

### 平衡锚点

| 维度 | 设计目标 |
|------|---------|
| 单次伤害 | 基础技能的 3~5 倍 |
| DPS | 考虑 CD 后，核心技能 DPS ≈ 基础技能 DPS × 1.5~2.5 |
| 元素联动 | 必须与对应元素的状态效果产生协同 |
| 增强抉择 | 两个分支必须代表不同的 Build 方向，不能有明显优劣 |

### 元素状态效果速查

| 元素 | 状态效果 | 机制 |
|------|---------|------|
| 火焰 | 燃烧(Burn) | 持续伤害 DoT，可叠加 |
| 冰霜 | 冻伤(Chill) → 冻结(Freeze) | 减速 → 累积到100%冻结，冻结目标受额外伤害 |
| 闪电 | 眩晕(Stun) | 完全停止行动 |

### 1. 火球 (fireball) 🔥

**技能身份**: 经典AOE爆破 — 高爆发、范围伤害、燃烧联动

#### 基础效果

| 属性 | 数值 |
|------|------|
| 元素 | 火焰 |
| 类型 | 主动·AOE |
| 伤害 | `60 + lv × 12`% 武器伤害 (72%~120%) |
| 法力消耗 | 25 |
| 冷却 | 5s |
| 爆炸半径 | 80px (基础) |
| 描述 | 投掷火球在敌群中引爆，造成%d%%武器伤害。命中燃烧中的敌人时伤害提高25% |

**核心机制**:
- 自动寻找最佳 AOE 落点（敌人最密集处）
- **燃烧协同**: 对已燃烧的敌人伤害 +25%（乘算，非增强节点也生效）
- 命中的敌人施加燃烧：武器伤害的 20%/秒，持续 3 秒（总计 60%）

#### 增强树 (Y形)

```
        [强化火球]          ← 根节点
        /         \
[闪烁火球]     [闪耀火球]   ← 两个分支
```

| 节点 | ID | 效果 | Build 方向 |
|------|----|------|-----------|
| **强化火球** | fireball_enhanced | 火球爆炸后留下**燃烧地面**，持续 3 秒，每秒造成 15%武器伤害 | 通用强化 |
| **闪烁火球** | fireball_flickering | 对已燃烧敌人的额外伤害提升至 **50%**（替代基础的25%）；击杀燃烧敌人回复 **3 法力** | 燃烧增伤流 |
| **闪耀火球** | fireball_glinting | 火球暴击时引发**二次爆炸**，造成原始伤害 40%；爆炸半径 +30% | 暴击AOE流 |

**设计理念**:
- 强化节点提供持续场地伤害，是通用DPS提升
- 闪烁分支聚焦**燃烧联动**：先用火焰弹点燃 → 火球收割，适合 DoT 叠加流
- 闪耀分支聚焦**暴击爆发**：堆暴击率获得连环爆炸，适合高暴击 Build
- 两者代表不同的伤害结构：持续 vs 爆发

### 2. 焚烧 (incinerate) 🔥

**技能身份**: 递增灼烧 — 持续伤害越打越痛，站桩高DPS

#### 基础效果

| 属性 | 数值 |
|------|------|
| 元素 | 火焰 |
| 类型 | 主动·引导·多段 |
| 每段伤害 | `32 + lv × 6`% 武器伤害 (38%~62%) |
| 段数 | 4段（每0.5秒一段，共2秒引导） |
| 递增机制 | 每段伤害倍率递增：1.0x → 1.25x → 1.5x → 2.0x |
| 总伤害 | 基础每段 × (1.0+1.25+1.5+2.0) = **5.75x** 单段 ≈ 218%~356% |
| 法力消耗 | 30 |
| 冷却 | 8s |
| 描述 | 引导火焰射线，每0.5秒造成一段%d%%武器伤害，伤害逐段递增 |

**核心机制**:
- 引导期间自动攻击最近目标，4段命中，伤害逐段递增
- 每段对命中目标施加 **燃烧叠层**：10%武器伤害/秒，持续 3 秒，最多 4 层
- 引导期间角色无法普攻（占用攻击槽位）
- **惩罚/奖励**: 打满4段获得最高DPS，被打断则损失大量伤害

#### 增强树 (Y形)

```
        [强化焚烧]          ← 根节点
        /         \
[闪烁焚烧]     [闪耀焚烧]   ← 两个分支
```

| 节点 | ID | 效果 | Build 方向 |
|------|----|------|-----------|
| **强化焚烧** | incinerate_enhanced | 引导期间移速 +20%，被击中不再打断引导 | 通用生存 |
| **闪烁焚烧** | incinerate_flickering | 第4段（最终段）对燃烧中的敌人额外伤害 +60%；每段命中回复 2 法力 | 燃烧增伤流 |
| **闪耀焚烧** | incinerate_glinting | 引导期间每段获得 **5%最大生命值屏障**（累计最高20%）；引导结束时释放火焰爆炸，对周围敌人造成 80%武器伤害 | 防御反击流 |

**设计理念**:
- 与火球形成**爆发 vs 持续**的分化：火球是瞬间高伤，焚烧是站桩高DPS
- 强化节点解决引导被打断的痛点，是实用性必点
- 闪烁分支极致 DPS：配合燃烧叠层，最终段伤害爆炸式增长
- 闪耀分支攻防一体：引导期间获得屏障保护，结束后释放AOE，适合近战法师

### 3. 电荷弹 (charged_bolts) ⚡

**技能身份**: 散射弹幕 — 多弹体随机散射，命中数量越多回报越大

#### 基础效果

| 属性 | 数值 |
|------|------|
| 元素 | 闪电 |
| 类型 | 主动·多弹体·散射 |
| 每弹伤害 | `35 + lv × 7`% 武器伤害 (42%~70%) |
| 弹体数量 | 5 |
| 理论满中 | 210%~350%（全部命中同一目标时） |
| 法力消耗 | 25 |
| 冷却 | 5s |
| 描述 | 释放%d枚电荷弹向前方散射，每枚造成%d%%武器伤害 |

**核心机制**:
- 释放5枚电荷弹，沿扇形向前方散射
- 每枚弹体独立判定碰撞，可命中不同敌人
- **聚焦奖励**: 敌人越少，单一敌人被命中的弹体越多 → 天然的 Boss 杀手
- 每枚弹体命中时有 **20% 概率**使目标眩晕 0.5 秒（短暂打断）

#### 增强树 (Y形)

```
       [强化电荷弹]          ← 根节点
        /         \
[闪烁电荷弹]   [闪耀电荷弹]   ← 两个分支
```

| 节点 | ID | 效果 | Build 方向 |
|------|----|------|-----------|
| **强化电荷弹** | charged_bolts_enhanced | 弹体数量 +2（共7枚）；眩晕概率提升至 30% | 通用强化 |
| **闪烁电荷弹** | charged_bolts_flickering | 弹体命中眩晕中的敌人时伤害 +40%；每次眩晕敌人回复 **2 法力** | 眩晕联动流 |
| **闪耀电荷弹** | charged_bolts_glinting | 3枚以上弹体命中同一目标时，触发 **电荷过载**：对该目标额外造成 60%武器伤害，并使周围敌人受到 30%武器伤害溅射 | 聚焦爆发流 |

**设计理念**:
- 与连锁闪电形成**散射 vs 弹跳**的分化：电荷弹适合正面输出，连锁闪电适合分散群体
- 强化节点直接增加弹体 = 直接提升DPS和眩晕覆盖
- 闪烁分支聚焦**眩晕循环**：先用电弧打击积累眩晕 → 电荷弹加伤收割
- 闪耀分支聚焦**单体爆发**：对 Boss 时7弹聚焦触发过载，输出极高

### 4. 连锁闪电 (chain_lightning) ⚡

**技能身份**: 弹跳链电 — 在敌群中弹跳，越弹越强

#### 基础效果

| 属性 | 数值 |
|------|------|
| 元素 | 闪电 |
| 类型 | 主动·弹跳·连锁 |
| 首次伤害 | `40 + lv × 8`% 武器伤害 (48%~80%) |
| 弹跳次数 | 6次 |
| 弹跳递增 | 每次弹跳伤害 +10% |
| 理论总伤 | 首次 × (1.0+1.1+1.2+1.3+1.4+1.5) = **7.5x** 首次 ≈ 360%~600% |
| 法力消耗 | 30 |
| 冷却 | 6s |
| 描述 | 释放闪电弹射%d次，首次造成%d%%武器伤害，每次弹跳递增10%% |

**核心机制**:
- 自动选择第一个目标（最近的），然后寻找最近的其他目标弹跳
- 若只有1个敌人，闪电在同一目标上反复弹跳（仍然叠加递增）
- **群体天赋**: 敌人越多弹跳越分散 → 覆盖面广；敌人越少 → 单体伤害集中
- 最后一次弹跳（第6次）伤害最高，设计上鼓励大量弹跳

#### 增强树 (Y形)

```
       [强化连锁闪电]         ← 根节点
        /          \
[闪烁连锁闪电]  [闪耀连锁闪电]  ← 两个分支
```

| 节点 | ID | 效果 | Build 方向 |
|------|----|------|-----------|
| **强化连锁闪电** | chain_lightning_enhanced | 每次弹跳 +3% 暴击率（第6次弹跳时累计 +18%）；弹跳次数 +2（共8次） | 通用暴击 |
| **闪烁连锁闪电** | chain_lightning_flickering | 闪电弹跳到已眩晕的敌人时，眩晕延长 0.5 秒并造成 **双倍伤害**；击杀敌人时闪电额外弹跳 2 次 | 眩晕延续流 |
| **闪耀连锁闪电** | chain_lightning_glinting | 每次弹跳的伤害递增从 +10% 提升至 **+20%**；最后一次弹跳触发 **雷暴**，对目标周围造成 50%武器伤害AOE | 递增爆发流 |

**设计理念**:
- 与电荷弹形成**弹跳覆盖 vs 正面散射**的分化
- 强化节点同时提升暴击和弹跳次数，是纯DPS增益
- 闪烁分支聚焦**控制链**：配合电弧打击的眩晕，形成"眩晕→连锁闪电双倍伤→延长眩晕→继续双倍"的循环
- 闪耀分支聚焦**纯伤害**：8次弹跳递增 +20%，最终弹跳伤害 = 首次 × 2.4，末尾还触发AOE

## 数值对比总览

### 单次释放总伤害（1级 / 5级）

| 技能 | 单次总伤 (lv1) | 单次总伤 (lv5) | CD | 有效DPS (lv5) | 特色 |
|------|---------------|---------------|-----|-------------|------|
| 火球 | 72% (+燃烧60%) | 120% (+燃烧60%) | 5s | ~36%/s | 爆发AOE + 燃烧联动 |
| 焚烧 | 38%×5.75=218% | 62%×5.75=356% | 8s | ~44.5%/s | 站桩递增，高总伤 |
| 冰碎片 | 30%×5=150% | 50%×5=250% | 0(30蓝) | 取决于法力 | 无CD弹幕 + 冻结加伤 |
| 电荷弹 | 42%×5=210% | 70%×5=350% | 5s | ~70%/s | 散射多弹 + 眩晕概率 |
| 连锁闪电 | 48%×7.5=360% | 80%×7.5=600% | 6s | ~100%/s | 弹跳递增，群体覆盖 |

> **注**: 
> - 火球的燃烧后续伤害(60%)未计入有效DPS，实际DPS更高
> - 焚烧需要打满4段才能达到理论总伤，被打断会大幅降低
> - 冰碎片DPS取决于法力回复速度，蓝量充足时DPS极高
> - 连锁闪电理论总伤最高，但分散在多个目标上

| 基础技能 | 单次伤害 (lv5) | 攻速周期 | 基础DPS | 核心技能倍率 |
|---------|--------------|---------|---------|------------|
| 火焰弹 | 35% (+燃烧70%) | ~1s | ~35%/s | 火球≈1.0x, 焚烧≈1.3x |
| 冰霜弹 | 50% | ~1s | ~50%/s | 冰碎片≈法力决定 |
| 电花 | 25%×4=100% | ~1s | ~100%/s | 电荷弹≈0.7x, 连锁≈1.0x |
| 电弧打击 | 120% | ~1s | ~120%/s | 连锁≈0.5x (但含控制) |

> 核心技能的优势不仅在DPS，更在于**AOE覆盖**、**元素效果**、**增强节点带来的质变**。

## 元素联动设计图

```
火焰系:
  火焰弹(基础) ──点燃──→ 燃烧状态 ──增伤──→ 火球(核心) +25%/50%
                                    ──递增──→ 焚烧(核心) 闪烁分支+60%

冰霜系:
  冰霜弹(基础) ──冻伤──→ 冻结状态 ──增伤──→ 冰碎片(核心) +50%

闪电系:
  电弧打击(基础) ──眩晕──→ 眩晕状态 ──增伤──→ 电荷弹(核心) 闪烁+40%
                                     ──双倍──→ 连锁闪电(核心) 闪烁×2
  电花(基础) ──高频──→ 触发概率 ──────→ 电荷弹(核心) 补充眩晕覆盖
```

**跨元素组合示例**:
- **火焰全套**: 火焰弹点燃 → 火球(闪烁)+50%增伤 → 焚烧叠层收割
- **冰法控制**: 冰霜弹冻伤 → 冰碎片冻结加伤 + 弹射
- **雷电爆发**: 电弧打击眩晕 → 连锁闪电(闪烁)双倍伤 → 眩晕延长循环
- **雷电弹幕**: 电花高频 → 电荷弹(闪耀)聚焦过载 → 单体爆发

## 增强节点命名规范

所有核心技能统一使用 Y形增强树，命名格式：

| 位置 | 命名规则 | 含义 |
|------|---------|------|
| 根节点 | 强化{技能名} | 通用强化，两分支的前置 |
| 分支A | 闪烁{技能名} (flickering) | 偏向元素联动/持续/法力效率 |
| 分支B | 闪耀{技能名} (glinting) | 偏向暴击爆发/AOE/防御 |
