## 1. 经验与升级

| 参数 | 值 | 来源 |
|------|----|------|
| 每级获得属性点 | 3 | `Config.POINTS_PER_LEVEL` |

### 经验获取

```
实际获取 = 基础经验 × (1 + 药水加成 + 称号加成)

药水加成:
  小 = +30%
  中 = +30% × 1.4 = +42%
  大 = +30% × 2.0 = +60%
```

### 升级流程

升级时自动:
- `freePoints += 3` (每级)
- 无等级上限

## 2. 攻击力 (ATK)

| 参数 | 值 |
|------|----|
| 基础攻击 | 18 |

### 计算公式

```
TotalAtk = floor(SM.Apply("atk", baseAtk + 装备atk))
```

- `baseAtk = 18` (术士基础)
- `装备atk` = 所有已穿戴装备的 atk 词缀之和 (含宝石)
- `SM.Apply` = 属性修饰器 (buff/debuff 乘算)

> 攻击力**不来自属性点加点**，纯装备+技能+称号。

### 攻击药水

```
伤害倍率 = 1 + 药水效果值

小: +5%   (0.05)
中: +7%   (0.05 × 1.4)
大: +10%  (0.05 × 2.0)
```

## 3. 攻击速度 (SPD)

| 参数 | 值 |
|------|----|
| 武器基础攻速 | 1.0 |
| 第一类上限 | 100% (`Config.ATK_SPEED_CAP1`) |
| 第二类上限 | 100% (`Config.ATK_SPEED_CAP2`) |
| 攻速下限 | 0.15 |

### 计算公式

```
总攻速 = (1 + 第一类攻速加成 + 第二类攻速加成) × 武器攻速
最终攻速 = max(0.15, 总攻速)
```

| 类别 | stat key | 说明 | 来源 | 上限 |
|------|----------|------|------|------|
| **第一类** | 直接计算 | 面板加成（换装/升级时变化） | 装备spd词缀、套装atkSpeed%、速射技能(每级+8%) | 100% |
| **第二类** | `atkSpd2` | 触发加成（战斗中动态变化） | 迅捷猎手6件连击风暴、裂变之力6件脉冲后攻速、减速/孢子云/灼烧debuff | 100% |

- 第一类在 `GetAtkSpeedPool1()` 中加算，第二类通过 `SM.Apply("atkSpd2", 0)` 汇总
- 两类各自独立 clamp，互不侵占
- debuff 以 `flatSub` 方式扣减第二类，可将第二类压为负值（但不低于 -100%）

## 4. 暴击率 (CRT)

### 计算公式

```
原始暴击率 = 基础暴击率
           + DEX职业加成        -- 每点DEX +0.02%
           + 装备crit
           + 奥术感知天赋       -- 每级 +3%
           + 称号加成

实际暴击率 = clamp(SM.Apply("crit", 原始暴击率), 0, 1.0)
```

### 暴击率溢出

```
溢出部分 = max(0, 原始暴击率 - 100%)
转换暴伤 = 溢出部分 × 2          -- 1%溢出 → 2%暴伤
```

## 6. 血量 / HP

| 参数 | 值 |
|------|----|
| 基础HP | 500 |
| 每级+HP | 12 |

### 计算公式

```
基础值 = baseHP + level × hpPerLevel + floor(装备hp)
       = 500   + level × 12          + 装备hp

hpMul = 1.0
      + STR职业hpPct加成      -- 每点STR +0.1%
      + 装备hpPct
      + 称号hp加成
      + 套装hp%加成
      + HP药水加成

MaxHP = floor(基础值 × hpMul)
```

### HP药水

```
小: +10%
中: +20%  (独立倍率, 非通用 sizeMul)
大: +30%
```

## 8. 防御力 (DEF)

| 参数 | 值 |
|------|----|
| 基础DEF | 5 |
| 每级+DEF | 0.3 |
| 玩家 K 基础 | 200 |
| 玩家 K 每级增长率 | 4% |
| 怪物 K 基础 | 100 |
| 怪物 K 每级增长率 | 3% |

### 总防御力

```
TotalDEF = floor(基础值 × (1 + 倍率池))

基础值 = baseDEF + level × defPerLevel + STR护甲加成 + 装备def
       = 5      + level × 0.3          + STR×1×AttrScale + 装备def

倍率池 = 套装def% + 称号def%
```

### DEF 减伤公式

```
K_player(怪物等级) = 200 × (1 + 0.04 × (怪物等级 - 1))

伤害保留率 = K / (DEF + K)
减伤率     = 1 - 伤害保留率 = DEF / (DEF + K)
```

- K 随怪物等级增长 → 玩家需持续提升 DEF 才能维持减伤
- 腐蚀 debuff: 每层降低一定比率 DEF


## 12. 闪避 / 全抗 / 超杀

### 闪避 (Dodge)

```
闪避率 = DEX点数 × 0.00025
上限   = 30%
```

### 全元素抗性 (All Resist)

```
全抗 = INT点数 × 0.0005
上限 = 40%
```

### 抗性减伤公式 (三段曲线)

```
resist < 0:        伤害倍率 = 1 - resist/2        (负抗增伤)
0 ≤ resist < 0.75: 伤害倍率 = 1 - resist          (线性减伤)
resist ≥ 0.75:     伤害倍率 = 1 / (1 + resist/4)  (渐近线递减)
```

### 超杀伤害 (Overkill)

```
超杀加成 = WIL点数 × 0.0025
触发阈值 = 目标HP < 30%
```

## 13. 技能冷却缩减 (CDR)

| 参数 | 值 |
|------|----|
| 最大 CDR | 75% |
| 半值常数 K | 0.50 |

### 渐近线递减公式

```
totalCDR = 天赋CDR + 装备CDR + 套装CDR + WIL职业CDR

cdMul = 1.0 - maxCDR × totalCDR / (totalCDR + K)
      = 1.0 - 0.75   × totalCDR / (totalCDR + 0.50)
```

- 投入 0.50 时获得 37.5% 实际 CDR
- 永远无法达到 75%

## 19. 总 IP 计算

```
总IP = 所有已穿戴装备的 itemPower 之和
```

### 单件装备 IP 计算

```
BaseIP(chapter) = floor(100 + 825 × ln(chapter) / ln(100))
  ch1=100, ch16≈568, ch100=925

实际IP = BaseIP × 品质系数 + 升级加成
  每次强化: IP += Config.IP_PER_UPGRADE
```

### 词缀 IP 缩放分类

词缀分为两类，决定其值是否随 IP 增长：

| 分类 | ipScale | 包含词缀 | 设计逻辑 |
|------|---------|---------|---------|
| **IP 增长型** | 1.0 | atk, hp, def, hpRegen | 绝对值属性，与敌人数值赛跑 |
| **Roll 品质型** | 0 | 其余 25 条 (crit, critDmg, spd, 元素增伤, 抗性, 触发词缀等) | 百分比/效率属性，价值来自 roll 品质 |

词缀值公式: `value = base × (1 + (ip/100 - 1) × ipScale) × roll`
- ipScale=1.0 时: IP 翻倍 → 值翻倍
- ipScale=0 时: ipFactor 恒等于 1，值 = base × roll

### 主线掉落品质 (IP_BRACKETS)

主线章节掉落 roll 品质封顶 0.60，为后期系统留出增长空间：

| IP 区间 | minRoll | maxRoll | 阶段 |
|---------|---------|---------|------|
| ≤150 | 0.15 | 0.30 | 新手期 |
| ≤300 | 0.20 | 0.38 | 开始关注词缀 |
| ≤450 | 0.25 | 0.45 | 学会比较装备 |
| ≤600 | 0.28 | 0.50 | 追求特定词缀 |
| ≤800 | 0.30 | 0.55 | 主线中后期 |
| >800 | 0.33 | **0.60** | **主线天花板** |

后期系统提供 0.60 → 1.00 的增长空间：
- 装备副本 (0.50~0.85)
- 进阶/洗炼 (单条词缀 +0.05~0.15)
- 附魔 (重 roll)
- 世界Boss (0.70~0.90)
- 传说打造 (0.85~1.00)

## 20. 章节数值缩放

### 装备 Tier 系数

```
TierMul(chapter) = 1.0 + 99 × ln(chapter) / ln(100)
  ch1=1.0, ch100=100.0
```

### 属性点缩放因子

```
AttrScale(chapter) = sqrt(TierMul(chapter))
  ch1=1.0, ch100=10.0
```

- STR 护甲加成使用 AttrScale 缩放
- 攻速增量上限使用 AttrScale 缩放

## 21. 属性修饰器系统 (StatModifiers)

所有 `Get*` 函数最终经过修饰器管道:

```
result = (base + flatAdd - flatSub)
       × (1 + sum(pctPool))
       × prod(1 + pctMul_i)
       × prod(1 - pctReduce_j)
```

| 类型 | 说明 |
|------|------|
| `pctPool` | 加算百分比池, 求和后统一乘算 |
| `pctMul` | 独立乘算, 逐个乘 |
| `pctReduce` | 独立减算, 逐个乘 (1-val) |
| `flatAdd` | 固定加值 |
| `flatSub` | 固定减值 |
